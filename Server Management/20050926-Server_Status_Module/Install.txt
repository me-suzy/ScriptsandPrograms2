Server Status Module for PHP-Nuke on CPanel

This module has been tested on PHP-NUKE 7.8

What you need:
- Cpanel hosting based domain
- Zend Optimizer installed

How to install this module:
1) unzip
2) add the file under "html" folder to your PHP-NUKE
4) edit modules/ServerStatus/c_config.php and set you config params

that's all!
