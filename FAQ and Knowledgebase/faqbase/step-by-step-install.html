FAQBase Installation Instructions

The first thing to do is open faq-config.php. The first set of variables are your MySQL settings. The ones that are left blank MUST be completed. The last 2 are the table names which can be left as they are, or you can change them if you need to. Complete these MySQL settings now.

$adminpass needs to be set to your own admin password. Do NOT leave this blank, and do not leave it as the default password. The rest of the variables here are for the design of FAQBase. If you'd liek to get started on using FAQBase now, you can come back to these later. The variables should all be self-explanatory. :)

Now save faq-config.php and upload all these files to your site, anywhere you want. :) Once you've finished, run create_tables.php from your web browser. This will install the MySQL tables. Follow the instructions from there.

To gain admin priviledges, a cookie needs to be set in your web browser. That cookie will be your admin password. Run setmycookie.php from your web browser to set that cookie. When you're finished, BE SURE to delete setmycookie.php from your site! :) Now, when you go to your FAQ page, you'll see links for adding categories. This is the first place to start for adding content to the FAQ. You'll find more links after you've added categories. :)

That's it! You're ready to start using FAQBase. By now you've probably noticed the default design and color settings. You can edit faq-config.php to change these. If you have questions or suggestions regarding FAQBase, email me.
