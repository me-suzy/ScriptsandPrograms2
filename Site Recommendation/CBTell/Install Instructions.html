Step 1: Unzip the CBTell.zip file.

If you do not have a software to unzip the CBTell.zip file, you can download a free trial version of WinZip by clicking here

After "unzipping" the CBTell.zip file you should have the following files as shown below. If you do not have a similar structure as the one below, check to make sure you have your zip program preferences set to allow the creation of directories and to preserve case and long filenames.

Files included in CBTell.zip:
• Copyright Notice.html
• Install Instructions.html
• admin.php
• cb.php
• codelock.php
• config.php
• footer.php
• header.php
• tell.php
• util.php

Step 2: Create a new directory on your website named cbtell (or any name you choose).

Step 3: Upload all files into your newly created cbtell directory. Be sure to set the file permissions of the config.php file to 777.

Step 4: Access your admin area at http://domain.com/cbtell/admin.php by entering the temporary admin password "pass" (without the quotes). Once in the admin area, be sure to change your password first. Then modify any of the other settings if you need to.

Email Templates Information
================================
The email templates found in the admin are used to send customized emails to those being sent emails recommending your website as well as the person who is actually recommending your website. Each email has variables which can be used. They are shown below:

[[sitename]] = Your Website Name
[[siteurl]] = Your Website URL
[[name]] = Name Of The Person Recommending Your Website
[[email]] = Email Of The Person Recommending Your Website
[[friend]] = Name Of The Recommenders Friend
[[friendemail]] = Email Of The Recommenders Friend
[[message]] = Optional Message Entered By The Recommender For Friend To read
[[emails]] = Email Address of Recommenders Friend(s) (In Confirmation Email)
