A1 - Menü: Adaptives PHP Navigations-Menü-System für größere Websites

Übersicht:

A1 - Menü erlaubt die optimale Navigation gerade bei umfangreichen Websites. Es ist kostenlos. Es wird aus den Seiten in jedem Verzeichnis automatisch ein Menü aufgebaut und nach eigenen Vorgaben sortiert. Es können bestimmte Dateien und Gruppen ein- ausgeschlossen werden.

 

A1 - Menue: Flexibel PHP-navigation system with automated menues

Summary:

A1 - Menue allows the easy navigation especially for larger websites. It is free. For each directory a customizable menue is created and sorted following own rules. Files and directories may be included or excluded following own definitions.
