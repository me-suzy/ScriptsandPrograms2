Install

Deutsch
=========

1. Menu.inc anpassen

Zeile 31: Link zur eigenen Homepage eintragen
Zeile 49: Selektierte Dateitypen auswählen (hier: PHP3)
Menu.inc ins Hauptverzeichnis einspielen
header.php3  "                           (nur für die DEMO)
footer.inc   "                                    "

2. Verzeichnis images im Hauptverzeichnis anlegen
Dateien aus dem Verzeichnis images dort einspielen

3. In die gewünschte Stelle des Menüs
   <? include "/server-Pfad/menu.inc"; zeigen("1"); ?>
einfügen (funktioniert auch mit URL)

4. OPTIONAL - Um die Demo auszuprobieren:
- Verzeichnis A1menu anlegen
- Unterverzeichnis Files anlegen
- alle Dateien/Verzeichnisse aus "Files" auf den Server aufspielen

Aufruf mit http://www.ihredomain.de/A1menu/



ENGLISCH 
=========

1. Make changes to menu.inc

Line 31: Change Link to your own homepage
Line 49: Select proper file type for menu (now PHP3)
Copy menu.inc to your main directory
Copy footer and header to main directory (for test files only)

2. Create directory images in your main dir
Copy GIF - Files from \images to this file

3. Insert in your file at the proper place the call of the menu
   <? include "/server-Pfad/menu.inc"; zeigen("1"); ?>
 (works also with URL)

4. OPTIONAL: To play with the demo:
- create A1menu and
- Subdir "Files" and
- copy the files from "Files" and all subdirs to your server

Call with http://www.yourdomain.com/A1menu/
