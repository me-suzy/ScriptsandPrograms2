XL-quiz

“Repetito est mater studiorum”

Skripti tutvustus

Tegemist on järjekordse testisüsteemiga. Sellega on võimalik esitada kolme tüüpi küsimusi:

  • radio-tüüpi - ühe õige valikuvariandiga
  • checkbox-tüüpi - mitme valikuvariandiga
  • tekst-tüüpi - vastus tuleb ise väljakirjutada

Küsimusi / variante võib olla piiramatul hulgal.

Kuna ma olen piisavalt laisk siis administreerimisliidest asjal ei ole — testide koostamine toimub… MS Exceli (või OOCalc) kaudu :P

Demo

On siin php.bytez.net/xlquiz/

Paigaldus

  1. Kopeeri failid serverisse
  2. chmod'i "cache" kataloog 777
  3. Tee mysql tabel
  4. Määra configuration.php-s mysql ühendumiseks vajalikud andmed

Kasutamine

Selline on näidistest: example-test.xls

Testifailid (exceli formaadis) tuleb kopeerida "db" kataloogi. Esimesel käivitusel parsitakse sealt info cache kataloogi, niiet järgmisel käivitusel peaks see toimuma juba kiiremini.

Lisamaks linki uuele testile kasuta sellist formaati:
<a href="taketest.php?id=failinimi">Testi nimi</a>

NB! Failinimes laeindit ei pea panema
