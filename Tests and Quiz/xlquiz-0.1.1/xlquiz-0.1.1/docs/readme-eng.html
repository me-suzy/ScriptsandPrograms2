XLquiz v0.1

“Repetito est mater studiorum”

Overview

It's yet another test-system. It allows 3 kind of questions:

  • radio
  • checkbox
  • text

Questions/variants can be unlimited number.

And because I'm so lazy, there aren't admin-panel — tests are made with … MS Excel (or OOCalc) :P

Demo

Is here php.bytez.net/xlquiz/

Installation

  1. Copy files into server
  2. chmod "cache" folder 777
  3. Create mysql table
  4. Modify configuration.php with your mysql connection data

Usage

Here is example test: example-test.xls

Testfiles (in Excel format) should go to "db" folder. At the first running there is parsed data into "cache" folder.

For adding link to new test use such format:
<a href="taketest.php?id=filename">Test name</a>

Nb! Don't add file extension!
