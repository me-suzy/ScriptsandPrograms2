Traffic Splitter 1.0


Installation Instructions

Installation is simple. Open the split.php file in a text editor and edit the variables at the top of the file (instructions are included in the file itself). Then upload the script in ASCII mode to your server.
Note: Some servers have trouble processing the script if you place it inside of your cgi-bin directory.

Usage Instructions

What this script does is allow users to shave some traffic off of their links, and send it to other urls, like a sponsor.
For example:

yahoo.com
<a href="http://www.smarterscripts.com/splitter/split.php?u=http://www.yahoo.com">yahoo.com</a>

90% of the time (which is the default hit percentage defined in the split.php), the user will be taken to yahoo.com, and the rest of the time, the traffic will be split between google, dmoz, and altavista (these sites are defined in the split.php and you can add/subtract as many sites as you want very easily).

The script can be used in many different ways. The default hit percentage, set in the the split.php file is 90 or 90%, but you can change that manually using a url like the one below:

php.net
<a href="http://www.smarterscripts.com/splitter/split.php?u=http://www.php.net&p=40">php.net</a>

Now, only 40% of the traffic will go to php.net.

If you dont want any variables in the url, you could do that like this:

msn.com
<a href="http://www.smarterscripts.com/splitter/split.php?http://www.msn.com">msn.com</a>

the default hit percentage is still 90

And, if you just want to send random traffic split between whichever sites you listed in the split.php file, you can simply link to the split php file like this:

google, dmoz or altavista?
<a href="http://www.smarterscripts.com/splitter/split.php">google, dmoz or altavista?</a>

I designed this script so I could shave traffic off of my links and send it to a sponsor to make money, without pissing off the majority of my sites visitors. It is a very simple, but useful script.
