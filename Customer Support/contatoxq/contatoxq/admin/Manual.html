Manual
 
Respostas:
-Evite responder contatos com Status: "Resposta" o sistema
cria nova hierarquia para cada contato respondido
-Para listar todas as respostas de um cliente, use o menu: "Clientes"
-Ao responder um contato, digite Re: na frente do assunto para
facilitar a identificacao das respostas
 Email:
-Antes de usar o sistema e necessario a configuracao dos dados
do Email acessiveis pelo Menu: "Config"
-Para que haja o envio e necessario que o servidor aceite a funcao
email() sem necessidade de autenticacao
Atualizacoes:
Atualizacoes sao avisadas via boletins, caso contrario visite o site.
 Design e estilos:
Para mudar o tema de cores do sistema utilize o arquivo common.php
na linha: 49 estilo e cores use as folhas de estilos CSS em cada pasta
de estilo
