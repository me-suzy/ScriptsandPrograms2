Class TMXResourceBundle

Description
Description | Vars (details) | Methods (details)

<p>This PHP Class reads resource text data directly from a TMX (XML) file.

First, the XMLTMXResourceBundle class instantiates itself with two parameters: a TMX file name and a target language name. Then, using an XML parser, it reads all of a translation unit's properties for the key information and specified language data and populates the resource array with them (key -&gt; value).</p>
