Software TSEP uses

We are using other software for TSEP to work the way it does.

Besides PHP a webserver and the database / database management system we are using:

  • PHPCrawl (see External Data Supply, Extensions, Plug-ins) by Uwe Hunfeld. Supported for crawling / spidering along URLs.
  • The DHTML / JavaScript Calendar by Mihai Bazon. Used in the admin area to enter dates and times.
  • Sitebar Bookmark Server by Ondrej Brablc. We use the translation tool for our translation.
  • Freemind by the Freemind Development Team. Used for creating an overview for this documentation.
  • WebSwoon - The web photographer by Igor Kouzmine. Will probably be used for making screenshots of URLs under Windows
  • Comment Script by Alex. This is used for letting you, the user add comments on the documenation which is provided online.

We will probably be using the following in future:

  • SmartTemplate by Philipp von Criegern. Will be used as direct templating engine.
