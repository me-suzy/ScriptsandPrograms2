$sqlhost Hostname or IP of the MySQL Server
$sqllogin Login/user name for the MySQL database
$sqlpass Password for the MySQL database
$sqldb Database name for the Engine
$table Table name for the main search engine entries
$adstable Table name for the advertisements database.
$enginestable Table name for the meta engines list.
$searchfeedid SearchFeed.com ID for the SearchFeed PPC SE Affiliate Program
$revenuepilotid RevenuePilot.com ID for the RevenuePilot PPC SE Affiliate Program
$webgerusername WebGer.com Username for the WebGer PPC SE Affiliate Program
$numsearchfeed Max number of SearchFeed results to grab
$numrevenuepilot Max number of RevenuePilot results to grab
$numwebger Max number of WebGer results to grab
$rpfamfilt Turn on Family Filter for Revenue Pilot? (on/off)
$purgedays Number of days old an entry in the database must be when it's removed. Set this to 0 to never purge the database.
$mainmod PHP Filename of the main search engine module, usually is index.php
$engtitle **One or Two Word** title for your search engine. (displayed in various places)
$header Header file, that points to a file containing the HTML thats included at the top of each page (e.g., ./head.php)
$footer Footer file, that points to a file containing the HTML thats included at the bottom of each page (e.g., ./foot.php)
$fontface Sets the font used when displaying search results.
$fontsize Font size for search results
$color1 First color used in the search results table
$color2 Second color used in search results table
$textcolor This color is used in text that appears inside the tables for search results
$tablewidth Table Width for search results (full screen is 100%, or you can drop the % to set it in pixel size)
$maxresults Maximum search results to display (only applies to the main engine, not meta results!)
$newwin Setting to "1" opens all clicked search result URLs in a new window ("0" will open in same window)
$showmeta Enables the placement of a 'Show Meta Results' link at the bottom of the main search engine search results. When enabled, if no main search engine results are found, meta web results will automatically be displayed. (1 = on, 0 = off)
$adminpass Password for administration page
$showadvert Shows a keyword advertisement from the textads database. (1 = on, 0 = off)
