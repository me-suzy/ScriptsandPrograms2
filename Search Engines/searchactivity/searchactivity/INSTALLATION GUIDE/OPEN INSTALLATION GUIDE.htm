Installation 1)  Variables to edit in searchbot_config.php

a) Change the variable below to false if you do not want the script to log the SearchBot crawl (kind of defeats the object of the script if you change it, but hey - you might just want to be emailed...)
$logit = true;

b) If the above variable is true and you wish to rename your log file, do so here.  It is recommended that you don't bother changing it...
$logfile = "searchbot_log.txt";

c) If you want to be emailed when the SearchBot has crawled a page on your site - change the below variable to true.  Note:  If your site is popular and is already well listed on Google, you may want to keep it set at false otherwise your server's sendmail program will be working overtime.
$emailit = false;

d) If the above variable is true, enter your email address below, like so:
$youremail = "email@site.com";

e) In the administration page (searchactivity.php) specify the number of crawls to show per page, the default (and recommended amount) is 10...
$perpage = 10;

e) When clearing the file you will be asked for a password, enter your password below - make sure it is original unlike the one below:
$password = "password"; 

2)  Upload

Upload all the files required (see above) into a directory preferably called "searchactivity" so your link to searchbot.php would be www.yourdomain.com/searchactivity/searchbot.php

3)  CHMOD

The log file needs to be written to online, therefore you need to set the access permissions.  For those that don't know; CHMOD is the Unix command and system call to change the access permissions of a file.

You must change the permissions (CHMOD) searchbot_log.txt to 777.  If you do not how to do this try searching on Google or click on the links below corresponding to your FTP program:

 Cute FTP
SmartFTP
WSFTP
