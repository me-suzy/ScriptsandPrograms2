 PhpMyComic is web based program written in PHP and MySQL! With PhpMyComic you
 can collect and organize your comics, and share the information with others!

 INSTALLATION INSTRUCTIONS

 To install PhpMyComic please do the following:

 1. Unzip the zip file to a directory on your web server

 2. Give the image, backup and config directory the permission 0777
    (this is for uploading images, system settings and backup system)

 3. Open your web browser and goto -> www.yourPMCurl/index.php

 4. Just follow the installation instructions.

 5. The install system will give you various options, when
    the installation is done, hit the finnish button and
    you will be taken to the index page and phpmycomic
    is now ready for use!

 7. Log in as the Admin to make further changes.
