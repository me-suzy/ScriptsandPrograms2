 1. INSTALLATION
    - upload all the contents of the HTML directory to the folder you prefer
    - create a MySQL database 
    - modify the db.php file in the includes folder to reflect your MySQL 
      database
    - gp to install/setup.php to create the Administration account for your 
      website.
    - use this coding to access the script using a popup window.



	[ <A class=popup_link onClick="window.open('requests/','NewWin','
	toolbar=no,menubar=no,scrollbars=auto,location=no,directories=no,
	status=no,resizable=false,width=370,height=460')" href="#" 
	value="Requests">Request A Song</A> ]
