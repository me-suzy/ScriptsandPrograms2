Installing EZ-Data 1.0.4

 

1st Step....

Log Into Your phpmyadmin account!

(If you dont know what or where that is.. ask your hosting company. Almost all host have one!)

 

2nd Step...

Click the SQL Tab as shown below

 

 

3rd step...

Copy and paste the contents of the file ez_data.sql

in "Run SQL Query" box.

Then

Click GO

4th step...

Copy and paste the contents of the file ez_counter.sql

in "Run SQL Query" box.

Then

Click GO

 

 

5th step...

Open the includes folder

 

6th step...

Edit the file data1.inc

and the data2.inc in that folder

making sure to put the proper username, password, & database name!

 

7th step...

UPLOAD ALL FILES FROM ZIP TO SERVER!

Then navigate to the web folder where you uploaded the files.

 

Thats it....

If you have any problems... make sure you are using the right username and password for the MySQL server. Double check it with your hosting company.
