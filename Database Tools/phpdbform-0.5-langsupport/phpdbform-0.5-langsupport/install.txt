phpDBform Installation Instructions
===================================

Installing this should be fairly  straightforward.  First,
unpack the file in any directory inside  your  http server
document root. In this packages there are some  files  for
testing,like test.php, test_contact.php and test_type.php.

To test the files that uses a database, there is a 
db_mysql.sql with the tables used.

Note that the only files need for phpdbform works are at
/phpdbform.

Take a look at MANUAL.TXT for learn how to use phpdbform and
phpform.

Instruções de instalação do phpDBform
=====================================

A instalação desta aplicação é bem fácil. Primeiro deszipe
o arquivo em qualquer diretório dentro da raiz do seu 
servidor web. Neste pacote existem alguns arquivos para 
testes, como test.php, test_contact.php and test_type.php.

Para testar os arquivos que usam um banco de dados, existe
o arquivo db_mysql.sql com as tabelas utilizadas.

Note que somente os arquivos em /phpdbform são necessários
para o phpdbform funcionar.

Dê uma olhada no MANUAL.TXT para aprender como utilizar o
phpdbform e phpform.
