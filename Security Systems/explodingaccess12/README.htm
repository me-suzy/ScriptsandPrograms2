ExplodingAccess V1.1 Readme

Requirements: Any webserver running PHP 4.*, with MySQL.

Please read the license before use.

ExplodingAccess creates a member's area for your website, using the same database as your forums. This means you can add and subtract users from the access list, simply by moving them in and out of allowed forum groups. The admin group is automatically enabled, so all forum admins will have access to the restricted area.

Setup is simple. Just open conf.php and configure the marked parameters to match the setup of your site. The $allowedgroup var is case sensitive, so enter it exactly as it appears on the control panel.

Usage is just as easy - include the restrict.php file to any pages which are secret, using the following code: <?php include('restrict.php'); ?> Using full links or just filenames where appropriate. Any sites with restrict.php enabled will require users to be logged in before displaying the content. Your existing site remains untouched - the php script does nothing if a valid login session is found for the user.
