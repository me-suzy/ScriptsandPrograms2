Changelog

0.7.0 (2004-11-22)

  • Checking if current document's content has been saved when changing node in Tree View or using Tabs
  • Next xmlhttprequest bug: "+" chars were replaced with spaces, hacked SimpleTextEditor.submit to fix it.
  • Added keyboard shortcuts: Save Document (ctrl+s), Document Info (alt+1), Edit Content (alt+2).
  • Added "Import content" module.
  • Publishing: an option to publish only specified section.
  • Fixed bug (ie): on management page, when clicking on a node, sometimes node and folder icons were disappearing.
  • Fixed bug (ie): in Tree Template menu branch was not showing completely when opening sub-folders.
  • Some other small javascript bug fixes and improvements, for example publishing: remembers last selected options.

0.6.0 (2004-11-20)

  • Added publishing directly to a directory, sending as zip is still an option.
  • Tree Template: shared folder is now lighter, removed unnecessary files.
  • Tree Template: in left frame in tree menu active document is highlighted.
  • In each document there is dynamically generated by javascript TOC (Table Of Contents) using H2, H3 tags.
  • Tree Template: in each document added "Permanent Link to this page", by using this url, the tree in left frame will open to that node and the requested document will be loaded in right frame.
  • Added windows-1250 charset support.
  • On Management page added "open Edit Content by default" checkbox, when you check it, Edit Content tab will open when you click on a document in Tree View instead of Document Info.
  • .html extensions are no more displayed in Tree View
  • Changelog added

0.5.0 (2004-11-08)

  • First release!
