circeOS_CV is a friendly web application to archive, manage, retrieve
curriculum vitae.

It allows to:

- insert and modify anagraphic data 
- upload curricula documents (word, html, excel, txt, etc) 
- list the curricula
- search curricula setting anagraphic data filters
- search curricula setting words content (search word inside uploaded documents) filters

circeOS_CV, using Oracle TEXT features, is a very powerful application to search words inside every 
type of documents (.doc, .xls, .htm, .txt, etc), enabling the user to easy retrieve interesting curricula.


Requirements: 	OS: indipendent, WEB SERVER: IIS/Apache, DB:Oracle 9i or >
