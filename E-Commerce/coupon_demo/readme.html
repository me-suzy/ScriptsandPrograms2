Below is a quick over view on how to setup the script. To access the the online user Manual click here.

  1. Upload all folder and file to your web site keeping the directory structure.
  2. Make sure that you use ASCCII mode for all the files your upload. Expect for the files in the "img" folder upload them in Binary mode.
  3. Chmod 777 the following file(s) and folder:
  1. "inc" (folder under the admin directory)
  2. "admin_l.php" (folder under the admin directory)
  3. "reg.php" files (In the "coupon" folder)

 

Next you will need to password protect the "admin" directory. This can normally be done via your web server admin panel. If you don't know how to do this contact your hosting company for help.

Now you will need to register your copy to receive an unlocking code. This can be done by going here:

http://repricing.com/regsoft/getsoftware/coupon-gen.php

Once you have your unlocking code you will need to enter it by going to: (This is a one time setup.)

http://yourdomain.com/coupon/index.php

Important: When you enter your unlocking code make sure that you do not have any spaces before or after the code.

After you complete the upload you will need to login in to your admin panel. Located at http://yourdomain.com/coupon/admin/admin.php. (Change "yourdomain" to your own domain name).

Use the Admin panel to setup your coupon information and payment terms.

Your coupon is located at: http://yourdomain.com/coupon/index.php?cn=your_coupon_number
