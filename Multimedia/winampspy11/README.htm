Spyamp Track Display Console - Winamp Spy V1.1 Readme

Released Under The GPL 2 License.

Requirements: A webserver with php 4.3.x, and the pc playing the mp3s must have winamp and the spyamp plugin.

Please read the license before use.

This script shows the currently playing item in winamp, or informs the user if winamp is closed (ie a connection couldn't me made).

It parses the spyamp source code and splits of the relevant information, so no templates can be used, it must be the default spyamp look. (Colours are fine).

If the track length is too long to be accomodated (over 50 chars), it will be chopped, and ... will be added to the end.

Installation is simple. Just edit ticker.php in a text editor like notepad, and modify the marked parameter to point to your spyamp url. Ensure spyamp works correctly before trying to use the script, ie access the page when winamp is running, and ensure it is displayed (ie check the connection all works correctly.)

Then just upload the script to your server after ensuring your edits are correct, and include it on any relevant pages using <?php include('ticker.php'); ?> (or custom code), or alternatively access the ticker.php page directly.

I may release more skins for the script in the future, but I wanted to get the initial release out quickly.
