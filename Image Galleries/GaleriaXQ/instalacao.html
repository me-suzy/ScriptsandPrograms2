GaleriaXQ - Versão 1.1 - em 21/10/2005
Anterior V. 1.0 em 08/10/2005

Descrição:
A GaleriaXQ é um Script em PHP que provê uma Galeria de imagens online muito completa. Possui envio de formulário com Upload, recurso de Destaques, menú e categorias dinâmicas, com Zoon para as imagens e navegação facilitada entre as categorias ilimitadas. Possui autenticação e criação de conta por parte de usuários com dois níveis de acesso e configuração completa registrada em Banco de Dados.

Caracteristicas:

-Design EM HTML e folhas de estilo em separado do código PHP
-Compatibilidade de autenticação e Design com os demais modulos XQ
-Troca de templates através de uma única variável
-Upload, dimensionamento e delete de imagens com apenas um clique
-Textual para a parte inferior das categorias editável em formulário através do banco de dados
-Administração de contas de Administrador com login de acesso
-Sete templates inclusos pronto para o Uso
-Páginação para as galerias individuais ou gerais
-Troca de categorias para as fotos existentes
-Zoon para tamamho real
-Blocos de design para Banners de publicidade ou conteúdo livre que dispende conhecimentos de HTML uma vez que a administração possui editor para tal
-Menú do site editável pela adminitração que torna possível criação ilimitadas de links para páginas internas ou externas e outros
-Modulo Fotodetelas que lista as categorias com navegação nas imagens em tamanho normal.

Atualização em 21/10/2005:
-Incluido o módulo Fotosdetela que lista a categoria e as fotos em tamanho real.
-Para listar uma categoria ou foto de um produto em especial, use a seguinte URL:
URL Absoluta:
http://www.seusite.com.br/galeriaxq/Navegacao.php?x_procate=PainelXQ2
URL Relativa:
Navegacao.php?x_procate=PainelXQ2

Onde "PainelXQ2" deverá constar a categoria para a qual deseja apontar o link (esta categoria já deve existir no banco)


Instalação:
-Crie um banco no seu servidor e atribua um usuário para ele com plenos poderes.
-Use o arquivo da pasta Mysql e popule as tabelas deste banco com ele.
-Crie uma pasta na raiz do seu site com qualquer nome e coloque o conteúdo da pasta GaleriaXQ dentro dela ou envie a pasta GaleriaXQ para a raiz do seu site.
-Abra o arquivo common.php e entre as linhas 11 e 16 entre com a conexão ao banco que foi criado. Por volta da linha 56 edite a variável " $CCProjectStyle " para usar o Estilo de template que deseja.
-Dê autorização CHMOD 777 para a pasta IMG e TEMP ou o sistema não conseguirá enviar as imagens

- Use o seguinte Login para o primeiro acesso:
Apelido: admin
Senha: admin

Design e cores:

Bloco de conteudo bla bla bla
