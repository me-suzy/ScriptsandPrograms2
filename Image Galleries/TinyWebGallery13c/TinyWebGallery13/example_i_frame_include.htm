Top row - please note - iframes have a white border on the right side when using IE - this is the area for the scrollbar
If you want to disable this you have to use SCROLLING=no in the iframe - but you lose the scrolling in the TinyWebGallery!
I recommend to leave white as background of the gallery for the best layout.
I only recommend integration of the Gallery if you have a fix layout - otherwise the iframe part can create problems!!
