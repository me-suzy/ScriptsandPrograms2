De quoi avez vous besoin
-------------

php >= 4.3.0, gdlib > 2.0 et quelques jolies images.

Installation
------------

1. Extraire l'archive dans un répertoire
2. Adaptez le fichier config.php à vos besoins (vous pouvez le modifier à tout moment - Je vous recommande de le faire plus tard ;).
3. Copiez TWG sur le répertoire de votre serveur Internet.
4. Changez les permissions des répertoires 'cache', 'counter', 'xml' en 777 (chmod 777). 
   Vous devriez trouver la commande chmod (ou une fonction équivalente) quelque part dans votre client FTP. 
   vérifiez le 'howto' 1 (sécurité) pour avoir une version plus sûre de TWG!
5. Démarrez TWG en demandant index.php.

-  Si vous avez des erreurs - vérifiez si vous avez correctment défini les permissions d'accès aux répertoires. 
   Sur la page Internet il y a une FAQ et quelques 'howtos' (en anglais) - il est souhaitable de s'y reporter.
   Si TWG ne fonctionne toujours pas, vous pouvez contacter l'auteur sur le forum ou par courriel (en Anglais ou en Allemand).
-  Si les galeries de démonstration fonctionnent vous pouvez copier vos images dans un répertoire situé dans le répertoire 'pictures'.
   (Et effacer mes galeries  ;) - effacez/changez le fichier folder*.txt aussi!)
   TWG cache (stocke) beaucoup de fichiers. Merci de fermer votre navigateur pour mettre à jour le répertoire de cache.
-  Si vous voulez aider TWG merci de renvoyer l'ascenseur, cliquez sur une publicité quelconque  du site Internet ou votez pour TWG (les liens sont sur le site Internet).
-  Si tout fonctionne parfaitement, allez sur le site et lisez les "howto's". 
   Il y a beaucoup de possibilités d'utiliser TWG de manière plus efficace. 

Si vous faites des changements de dimensions d'images dans le fichier config.php, vous devrez effacer les images situées dans le réprtoire 'cache'.

Configuration
-------------

L'essentiel de ce que vous pouvez configurer dans TWG est décrit dans l'aide et dans les 'howtos' 
(Lisez les - il y a beaucoup de choses que vous n'espériez pas!)
Si vous voulez modifier le script: N'hésitez pas! Mais assurez vous que vous avez un lien vers la page Internet de TWG quelque part.
