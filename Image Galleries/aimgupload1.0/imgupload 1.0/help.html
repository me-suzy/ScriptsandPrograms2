User directory size:
This defines how much space each user will obtain for uploading. When the run out of space they will no longer be able to upload images.

Image size limit:
This will restrict the size of images that can be uploaded. For example, if you set the Image size limit to: 500kb, images larger than 500kb will not be uploaded.

Extension validation:
Before using this, you should proceed with caution. This is NOT the safest or suggested method for image validation. Only use this method if you are aware or sure that you will not become attacked for using this method.

Profile editing:
Enabling this will allow users to edit their profile, such as: Edit their e-mail address, username, and password.

Enable registration:
This will allow people to register for uploading accounts and begin uploading away. This is completly optional, and is disabled by default.

Use custom guest message:
This gives you the option to display your own custom message to guests. HTML is allowed, however, do not include <br /> for new lines.

Display log in form:
This will determine if you want a login form to be displayed on the main index page for guests. Custom guest message must be enabled for this to work.

Custom user message:
Similiar to the guest custom message, this displays your own custom message for logged in users.
