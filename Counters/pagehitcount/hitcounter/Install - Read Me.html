DOWNLOAD HERE FOR FREE

This is a really simple free php hit counter script that uses the GD library to generate
a picture that displays the users IP address and the amount of views
the picture / page has had.

This can be used as a hit counter for your site
or as a cool forum signature.

Your host will need php and the GD library installed
to use this Free Hit Counter..


--------------------------------------------

INSTALLATION:

To install this script upload all of the files
to the same directory on your server.

CHMOD 777 counter.txt so that it can be written to, 
then call index.php in your browser.

Then just link to the index.php with the
normal image tags.

thats it!

--------------------------------------------
TIP:

If you upload these files to a folder called pic.gif
you will be able to use it on forums by using the url
www.yourhost.com/pic.gif and it will run the index.php 
and output the image, Useful if the site or forum don't
allow php images or html code.
