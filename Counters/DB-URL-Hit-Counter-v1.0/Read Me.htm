Installation Guide
1. Upload all files to your web server.
2. CHMOD the 'config.php' file to '0777'
3. CHMOD the 'install' Directory to '0777'
4. Access 'http://yoursite.com/path/to/linker/install' from a web browser, and setup the SQL server details
5. Remove the '/install/' folder from your web server to prevent another installation
6. Enjoy the script.

Statistics: Statistics can be viewed by going to: http://yoursite.com/path/to/linker/linker.php?stats.

Note: Details about usage is availabe after install. All you need to do is, whenever you create a link to an external site, add the URL to the external site after the following URL: path/to/linker/linker.php?url=... <- url goes after the the '=' sign, don't forget to include the 'http://'!
