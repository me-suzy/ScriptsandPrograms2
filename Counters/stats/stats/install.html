Maxers stat counter installation

To install-

1. Edit dbinfo.php with your mysql details
2. Create and suitable database and run this sql-

CREATE TABLE `stats` (
`id` int(11) NOT NULL auto_increment,
`date` date NOT NULL default '0000-00-00',
`page` varchar(200) NOT NULL default '',
PRIMARY KEY (`id`)
) TYPE=MyISAM;

Finally you need to add the tracking code to your pages that you wish to track. For this you need two bits of information-

1. where is your stats script, where you have uploaded your script to- example http://www.yourdomain.com/stats/index.php
2. the name of the page you want to keep statistics of- example - http://www.yourdomain.com/index.php

now you put these into the following script

<script src="PAGE-SOURCE?page=PAGE-NAME"></script>

or for the example we have above
<script src="http://www.yourdomain.com/stats/index.php?page=http://www.yourdomain.com/index.php"></script>

The red bits must be included and the black bits are your information
