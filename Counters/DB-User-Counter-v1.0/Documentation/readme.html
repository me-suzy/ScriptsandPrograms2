Welcome to the DB User Counter v1.0 Installation Guide.
Please follow the relevant section below to get started.

New Installations

If you're installing DB User Counter for the first time, then you simply need to follow the instructions below.

Upgrading

There are no previous versions to upgrade from.

Features

Below is a list of features for this script:

Features
  • Real Time Statistics
  • Easy to integrate
  • IP Logging
  • Records user's Last Click
  • Shows online users in the past 15 minutes

Requirements

Please check the below list of requirements you will need to install and use DB User Counter.

System Requirements
  • PHP 4.1.0 or better and MySQL 3.2.3 installed. If you are unsure, you can check with your web host. If your server does not support these, please contact us for an alternative solution.
  • Apache HTTP Server, if available. It may work on others, but it has only been tested on Apache.
General
  • A free SQL Database, with full permissions, along with the host, username and password information
  • 100KB of Web space
  • 100KB free SQL Disk Space

If you are unsure about any of these requirements, please contact your web host for more information.

Installation Step 1: Uploading

You first need to upload all files found in the 'upload' to your web server, and into your chosen directory, we will refer to this as 'http://www.yourdoman.com/online/' in this guide.

Once all files have been uploaded, you need to CHMOD specific files to allow the installer to correctly install the script. The following files/directories need to be CHMOD '777' using your FTP Client:

  • 'config.php'
  • '/install'

Once this is done, you are ready to run the install script.


Installation Step 2: Installing

Simply access 'http://www.yourdomain.com/online/install' using your web browser to access the install script. Once here, simply follow the step by step instructions to help you install the script. Should you run into any difficulty, errors will be provided during the install. If you cannot resolve any errors, please contact us for help.

Important: If the tables used by this script already exist in your chosen database, they will be deleted, and new tables will be created. If you do not wish to overwrite the existing tables, you must select a empty database.


Installation Step 3: Post-Installing

After successful installation, you will be provided with instructions on how to use the script, as well as code to help you integrate the script. The installer script will be locked automatically, for additional security, you may remove the 'index.php' file found in the 'install' directory.

Once you have updated the pages that will use the script, you are ready to go. Stats can be found at http://www.yourdomain.com/online/index.php?stats.

If you require further help, you may visit our Community Forum to get additional help and support. If installation was not successful, please contact us, and we will guide you through the install process to identify errors.
