In the first place, I hope that you excuse my poor English. I'm spanish.
Develooping flash chat is a php based chat script with a flash interface. It requires to work an unix or similar server with php4 (or higher) support. The client can be any browser supporting JavaScript with flash 5 o higher plug-in. Develooping flash chat is free, you can use and modify as you want.

Between its features you'll find:

  • Public and private mesages
  • Alert sound
  • Word filtering
  • Messages review
  • Administration section with banning and kicking
  • Limited html tags support(<b>bold</b> e <i>itálic</i>)
  • Links automatically generated in URLs.
  • Emoticons :) :|;):D:O:o:P:(:S8)
  • Easy setup and configuration

First steps

I was looking for a chat script working under php3 to place in my web. I found one with a flash interface and a simple script whose author was Nicola del Bono. The flash interface was developed by Joost van Brug. This script was originally very simple with only two files (the script
itself and a text file to save messages). It had several erros.
Although the script lacked capacities that I wanted, I thought that it could be a good starting point.I put hands to the work and I added control over the users, hour, private messages, sound, review of messages... everything you have now in your hands.
