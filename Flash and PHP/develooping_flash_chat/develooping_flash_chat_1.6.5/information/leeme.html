Develooping flash chat
Versión 1.6.3
in english


Develooping flash chat es un script de chat basado en php con un interface realizado en flash. Para su funcionamiento requiere un servidor con sistema unix o similar y soporte php4 o superior. Como cliente puede usarse cualquier navegador con soporte JavaScript y el plug-in de flash 5 o superior. Su uso es gratuito, puedes usarlo libremente y modificarlo como desees.

Entre sus prestaciones pueden destacarse:

  • Mensajes públicos y privados
  • Sonido de aviso
  • Filtrado de palabras
  • Revisión de mensajes
  • Administración con "banning" (inhabilitación de usuarios) y "kicking" (expulsión de usuarios)
  • Soporte de algunas etiquetas html (<b>bold</b> e <i>itálica</i>)
  • Vínculos generados automáticamentes en las URLs
  • Emoticonos :) :|;):D:O:o:P:(:S8)
  • Instalación y configuración sencilla

Primeros pasos

Buscando un script de chat que funcionara sobre php3 para colocar en una web, me encontré con uno basado en un interface flash sobre un sencillo script cuyo autor es Nicola del Bono.
El interface flash lo había desarrollado Joost van Brug. El script de php original era muy simple y se valía sólo de dos archivos (el script propiamente y un archivo de texto para guardar los mensajes). Además, el interface flash tenía algunos fallos.
A pesar de que no tenía todas las prestaciones que yo deseaba pensé que podía ser un buen punto de partida para implementar un sistema propio. Me puse manos a la obra y le añadí control de los usuarios, hora, mensajes privados, sonido de aviso, revisión de mensajes... en fín todo lo que ahora tienes entre las manos.
 Espero que te resulte útil.
