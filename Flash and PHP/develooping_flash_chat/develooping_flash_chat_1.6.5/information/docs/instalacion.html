Requisitos en el servidor

Requisitos en el cliente

  • JavaScript

Instalación

  1. Busca el archivo "config.php" dentro del directorio "required" y ábrelo.
  2. Reemplaza el valor de la variable $url="http://www.yourdomain.com/chat/" por el url del directorio "chat" tal y como vaya a quedar en tu servidor (terminado en /).
  3. Sube el archivo "intro.php", "index.php" y el directorio "chat" a tu servidor (justo de forma que el directorio "chat" coincida con la dirección que acabas de configurar en el archivo "config.php"). No subas los archivos .swf en modo ASCII. Si subes en modo ASCII los archivos de flash pueden darte problemas al ejecutarlo.
  4. Asigna los siguientes privilegios a los archivos "msg.txt", "users.txt" y "banned_ip.txt" (dentro de chat/required): el propietario tiene permiso de lectura, escritura y ejecución, y el grupo y el mundo permiso de lectura y escritura (chmod 766). Todos los demás archivos basta con que tengan permiso de lectura y escritura para el propietario y permiso de lectura para el grupo y el mundo (chmod644).
  5. YA LO TIENES FUNCIONANDO. Llama desde tu navegador al archivo "index.php", el que hay justo fuera del directorio chat y prueba su funcionamiento.
  6. Puedes insertar una llamada al chat en cualquier página, insertando esta misma página index.php donde quieras a través de un iFrame.

Personalización del script

En el archivo "config.php" que encontrarás en el directorio "required" hallarás un montón de variables para personalizar el script. Lo primero que deberías hacer es cambiar los valores "admin" que por defecto tienen asignadas las variables $admin_name y $admin_password, por el nombre y password que desees usar para administrar la sala.

  • $text_order : Sentido en que se muestra el texto en el chat. Puede ser ascendente o descendente.
  • $review_text_order : Sentido del texto en la ventana de revisión de mensajes.
  • $delete_empty_room : Permite seleccionar si se borran los mensajes de la sala cuando se queda sin usuarios.
  • $show_without_time : Permite ajustar si se muestra la hora de cada mensaje, o solo cuando el usuario entra o sale de la sala.
  • $password_system : Permite seleccionar si se usa la IP del usuario como método de identificación o una contraseña. Si se desea poder hacer banning (inhabilitación permanente de usuarios), ha de usarse en modo "ip"
  • $correct_time : Diferencia en segundos con el tiempo del servidor. Si el servidor no se encuentra en tu misma zona horaria puedes ajustar este valor para que no muestre la hora del servidor sino la de tu ubicación real. Admite valores negativos y positivos (valor por defecto 0)
  • $chat_lenght : Número de mensajes mostrados en la sala. No uses un valor muy alto para que no cargar demasiado el servidor. (valor por defecto 15)
  • $review_lenght : Número de mensajes mostrados al revisar mensajes desde el chat. El valor ha de ser necesariamente menor que el de la variable posterior (valor por defecto 500)
  • $total_lenght : Número de mensajes guardados en el chat. Cuando se alcanza este valor se conservan solo la cantidad definida en la variable anterior. De esta forma el archivo no puede crecer indefinidamente. Su valor ha de ser necesariamente superior al de la variable anterior (valor por defecto 1000)

Hay también un montón más de variables que puedes ajustar o traducir según tu propio criterio. El significado de esas variables se encuentra explicado en el propio archivo "config.php"

Traducción

Todos los textos empleados en el sistema se encuentran definidos en el archivo de configuración "config.php". Para traducir el chat a otros idiomas basta con modificar los datos correspondientes en este archivo, o reemplazarlo por otro ya traducido.
Develooping flash chat se suministra con interfaces en inglés, español, catalán, euskera, holandés, francés, alemán, húngaro, italiano, portugués, rumano y sueco definidos en archivos "config.php" que podrás encontrar dentro del directorio "languages". Para usarlo en una lengua u otra, basta con colocar dentro de "required" el archivo "config.php" correspondiente.
Si realizas una traducción a otra lengua, por favor, envíame el archivo "config.php" modificado para ponerlo a disposición de otros usuarios.

Personalización de HTML

Las páginas que el sistema muestra son sólo dos (más otras tres de administración). El archivo "intro.php" es en realidad un documento HTML completamente personalizable siempre y cuando respetes los scripts que contiene.
Lo mismo ocurre con el archivo "index.php" dentro del directorio "chat". Si quieres puedes ajustarlo al criterio estético de tu propia web.
Si quieres, también puedes personalizar las páginas de administración (admin.php, adminips.php y adminusers.php). El resto de los archivos son scripts de puro php.

Personalización de Flash

Dentro del directorio "flash_sources" encontrarás los archivo "chat.fla", "count_users.fla" y "private_messages.fla", para editarlos, ajustarlos a tus necesidades y exportarlos de nuevo en formato flash 5. Si no eres muy experto en Flash y Actionscript procura no modificar más que lo imprescindible (esencialmente gráficos, colores y tamaños).
Por supuesto eres libre de modificar todo lo que quieras en los archivos y en los scripts bajo tu propia responsabilidad.
