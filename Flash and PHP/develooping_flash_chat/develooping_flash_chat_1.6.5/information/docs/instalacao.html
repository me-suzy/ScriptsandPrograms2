Requerimentos do Servidor

  • Servidor Unix ou similar
  • PHP 3 ou superior

Requerimentos do cliente

  • JavaScript
  • Flash 4 ou superior

Instalação

  1. Abra o arquivo "config.php". Você pode encontrá-lo no diretório "required"
  2. Substitua a variável $url="http://www.seudominio.com.br/chat/" pela a url onde você colocou o diretório "chat" em seu servidor  ( terminando com /).
  3. Upload o arquivo "intro.php" e o diretório "chat" para seu servidor (o diretório "chat"  deve ser enviado exatamente para o diretório que você assinalou no arquivo "config.php", e o arquivo "intro.php" deve estar um nível acima). Não envie o arquivo "chat.swf" em modo ASCII . Se você enviá-lo em modo ASCII ele terá problemas para funcionar.
  4. Defina os seguintes privilégios para os arquivos "msg.txt", "users.txt" e "banned_ip.txt" (em chat/required/): O proprietário tem permissão para ler , escrever e executar , e o grupo e os demais permissão para ler e escrever (chmod 766). Todos demais arquivos devem ter permissão de escrita e gravação para o proprietário e permissão de leitura para o grupo, e os demais (chmod644).
  5. ISTO É TUDO. AGORA DEVE ESTAR FUNCIONANDO Escreva a url em seu browser (intro.php) e tente .

Customização do script

O arquivo  "config.php" no diretório "required" tem algumas variáveis onde você poderá customizar  o script . A primeira coisa que você deverá fazer é mudar o valor default "admin" das variáveis $admin_name e $admin_password. Digite o novo nome e senha que você utilizará para administrar a sala de chat .

  • $text_order : Ordem de texto para as mensagens na sala de texto (para cima ou para baixo).
  • $review_text_order : O mesmo para a janela de revisão de mensagens.
  • $delete_empty_room : Esta variável define se as mensagens na sala serão automaticamente apagadas quando a sala ficar vazia.
  • $show_without_time : Define se a hora, será mostrada em todas mensagens ou apenas junto a mensagem que mostra que o usuário entrou ou saiu da sala.
  • $password_system : Define se o sistema usará o IP do usuário ou senha como método de identificação. Caso queira utilizar o sistema de banimento esta variável deverá ser definida como "ip"
  • $correct_time : Diferença em segundos entre o seu horário e o de seu servidor . Caso seu servidor não esteja operando no mesmo fuso horário que o seu, você poderá acertar esta variável, para que o script mostre o seu horário de maneira correta.
    . Valores positivos e negativos são admitidos (valor default 0)
  • $chat_lenght : Número máximo de mensagens que são mostrados na sala de chat . Não use um valor muito alto, se você não quiser saturar o servidor.(valor default 15)
  • $review_lenght : Número máximo de mensagens mostradas quando você abre a janela de revisão de mensagens . Este valor deverá ser menor do que o valor a ser definido na próxima variável (valor default 500)
  • $total_lenght : Número máximo de mensagens gravadas pelo chat . Quando o chat alcança este valor , ele conserva apenas o montante definido na variável anterior . Deste modo o arquivo não ficará crescendo indefinidamente. Este valor precisa ser maior que o valor definido na variável anterior (valor default 1000)

Existe também uma porção de variáveis que você pode traduzir ou modificar de maneira a adaptar-se a seu estilo. Os significados destas variáveis são explicados por elas mesmas

Tradução

Todo texto usado pelo sistema é definido no arquivo de configuração  "config.php". Para traduzir o chat para outras línguas, apenas modifique o texto correspondente neste arquivo , ou substitua este arquivo por outro traduzido para outra língua.
O Develooping flash chat vem com arquivos em inglês , espanhol e português , bem como os respectivos arquivos "config.php" dentro do diretório  "languages". Para selecionar a linguagem desejada apenas copie para o diretório "required". O arquivo “config.php” na linguagem desejada.
Caso você traduza o develooping flash chat para outras línguas , por favor me envie o novo arquivo "config.php" de maneira que eu possa incorporá-lo (com seu nome) na versão distribuída .

Customização do HTML

O arquivo "intro.php" no primeiro nível de diretório, é um arquivo HTML que você pode customizar ainda que tenha que respeitar os  scripts que o mesmo contém.
O mesmo ocorre com o arquivo "index.php " no diretório "chat"
Se você quiser poderá customizar as páginas administrativas também.(admin.php, adminips.php and adminusers.php).
O restante dos arquivos são puros scripts de php.

Customização do Flash  

No diretório "flash_source" você pode encontrar o arquivo "chat.fla". Você pode editar este arquivo,  encaixá-lo em suas necessidades  e exportá-lo novamente no formato flash 4 . Se você não é muito experiente em Flash e  Actionscript, tente não modificar mais do que o essencial (tamanho , gráficos e cores ). É claro que você é livre para modificar tudo que quiser nos arquivos e scrpits , sob sua própria responsabilidade.
