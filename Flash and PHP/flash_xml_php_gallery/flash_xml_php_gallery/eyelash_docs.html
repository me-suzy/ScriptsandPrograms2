Eyelash is a Flash powered picture gallery that creates a portfolio of pictures from a php-based XML, taking input from a list of images in the images directory.

Requirements
To work properly eyelash will require you to make sure you have basic knowledge of the following words :
  • PHP (with GD 2.0.1 or later),
  • FTP,
  • JPG,
  • Flash
    • If you don't see the thumbnails, it's probably a php error :
