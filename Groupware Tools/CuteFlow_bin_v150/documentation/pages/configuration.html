Configuration

General

The complete configuration is done through settings in the /config/config.inc.php file.

Needed settings

Parameter Allowed settings Description
$DEFAULT_LANGUAGE DE, EN Choose your language. Available at the moment: german (value: DE) or english (value: EN)
$CUTEFLOW_SERVER URL The http-URL under that the application can be reached (i.e. http://192.1.2.100/cuteflow/). ATTENTION: Don't forget the trailing /

Database Settings
$DATABASE_HOST IP or URL The host Adress of your mysql database server
$DATABASE_DB   The database name of your cuteflow database (i.e. cuteflow)
$DATABASE_UID The user id that can access your mysql server
$DATABASE_PWD The password for the user id

E-Mail Settings
$SMTP_USE_AUTH true, false Use true here if your email server uses Authentification for the SMTP Protokoll, otherwise use false. If using true, you have to provide the user id and password (see below)
$SMTP_SERVER   Your E-Mail Server (i.e. IP-Adress or url)
$SMTP_PORT   Port for SMTP on the server (likely: 25)
$SMTP_USERID   The user id if you use authentificate smtp (Account name)
$SMTP_PWD   The password if you use authentificate smtp
$SYSTEM_REPLY_ADDRESS   You can provide an reply-email adress that is shown in the recipents message.
If you didn't want to allow replys enter an invalid string (i.e. CuteFlow-Replys_not_allowed)

Optional settings

Parameter Allowed settings Description
$SEND_TO_AFTER_DAYS   The amount of days after which the message is send to the substitute of the receive
$DELAY_XYZ   Circulations are arranged into 3 categories. The categorisation is done bei the days in process of the current station (receiver). if the days in process (DIP) is equal or below $DELAY_NORMAL the category is "normal", if DIP is greater than $DELAY_NORMAL and smaller than $DELAY_INDERMIDIATE the message is in intermediate category. If DIP is greater than $DELAY_INDERMIDIATE the document is delayed
$OPEN_DETAILS_IN_SEPERATE_WINDOW true, false The circulation details are opened in a seperate window if true
$SHOW_POSITION_IN_MAIL true, false If true a small overview of the done stations and current station is inserted into the circulation email
$DEFAULT_SORT_COL COL_CIRCULATION_NAME, COL_CIRCULATION_STATION, COL_CIRCULATION_PROCESS_DAYS, COL_CIRCULATION_PROCESS_START, COL_CIRCULATION_SENDER The column for that a circulations are sorted by default
$CIRCULATION_COLUMNS   An array of all columns that should be shown in the circulation view.
