Fields

The field list

To manage the, click on "Fields" in the menu (left-side). After that you will see a list of all available fields (for all circulations).


There you can use the following actions:
Adding a new field to the list
Deleting the selected field
Edit the settings for the selected field

New field / Edit a field

If you add a field or clicked to edit an existing one you will get a page like the following:


The following data must be provided to describe a field:
Fieldname The name of the input field (i.e. "productname"). This field is needed!
Fieldtype The input type of the field. Possible TEXT, TRUE/FALSE, DECIMAL and DATE values. This field is needed!
Standardvalue This field is ignored at the moment. Will be in use in later versions of CuteFlow
Read-Only This field is ignored at the moment. Will be in use in later versions of CuteFlow
