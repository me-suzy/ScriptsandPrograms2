Step 1: Unzip the TRotator.zip file.

If you do not have a software to unzip the TRotator.zip file, you can download a free trial version of WinZip by clicking here

After "unzipping" the TRotator.zip file you should have the following files as shown below. If you do not have a similar structure as the one below, check to make sure you have your zip program preferences set to allow the creation of directories and to preserve case and long filenames.

Files included in TRotator.zip:
• Copyright Notice.html
• Install Instructions.html
• admin.php
• admintemplate.php
• codelock.php
• ez_sql.php
• footer.php
• header.php
• install.php
• quotes.php
• template.php
• util.php

Step 2: Create a new directory on your website named testimonials (or any name you choose).

Step 3: Upload all files into your newly created testimonials directory. Be sure to set the file permissions of the ez_sql.php file to 777.

Step 4: Create a MySQL database. If you do not know how to do this, please contact your web host and they can help you with this.

Step 5: Point your web browser to the install.php file. So if your website is http://domain.com and you uploaded the files to the "testimonials" directory, then you would type http://domain.com/testimonials/install.php in the address line of your browser.

You will be asked for MySQL database information and an admin password. If you do not know your MySQL database information, please contact your web host and they can tell you this information.

After you have successfully installed the script, be sure to delete the install.php file.

Step 6: Access your admin area at http://domain.com/testimonials/admin.php by entering the admin password you chose in Step 5. Click on the "Configure" link to configure the script to your personal settings. Then click on the "Manage categories" link to add specific categories for testimonials (if you are going to have testimonials for several products or services).

Congratulations, you're done :o) For more free PHP and MySQL scripts, be sure to come back to PHPFreebies.com.

================================
Email Templates Information
================================
The email templates found in the admin are used to send customized emails to testimonial submitter and to you, the admin. The following email templates can be edited:

• New quote notification - this email will be sent to you when a new testimonial is submitted (if admin approval is required for all submissions).

• New quote confirmation - sent to the submitter after they submit their testimonial. This email will contain the information they submitted.

To customize emails use the variables below:

[[name]] = Submitter's Name
[[email]] = Submitter's Email
[[company]] = Submitter's Company
[[url]] = Submitter's URL
[[category]] = Category Name
[[quote]] = Submitter's Quote
[[edit]] = Quote details can be edited here
