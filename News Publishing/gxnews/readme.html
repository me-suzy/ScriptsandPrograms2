About GX News

GX News was created for the common webmaster to use to publish site news, using small files, and a small amount of files. I use this same system on Graphic X Web Design, which is what this is derived from. It is very easy to use, but does not include a multitude of functions. You can add news, and display it on your site. That's it, and that's about all I plan to do. (Make sure to stress the plan)

How to Install

Installing GX News is a very simple process. First, edit the file 'config.php' and replace each variable with the appropriate value. Then, save the file and run 'install.php' to actually install the news. This will prompt you to make sure you have completely edited the 'config.php'. If you have not, please make sure to go back and fix it. If you have, click 'Submit' and it will install the news system. Once you have received the success message, close the page and delete 'install.php' immediately! You can now begin using the script.

How to Use

Because GX News is a simple script with really only two features, this section won't take long at all. To begin adding news, point your browser to 'add_news.php' and fill in the necessary information. Make sure to use the password you set in the config file! Then, after adding the news, you should receive a success message, and your entry should show up in both the 'news.php' and 'latest_news_title.php' files. The entire title, body, and date will show up in the 'news.php' file, whereas only the title is shown on the 'latest_news_title.php' file, but that title links to that same entry in 'news.php'. Nifty, eh?

Because no CSS or any customization at all is included, these files should either be customized or included into a different file. To see a working version of GX News, check out Graphic X.
