Help Index

This help section is meant to make sure difficuilty tasks are easier by having a quick reference to use to be able to complete the task. Its also here to get the most out of WB News.

Emoticon Code

A unique code which when using the Editor will be later converted to the Emoticon Image

Emoticon Image

The absolute location to the emoticon image to use.

Category Avatar

The file name of the category image located in the WB News avatar directory.

Category Avatar Link

An absolute link i.e. http://mydomain.com/path/to/myimage.png it can be any file type accessible by the browser.

User Search

The Search user link opens in a popup windows to search for username(s). You can only find usernames that are in a usergroup lower than the group your in.

Theme Name

The name of the theme you want.

Theme Directory

The directory name which is located in the templates directory of WB News.

Backup Format

You can download or view a online text version of each format.

SQL

The SQL (Structured Query Language) format allows for easy backup being core language of the Database. The format is not of MySQL dump version or PHPMyAdmin however using either tool to re-use an old backup of the SQL format should work.

WBSML

The WBSML (Webmobo Structured Query Markup Language) format is an XML format developed by the Webmobo Development Team.

This format will only work with Webmobo Applications the format doesnt use to the Core Database Language which is SQL (Structured Query Language), however since its built for Webmobo Applications you do not require the knowledge of the Table Formats.

This format will only store data records of each tables.
