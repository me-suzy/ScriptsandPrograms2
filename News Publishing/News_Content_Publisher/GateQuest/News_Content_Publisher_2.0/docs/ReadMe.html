[You will need]:

	A PHP enabled host



[Features]:

	Contains secure login system

	Simplistic interface with mouseover help text

	The option to preview your news/content in a popup window before submitting

	Extremely small codebase

	Ability to easily deploy News/Content Publisher in multiple locations throughout your site


   
[Installation]:

	1. Create and name a folder on your web server (ie. News_Content_Publisher)

	2. Upload index.php and the folder entitled "edit"

	3. chmod news.txt to 666

	4. Modify/use the included stylesheet (docs/styles.css) or merge into your existing styles

	5. Open protect.php and, in the Config section, change the username and password

	6. Make certain you retain  at the top of edit/index.php

	7. Place  where you want the news/content to be displayed.

	8. Browse to www.yourdomain.com/News_Content_Publisher/edit/ and log in

	9. Begin editing!


	
[Notes]:

	You do not need to name your folder "News_Content_Publisher" - choose whatever name you like

	The "docs" folder is not mandatory and does not need to be uploaded

	Please feel free to modify this file if you feel it will assist future users with the implementation
