PHP Script Index Installation Instructions

There's not much but a few settings to set, to get started with PHP Script Index. First, open config.php. The first section is for your MySQL info, database name and table names. The table names can be left as they are, or change them if you really need to. The next set of variables are described below:

$sitetitle - This should be a one-word title of your main website.
$main_site_url - This should be the URL to your main website.
$script_index_url - This should be set to the URL where the script index resides.
$adminemail - Your email addressed (used for the setting below)
$emailnotify - Set to "1" to send an email TO YOU when a script is added.
$max_search_results - Maximum number of search results to display. "0" will disable this limit and return all matching results.
$adminpass - Password for administration.

The next paragraph of variables set the design of the script index, and should be self-explanatory. :)

$cat_link_color - The color of category links on the main page.
$empty_cat_link_color - The color of category links on the main page when a category is empty.

Now save config.php and upload the following files to your website: add.php, cat.php, config.php, create_tables.php, foldericon.disabled.gif, foldericon.enabled.gif, in.php, index.php, modify.php, out.php, search.php, setcookie.php, head.php, foot.php, updatecounters.php and view.php

Access create_tables.php through your web browser. This will create the necessary tables in MySQL to store the scripts and the script categories. From there you'll be prompted for further instructions.

Now that the tables are created, we'll set a cookie in your web browser to give you admin priviledges to the script database. Access setcookie.php through your browser, and enter the admin password you set in the config file. Once the cookie setting has been confirmed, you'll see an [X] next to each script listing that gives you the ability to delete that script from the database. Note: Remember to DELETE setcookie.php before continuing.

You're done! Now, for a little insight into how the script works.

The entries are listed in the category pages and search pages in the order of how many hits have been sent in to the site. When a person submits a script, they're given the HTML code for their site to link back to your site. Clicks from that link are tracked in the "hitsin" field in MySQL, and the scripts are ranked accordingly. Scripts that have sent in 0 hits are listed at the bottom.

Those who submit scripts have the option of editing all fields of their listing(s), except the email address, which must stay the same.

updatecounters.php
If for some reason you delete script listings via phpMyAdmin, or somewhere other than inside the script, run updatecounters.php to recount the category counts.

Scripts that are added are added automatically, and the indexes are updated immediately. Scripts that are modified are updated immediately.
