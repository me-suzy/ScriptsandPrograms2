Instructions

If you would like to use one of the variables listed here in your page, you must put it in the template like this:
{$var}

You can also include a file using one of the listed variables, like this:
{include file=$var}
Or, if the file has a .php extension, then like this:
{include_php file=$var}

Template Features

With the Smarty template engine that SimpleDir uses, you can also use if statements and put comments into your templates that won't show up on the actual HTML code.

To comment a part of your template out or to help explain something for further down the line, you can do something like this:

{* commented code *}

These comments can be spread over several lines or just even a part of one line. All you have to do is put a {* at the beginning of the comment and a *} at the end.

Some of the default templates use if statements to incorporate the options that you have chosen into the look of the page. Here is an example:

{if $var == "Y"}
     HTML to show up if the above evaluates to true
{/if}

Here is an example with a multi-step if statement:

{if $var == "0"}
     HTML to happen if $var equals 0
{elseif $var == "1"}
     HTML to happen if $var equals 1
{else}
     HTML to happen if neither of the above conditions are true
{/if}

General Variables

{$adminname} - the administrator's name
{$adminemail} - the administrator's e-mail address
{$allowdesc} - if descriptions are turned on or off (set to 'Y' if descriptions are enabled, set to 'N' if disabled)
{$pend} - if pending status is turned on or off (set to 'Y' if pending status is turned on, set to 'N' if turned off)
{$sdversion} - the version of SimpleDir that you are using
{$self} - the current page
{$sitename} - your site's name
{$siteurl} - your site's URL

Header/Footer Templates

{include file=$headfile} - include the header file
{include_php file=$headfile} - include the header file (use this if it has a .php extension)
{include file=$footfile} - include the footer file
{include_php file=$footfile} - include the footer file (use this if it has a .php extension)

Successful Link Template

{$category} - name of the category the link has been added to
{$linkdesc} - the link's description
{$linkname} - name of the link
{$linknotes} - misc. field in the Add A Link form
{$linkstatus} - the link's current status (set to '0' if pending, set to '1' if approved)
{$linkurl} - the link's URL
{$ownername} - the owner's name

Successful Modify Template

{$anythingelse} - the misc. field in the Modify A Link form
{$name} - owner's name
{$newcat} - name of the new category (if a new category is set)
{$newcatid} - ID # for the new category
{$newemail} - owner's new e-mail address
{$newsitename} - site's new name
{$newsiteurl} - site's new URL
{$oldcat} - name of the old category
{$oldcatid} - ID # for the old category
{$oldemail} - owner's old e-mail address
{$oldsitename} - site's old name
{$oldsiteurl} - site's old URL
