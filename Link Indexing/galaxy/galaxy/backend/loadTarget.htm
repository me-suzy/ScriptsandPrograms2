Welcome to the backend for Galaxy 2004 Link Database By Richard B Mowatt
Please make a selection from the left hand side.
Here is a brief description of what each function will do

SHOW LINK
Show link allows you to make links viewable. All submitted links are initially set to be non viewable.
You must visit this page and make link viewable before it will show up on the link page
To change this option, open up your database and make the "switch" column in the linkInformation table of your database default to the number '1'.

HIDE LINK
This will make links non visible but does not erase any data

DELETE LINK
This PERMANENTLY erases the specified link and all comments related to it.
One this is performed all data pertaining to that link is lost
Use With Caution

DELETE USER
This will PERMANENTLY erase a user from the database
Additionally all comments and links submitted by as well as responses to those links will be deleted
Use with caution!

DELETE COMMENT
This will PERMANENTLY erase a comment from the database
Use with caution!
