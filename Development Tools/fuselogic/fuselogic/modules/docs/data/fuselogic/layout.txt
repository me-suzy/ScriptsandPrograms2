==== Layout ====
When we give command Queue('test3/test','layout_test'), at the end of fuse 'test3/test' execution all the out string than want to be send to browser will be save at FuseLogic layout system with location name "layout_test" and location name "noname". If you want to get the layout contents you can use command getLayout('layout_test');

If you only do Queue('test3/test'), the layout will only be saved at locataion name "noname". You can get the contents of this layout by using comand getLayout() or getLayout('noname').

Layout "noname" always send to the browser at the end of FuseLogic scripts execution.
