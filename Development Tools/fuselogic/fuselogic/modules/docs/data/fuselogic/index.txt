**API**

**index();**

return the FuseLogic Website Root.

**example1:**

if we put fuselogic root in the DOCUEMENT_ROOT.

index() will return '/index.php/'.


**example2:**

if we put fuselogic root in the DOCUMENT_ROOT/some_directory.

index() will return '/some_directory/index.php/';
