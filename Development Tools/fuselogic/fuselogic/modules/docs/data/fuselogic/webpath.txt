**API**

**WebPath(string $module_name);**

$module_name **(optional)** = name of  Module,  if you don't give it will use current module.

This function will return the Web Path of module.

**Example1:**

if module "forum" located in the "DOCUMENT_ROOT/FUSELOGIC_ROOT/modules/openbb", WebPath('forum') will return
**
"/DOCUMENT_ROOT/FUSELOGIC_ROOT/modules/openbb"**
