**API**

**saveLayout(string $contents, string $layout_name);**

$contents = string that want to be save in the layout system.
$layout_name = the name of layout.

To save string in the Layout system.

**Example:**

saveLayout('Hi I am string', 'string1');

echo getLayout('string1'); #will print "Hi I am String"
