====Requirement====
Before you can play arround with FuseLogic you need the following things:

1. Running Apache HTTP Server with "url rewrite mode" enable

Free Download from: [[http://www.apache.org]]


2. PHP running as CGI or as "modules" on Apache

Free Download from: [[http://www.php.net]]

3. Download FuseLogic from:

[[http://www.haltebis.com/fuselogic_download/fuselogic.zip ]]
