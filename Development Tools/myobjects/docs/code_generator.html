Code Generator

Overview

MyObjects Code Generator generates Php 5 classes depending on your database schema file. Database schema files are xml files in Database Description Language format. These files define the technical details of your database tables, relations between your tables and contain information about the logical constraints that you’ve added for your database fields.

Code generator creates seperate classes for each table that is defined in your database schema file. The classes represent the objects that map to a specific table on the MySql server. A seperate class property is defined for each field in the database table. Code generator creates validation code in the setter methods for your fields depending on the logical constraints that you have defined. Getter and setter methods for the related objects that are defined using foreign keys are also created.

Code generator also adds phpDoc style comments in the generated classes. The added comments can be transformed to an API documentation using the phpDoc tool.

Using the Code Generator

After creating your database schema file in one of the ways that are described in this page, you can generate Php 5 classes that will map to your database tables using the code generator.

Classes can be generated using the MyObjects Code Generator in several ways:

  • Using the Web Client
  • Using the Command Line Client
