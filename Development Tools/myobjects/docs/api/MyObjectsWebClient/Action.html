Class Action

Class Overview
Class Overview | Descendants | Properties Summary | Properties Detail | Method Summary | Methods Detail

Abstract Class For Web Actions

This class defines the method handle() that all the Action classes should implement. It also parses the ddl file in object creation that is registered with the session.
