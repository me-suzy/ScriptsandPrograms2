Class Description
DDL Defines methods for parsing, generating ddl files, getting database schema details.
DDLDatabase Represents the databases defined in the ddl file
DDLTable Represents tables defined in the ddl file
DDLField Represents fields of tables defined in the ddl file
DDLData Represents the data model of the fields.
DDLNumericData Represents the data model for the numeric fields.
DDLTextData Represents the data model for the text fields.
DDLEnumData Represents the data model for the enum fields.
DDLEnumValue Represents the values that enum data objects can have
DDLSetData Represents the data model for the set fields.
DDLTimeData Represents the data model for the time fields.
