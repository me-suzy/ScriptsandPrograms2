Class DDLData

Class Overview
Class Overview | Descendants | Properties Summary | Properties Detail | Method Summary | Methods Detail

Represents the data model of the fields.

This class represents the data models for the DDLFields. This class can not be initiated. It's subclasses must be used to create DDLData objects.
