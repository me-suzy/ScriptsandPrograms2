Installation

To install MyObjects on your system extract the downloaded archive file to a web accessible folder. The webclient/tmp folder which holds the database schema files that are currently under developement by the user, should be writable by the web server. You can make this folder writable by typing Chmod 777 webclient/tmp on the installation folder.

MyObjects does not store any user information, database password etc. on the server. This makes MyObjects more accessible, but installing MyObjects webclient on an internet accessible server is not encouraged. The WebClient and Command Line Client should be used on the developers local computer.
