Invalid Value Exception

InvalidValueException’s are thrown by the setter methods when an invalid data is tried to be assigned as an object property. Setter methods can be specialized in the database schema file for the type of validation that is going to be made.

Here is an example: 

try {
    $usr = new User();
    $usr->setUsername("validname");
    $usr->setName("213 21312"); // Exception is thrown here
} catch(InvalidValueException $e) {
    echo "Error in " . $e->getMessage();
}

The getMessage() method for the exception returns the field name that caused the exception. You can use a switch statement in the catch block, to inform the user about the exception in more detail: 

} catch(InvalidValueException $e) {
    switch($e->getMessage()) {
        case 'username':
        // Report error about the user name
        case 'name':
        // Report error about the name
    }
}
