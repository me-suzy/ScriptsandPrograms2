
Bumpy Tables V1.0

=====================

Requirements:

PHP enabled server
Some HTML knowledge

=====================

Features:

Valid XHTML 1.0 Transitional Tables
Custom number of rows and cols
Width and height of the table
Cellpadding and cellspacing
Custom table color, border color and border style
Cell font face, size, color and background color
Cell content alignment
Formatting with CSS

=====================

Setup instructions:

Upload table.php and new_table.php to your server. Make sure they are in the same directory.

Run new_table.php and follow the instructions.
