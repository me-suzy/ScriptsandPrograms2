Class spell_checker

Description
Description | Methods (details)

Simple PHP MySQL Spell checker class.

MySQL SQL to make the word table: 

CREATE TABLE `words` (
   `WORD` char(50) NOT NULL default '',
   PRIMARY KEY  (`WORD`)
  ) TYPE=MyISAM;


You can download the dictionary: TAR.GZ version ZIP version
