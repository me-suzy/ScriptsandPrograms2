Updates:
Alle .php-Dateien auf dem Server durch die neuen ersetzen.

Nehmen Sie die aktuell installierte Version (z.B. 0.102) und entfernen Sie den Punkt
innerhalb der Versionnummer (im Beispiel wird das dann 0102).
Nun haben Sie die benötigte Updateversion. Benutzen Sie nun alle Updateskripts von
dieser Version an bis zur Version 0.130 (in der richtigen Reihenfolge).
Für unser Beispiel wäre das zuerst upgrade_0102_to_0103.php
danach upgrade_0103_0109_to_01110.php usw, bis Sie die Version 0.130 erreicht haben.
Sollte in dieser Reihe ein Skript fehlen, so wurden für diesen Update keine Datenbankänderungen
vorgenommen. Fahren Sie dann einfach mit dem nächsten vorhandenen Skript aus der Reihe fort.

Nachdem Sie auf 0.130 aktualisiert haben, fahren Sie bitte mit der im Hauptarchiv beschriebenen
Methode fort, um auf die aktuellste Version zu gelangen.

Das Archiv enthält keine Updateskripte für Versionen <0.70. Sollten Sie eine so alte Version installiert haben,
so müssen Sie leider eine Neuinstallation durchführen.
