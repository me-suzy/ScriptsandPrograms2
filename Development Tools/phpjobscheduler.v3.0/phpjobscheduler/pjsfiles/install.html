Database Parameters
Database Host:
MySQL host address - localhost is fine if you host with Seiretto
Database Name:
The database must already exist before installation
- this is the same as your username for FTP if you host with Seiretto
Database Username:
Username to access MySQL
- this is the same as your username for FTP if you host with Seiretto
Database Password:
Password to access MySQL.
