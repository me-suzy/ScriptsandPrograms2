1. INSTALLATION DE PORTAIL PHP :
--------------------------------

[ Sous PHPMyAdmin chez votre Provider, a la création de votre user utiliser la fonction MD5() sur le champ US_pwd
Pour crypter votre mot de passe ]

- Pour de plus amples informations : http://www.yoopla.net/portailphp/  ou http://www.portailphp.com

- Decompresser le fichier ZIP.

- Le copier en local ou chez 1 hebergeur via FTP.

- Lancer dans votre navigateur la page install.php (ex : http://monsite.com/install.php)

- Personnaliser le message de la page d'accueil dans le fichier /include/edito/lang-french.php

- Ajouter tous les droits sur le répértoire  /mod_file/upload/ idem pour les photos et les vignettes dans /mod_photos/ .

------------------------------------
2. PERSONNALISATION de PORTAIL PHP :
------------------------------------

- Pour de plus amples informations : http://www.portailphp.com

- Les themes : Vous pouvez créer ou choisir 1 theme existant

- Créer 1 theme : Créer 1 dossier contenat votre thème dans /themes/ puis modifier
  la variable $App_Theme et indiquer le nom de votre dossier, dans le fichier config.php
  qui se trouve ici : /include/config.php
  Vous pouvez personnaliser la feuille de style,les couleurs ( $App_Couleur ), les images
  en utilisant les memes noms de fichiers.

- Vous pouvez faire 1 copier coller du fichiers lang-french.php pour creer et traduire 1 fichier de langues différents
  modifier $App_Langue : dans /include/config.php
