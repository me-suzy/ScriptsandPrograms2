!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!FINAL VERSION!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!

***REQUIREMENTS
- PHP 4.x Webspace 
- A fast webspace server is recommended
- some free webspace 
- safe modus should be off

***INSTALLATION
Extract the downloaded zip file and open the file "config.inc.php".
Change all paths and variables.
Then upload all files to your webserver.

CHMOD (=set attributes) to:
"store" directory: 777
all files in "data" directory: 777
