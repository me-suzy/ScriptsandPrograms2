Installation

Before installing CMScout first make sure that your webserver meets the following requirements:

  • A Webserver running apache 1.3.33 or newer
  • PHP 4.1.0 or newer (4.3.11 or 4.4.1 recommended)
  • MySQL 4 or newer
  • At least 6Mb of space
  • A basic knowledge of how to add a MySQL database and set the permisions (Most hosters have a simple control panel to help with this task)

If your webserver meets the requirements then follow the following steps to install CMScout:

Step 1

Download the latest version of CMScout from the downloads page.

Step 2

Decompress the archive that you downloaded

Step 3

Upload all files to your webserver (Keeping the directory structure)

Step 4

Make sure that the following directories have write access (chmod 777)

  • avatars/
  • cache/
  • images/
  • photos/
  • templates_c/
  • install/templates_c/

Step 5

Goto the url that you uploaded CMScout to. You will automatically be taken to the installation script.

 


Page 2 of the installation script

 

Step 6

Fill out all required information (Marked with a * ) and then click "Install" on the last page.

Step 7

If CMScout was not able to write to config.php you will be shown the following screen:

 


Download Config File

 

Click on the "Download config.php" button to download the configuration file. Upload this file to your CMScout root directory.

Step 8

Delete the "install" directory from your CMScout installation

Step 9

Enjoy!
