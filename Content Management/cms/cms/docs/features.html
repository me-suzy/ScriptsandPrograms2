Features

  • Easy to use and administrate content management system.
  • Valid W3C CSS and W3C HTML 4 design.
  • Advanced search engine using lex's rules.
  • Functionality available for each page include:
    • Printable version.
    • Send that page URL to friend email.
    • Add to favorite list.
    • PDF version (required PHP 5).
    • Show last updated date, and last user edited the content.
    • Page views counter.
  • Powerful WISWING editor for page content.
  • upload/include media collection (graphics, movies, documents, etc...)
  • Dynamic site map.
  • Multi language system.
  • Easy to customize system (messages,template, css file).
  • Admin can add, edit, delete, activate, deactivate, and schedule any page in the system.
  • Fully web based install script.
  • Admin tool enable to edit html META tags, Marquee bar, Backup and restore database (using zipped XML files).
  • Support WAP and RSS technology.
  • Use ADOdb technology, so system is database portable.
  • Shopping cart system where admin can sell online any item browsed in any page (support CashU and PayPal).
  • Improved performance by sending gz-encoded pages to web browsers that support compressed web pages.
  • Harden scripts security by using magic_quotes_gpc, SSL, htmlspecialchars, XSS cleaner, etc…
  • Using HTML extensions to increase noticeable by search engines (required Apache Web Server).
  • Pages can have an interactive section either moderated or not or even restricted to be browsed by administrator only.
  • Administrator can add email form to any page (form fields can be customise by the administrator).
