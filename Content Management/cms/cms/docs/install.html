Installation

  1. Untar/unzip the source package in a directory accessible by your webserver.
     
  2. Edit db.php to fit your database configuration.
     
  3. Start Rama CMS installation process by pointing your webbrowser to:
    http://yourserver/[php-install-path/]/install.php
     
  4. Delete that install.php script (it is risky to keep it there!).
     
  5. Start surfing Rama CMS by pointing your webbrowser to:
    http://yourserver/[php-install-path/]
     
  6. Start admin Rama CMS by login using (Username: admin, Password: admin).
     
  7. (Optional) Edit the file config.php to adjust the parameters to match your environment.

Upgrading:

Remove old scripts/subdirectories except uploads, design, and images directories, then upload the new scripts/subdirectories and run "install.php" script to do an upgrading process.
