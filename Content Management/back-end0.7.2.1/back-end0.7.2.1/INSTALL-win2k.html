Windows 2000 Server Configuration for back-end-0.7.0.x

Pre-requisites

The following configurations have been tested. Other combinations of software may also work, but are not validated. 

 1)    Windows 2000 server with service pack 4 installed
       Windows 2000 professional with service pack 4 installed

   2     PHP Version 4.3.2 (ISAPI)
         PHP Version 4.3.4 (ISAPI)

   3) MySQL Version 3.23.53-MAX
         MySQL Version 3.23.58-MAX

Configure the server

Manual Setup

Create a directory that iis has access to (example below) 

 d:\www\be7

Copy the Unzipped Back-end release into the directory to end up with the following directory structure: 

 d:\www\be7
    class
      contrib
    doc
     public_html
      tables

Using Start - Settings - Control Panel - Administrative Tools - Internet Services Manager create a new website using the web site wizard 

 Description -be7
 IP address  -your ip (say 192.168.0.1)
   port     -80
  Host Header -your header (say be7 - on my intranet I set my hosts file to map my ip to be7)
 Path     -d:\www\be7\public_html

Edit the file d:\www\be7\public_html\config.ini.php as follows: 

;***********************************************************
 
;;;;
;; Site determinant variables
; basedir  = "/var/www/be7/public_html"
; classdir = "/var/www/be7/class" 
;; Windows versions:
basedir = "d:/www/be7/public_html"
classdir = "d:/www/be7/class"

rooturl  = ""

;;;
;; Database variables
DB_Host     = "localhost"
DB_Database = "be70"
DB_User     = "user"
DB_Password = "password"

 
;***********************************************************

Create the database

The SQL scripts have been tested on recent versions of phpMyAdmin or MySQL directly through the command line. MySQL-Front should also work (although people have reported errors).   Any of these should allow you to create the database called "be70" and run the script below 

        d:www/be7/tables/BE_all.sql

 

PHP Configuration

Take a look at test.php through your browser (http://be7/test.php) and note the values of

    Welcome to the Back-End configurator screen should appear

    Press 'Start' and follow the instructions

    When the 'Set up configuration files' screen appears set the following:

    Base Directory d:\www\be7

    Public Directory d:\www\be7\public_html

    Root Directory

    Root Domain localhost

    Root Sub-Domain

    Database host localhost

    Database name be70

    Database user user

    Database password password

    Press 'Set values'

    The 'Initialize the database screen appears'

    Set username and password

    I recommend set sample data checkbox

    press 'Set values'

'Configuration complete' is the last screen and we're done!

If you still have problems

Some people have reported having trouble logging in after BE is installed.  The following is reported to work to resolve this problem:

Change/Add the following to your php.ini


Set your session path to be set to and make sure that the directory exists:
session.save_path= C:\temp

And also ensure that sessions are using cookies:
session.use_cookies = 1
