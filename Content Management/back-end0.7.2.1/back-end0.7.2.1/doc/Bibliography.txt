Back-End Bibliography and Profiles
Documentation

June 17, 2003
by Peter Bojanic

FILES

public_html
    bibliography.php         Main page to display user options for
                             Bibliographies
    categories.php           Browse Bibliographies by Category 
    nominations.php          Form to nominate a person for a Profile
    profiles.php             Main page to display user options for Profiles
    profileimage.php         Display a Profile photo from the database (used by
                             other pages)
    regions.php              Browse Bibliographies by Region
    result.php               Display the results of a search performed using
                             searchProfiles.php
    searchProfiles.php       Search the Profiles by name and other fields 
    thanks.php               Display a thank you message for submitting a 
                             Profile nomination
    viewProfiles.php         Browse Profiles by name of person
    
admin
    BE_bibAdmin.php          Administration page for Bibliography entries
    BE_categoryAdmin.php     Administration page for Bibliography Categories
    BE_professionAdmin.php   Administration page for Profile Professions
    BE_profileAdmin.php      Administration page for Profiles
    BE_pubAdmin.php          Administration page for Bibliography Publishers
    BE_regionAdmin.php       Administration page for Bibliography Regions
        
class
    BE_Bibliography.class    Implementation of Bibliographies and associated 
                             objects
    BE_Profiles.class        Implementation of Profiles and Professions
    BE_ProfileImage.class    Implementation of Profile Photo display
    BE_ProfileUpload.class   Subclass of BE_Upload.class, used for performing
                             uploads and associating them with a specific 
                             Profile
