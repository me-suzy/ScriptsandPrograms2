ZIRCOTE Web Log and Reporting Module for PHPSlash

Setup The SQL Table

MySQL Table Schema

#
# Table structure for table 'activity_log'
#

CREATE TABLE activity_log (
   time datetime DEFAULT '0000-00-00 00:00:00' NOT NULL,
   user_agent text,
   remote_addr text,
   remote_host text,
   uri text,
   os text,
   browser text,
   page text,
   referer text,
   sid text,
   KEY time (time)
);

To create the MySQL Table execute commmand as such:

zircote:~$ mysql --password=mypass mydatabase < activity_log.sql

File Setup

Place the report.php, referrer.php in a directory of your choice as well as the logit.inc. The gif11.phpmay be place in any directory but it also must have the includestatement reflect the proper locations.

Acquiring the data

PHP Pages

At the end of the page to be logged inset the tag:
<?
include ("/your_directory_path/logit.inc");
?>
This will provide the logging of all data relevant to the reporting tool.

Non-PHP Pages

For non PHP pages all data will be extracted via an image tag call. Howerver for certain sections of the data you must declair in the tag. This is the "Pagetitle" Variable. This is done via a form variable ?pagetitle=Web Search Engine this is case sensative.

<img src="http://www.yourdomain.com/images/gif11.php?pagetitle=Web Search Engine" border=0 alt="" height=1 width=1>
