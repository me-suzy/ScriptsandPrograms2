<b style="color:blue"><u>.meta folder explained:</u></b>
<br><br>
Each folder contain a hidden .meta folder, this folder is used by webk for thumbnails generation and storage needs.
Several modules use this folder to store data. For example, The thumbnails generation is handled by preview modules
that store images in the .meta folder.
<br><br>
The .meta folder usually contain 4 things: 
<br><b><br>
<li>Hits counter data
<li>.locked file for password protection
<li>Image thumbnails 
<li>Modules specific data
</b><br>
