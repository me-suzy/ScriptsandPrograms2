<b style="color:blue"><u>Designing preview modules:</u></b>
<br><br>
Preview modules are used to generate thumbnails, right now WEBK have <b>gif</b>, <b>jpg</b>, <b>png</b> and <b>txt</b>. (txt preview module contains aliases for sourcecodes like cpp)

If you want to create a new preview module, you need to add it in webk/include/init.php
Check the existings module to understand how to proceed, further information is coming soon.
