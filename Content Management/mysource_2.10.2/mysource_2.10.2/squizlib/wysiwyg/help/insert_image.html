Inserting an Image (WYSIWYG)

Inserts a picture into the content at the current cursor position.  To insert a picture: click on the 'Insert Picture' icon.  A new window will appear, where you can choose which image to attach.

New Image Click "Browse" to search for the file you want on your hard drive then click "Upload" to upload the file.
Images Attached to this page If you have already uploaded the desired image to that page you will be able to select it from the list.
Other File ID If you have already uploaded the desired image to another page in the site and know the file's ID you can enter it here.
Align Alignment of the images relative to the rest of the text, not the page.
Border Whether the image is to have a border or not, and how thick it should be.
Hspage Horizontal spacing around the image.
Vspage Vertical spacing around the image.
Image ALT text Text that will appear before the image loads and for users using text only browsers.
Image URL If you are sourcing an image on a different website, type in the complete address here including "http://".

While your image is uploading a progress bar appears reading "Uploading -->", which informs you that the image is uploading.  (As shown in the first image below).  Once the image has uploaded the progress bar changes to say "Upload Complete" which shows that the image has been uploaded.

Finally, click "Insert" to insert the image into the page.
