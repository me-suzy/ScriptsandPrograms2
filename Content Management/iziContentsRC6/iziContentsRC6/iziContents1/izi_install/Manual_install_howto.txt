Manual Install in three easy steps.
1. edit /izi_install/config.php
	a. Changing DbServerName, DbName,DbUsename and DbPassword to fit you setup.

2. Copy /izi_install/config.php to /include/config.php
3. Using PhpMyAdmin (or your favorite Db admin tool) run /izi_install/sql/MySQL_izicontents1manual_new_install.sql

Ok four steps:

4. Goto http://yoursite.com/admin/index.php or http://yoursite.com/iziContents/admin/index.php 
	and login with useranme=admin and password=izicontents
