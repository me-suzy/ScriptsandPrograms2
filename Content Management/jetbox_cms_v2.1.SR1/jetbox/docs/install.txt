Please read this document before proceeding with installation.

- INSTALLATION
- Decompress the archive to a local directory on your system. 
- Upload all the files contained in this archive (retaining the directory
  structure) to a web accessible directory on your server or hosting account. 
- Point your browser to the location of Jetbox and follow the onscreen
  instructions. http://your.host/jetbox/
  - Be sure to check system compatibility if you install this software for the
    first time.
