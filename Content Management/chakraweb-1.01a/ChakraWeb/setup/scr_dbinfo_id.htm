Silakan isi informasi tentang basisdata yang anda gunakan. Jika anda tidak memiliki akses sebagai root (karena website anda berada di host virtual misalnya), anda perlu membuat basisdata terlebih dahulu sebelum proses ini. Sebagai patokan, jika anda tidak dapat membuat basisdata melalui phpMyAdmin karena host virtual atau aturan keamanan di mySQL, skrip ini tidak akan bisa membuat basisdata untuk anda. Skrip masih tetap perlu dijalankan, setelah anda membuat basisdata tersebut, untuk membuat tabel-tabel yang diperlukan.

Jika anda tidak mengetahui nama dari host basisdata, nama pemakai, atau password, hubungi ISP (Internet Service Provider) anda. Mereka akan dengan senang hati membantu memberikan informasi yang anda butuhkan.

Ada dua langkah agar ChakraWeb dapat bekerja dengan baik. Pertama sebuah basisdata kosong akan dibuat, kemudian tabel-tabel baru akan dibuat dalam basisdata tersebut.

Jika anda memiliki hak akses root, pilih "Buat basisdata" di bawah ini dan skrip ini akan membuatkan basisdata kosong untuk anda. Jika tidak, anda tinggal mengklik "LANGKAH BERIKUTNYA". Akan tetapi sebelumnya anda harus yakin basisdata tersebut telah dibuat. (Mungkin anda perlu pertolongan ISP anda untuk itu).

Apapun pilihan anda, skrip ini akan membuat tabel-tabel yang dibutuhkan dan mengisinya dengan beberapa data esensial dan contoh-contoh halaman untuk anda.
