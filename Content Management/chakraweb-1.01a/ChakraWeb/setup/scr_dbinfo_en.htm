There are two steps to getting a working ChakraWeb database. First an empty database is created, then it is populated.

If you have root access, check the create the database box and this script will create the empty database for you. Otherwise, just click on NEXT STAGE. If you do not have root access you need to create the empty database manually first.

Either way this script will then create the tables and populate your database for you.
