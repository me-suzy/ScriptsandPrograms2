INSERTING AND MANAGING PAGES

This FORM is used required for the purpose of inserting new pages in the website. A brief descriiption of all the fields is given below:

PAGE NAME: This is field sets the name of the new page.This name would be displayed in the menu bar on the front page and should be unique for each page on the site.

HEADING: This field defines the heading of the new page as would be displayed in the browser when the page is accessed by the user.

PARENT PAGE: This is the most critical field in this form and should be set with utmost care. This field defines the parent page of the new page that is being entered. The new page would be displayed as a submenu item of the parent page on the main menu. In simpler terms the new page would become a sub set or a sub item of the parent page.

DESCRIPTION: This field contains the text that is to displayed on the page.
