ClanWeb translation guidelines

Introduction

This document wad created to give all the people who want to create their own translations for our application a push in the right direction. We want to make sure that every new translation follows a certain template so that anyone can use any translation he or she likes without having to recode the application.

Translation guidelines

Any new file in clanadmin tools must contain this header. The header are supposed to tell the end-user the files name, when the file was created and by who , a email adress and finally the type of license thats implied to that file. In our case it's a GPL license.

Header template


// -----------------------------------------
// $File: lang_main.php
// -----------------------------------------
// $Copyright: (c) ClanAdmin Tools 2003-2005
// $Created: 2005-02-21 by {nickname}
// $email: support@clanadmintools.com
// $License: GPL - See LICENSE for more info
// -----------------------------------------


After this header it's time to present additional information about who that created the translation, for which version it's created, what language it's in, a mail and last modified date.

Additional information


/*********************************************************
Language file to (c)ClanWeb(CW): Version 1.3.4
Language: English
By: {nickname}
Mail: {email}

Last modified: 2004-08-28
**********************************************************/

How to translate

It's very simple to translate the entire application. Just enter your translated word into it's right place.

Example

English

$lang_main = "Main";
$lang_news = "News";
$lang_crew = "Members";

Swedish

$lang_main = "Huvudsida";
$lang_news = "Nyheter";
$lang_crew = "Medlemmar";

Then it's just to translate all the words found in the different files in your language folder. Don't forget to name the folder with the name of the language you've translated into. Example, 'en' for english or 'se' for swedish.
