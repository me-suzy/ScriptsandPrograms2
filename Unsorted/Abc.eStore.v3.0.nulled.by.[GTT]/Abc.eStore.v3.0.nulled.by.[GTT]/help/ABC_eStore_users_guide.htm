ABC eStore 3.0 Users Guide

1. Overview

ABC eStore (later eStore) is a powerful and cost effective e-commerce instrument which helps you to create and manage efficiently your full-featured Internet Stores. The well developed templates system allows changing both design and structure of your e-store without interfering with the code. System's readiness for integration with Payments Gateways makes money receiving process much simpler and fully automated.

The newest technological solutions from the area of multilevel distributed e-commerce systems creation were used in the ABC eStore. It ensures significantly improved quality of the user's interface, searching tools and product presentation at ABC eStore based Internet Stores.

ABC eStore allows you to extend an assortment and increase quantities of goods in you e-store with no restrictions. ABC eStore is one of the most advanced electronic store solutions.

New in the version 3.0: ABC eStore supports well-developed tax and delivery calculation system. ABC eStore enables language management. Customers can add their comments at the checkout. Improved install process and templates design system.

 4. ABC eStore installation instruction


ABC eStore installation does not require special programming skills.

To install your eStore:
1. Go to your hosting provider's administrative panel and create a Database for keeping the eStore data. If you cannot do it yourself just ask your hosting provider so they will do it for you. You will need this Database name to install your eStore.
2. Extract Delivery Package zip file to any folder in your local computer.
3. Create a folder for eStore files at your web-server.
4 Copy all the eStore files from your local computer to your web-server folder which you created at step 3 and run install.php in your browser.
5. Follow step-by-step instruction of the installation wizard.

You can modify your eStore front (design, colors, graphics, pictures) easily. Well-developed template system allows you to perform any necessary changes. This can be performed through changes in files contained in the folder [Design].

6. ABC eStore Storefront Functions

Viewing the Catalogue

Goods in ABC eStore are organized as a single structured catalogue created by an eStore Administrator.

The goods categories catalogue is located on an individual page. It ensures the simplicity and informative character of the catalogue data and minimizes the goods searching time for eStore customers considerably.

Customers can switch the languages if there are any alternate language packs.

Viewing item detailed good description
Every category is presented on a separate page in a price-list form with products' names, descriptions, pictures, prices. The page has a button that places the selected products from this category to the cart - [add to basket]

If there is an extended description of an item, the cell with the brief information has a reference button to open a page with the item's detailed description, its specification and image.

Adding an item (preliminary goods selection)

While browsing through the catalogue pages (categories) a customer can generate an order. To do this he/she should click the button [add to basket] while picking the desired products. After that the customer comes to a page with all the goods he/she chose where he/she can make all the necessary order alterations like changing the quantity of any product, deleting some products or cleaning the basket.

Editing the selected goods in the basket


After collecting goods to the basket a customer goes to the page with all the items he/she has picked while browsing the catalogue.

Here the customer can do the following:
[Ñontinue shopping] - return to the catalogue and continue goods selection and/or further examination of the Web site.
[Recalculate] - recalculate the total price of goods in the basket according to the new quantity of goods.
[Clear cart] - clear the basket/delete all of the products.
[Proceed to checkout] - register as a customer and complete the order.

Ordering
When an order is completed the customer is to register in the ABC eStore by filling in the required registration form. Some fields in this form are mandatory. The information contained in those fields is absolutely necessary for a store manager to process the order. It is also used for customers' identification.

After the registration form is completed correctly, the system generates the order number, places the order information into the store database, notifies store managers about the new order and goes to a Payment Gateway credit card accepting page. The customer also receives the order notification by e-mail. The customer can view his/her order(s) status using the reference [My Orders] at the left-side menu.

Customers' information is stored in the store's database. It is not necessary to fill all the forms every time for a returning customer. A customer can always view and edit his/her information through the [My Accounts] reference.

To import data to the eStore database you should do the following:

  • go to [Import Data] -> [Import Products];
  • click [Browse] and pick a file you wish to import to ABC eStore at your local disk;
  • press [Upload]

After the file is uploaded to your server you will see a successful upload message or an error message.

A list of Worksheets becomes available after the successful file upload. For every Worksheet you can choose a category to import at the dropdown menu. E.g., you have created a category digital photography in administrative panel and youve got an Excel worksheet with appropriate listing, then choose the worksheet with digital photography listing.

If you wish to ban data import from some lists you should just leave the corresponding fields empty.

Press [Import] after you set all of the parameters above.

You can see the import results at the screen. These results contain the information about added or renewed products. If you had a position with the same name in your system all the data for this position will be overwritten by the data from the file imported. You can manage the importing process very flexibly. For instance such settings as fields to be imported, their types, positioning, key fields, obligatory fields could be adjusted during the import.

Note: To import data to a certain category make sure this category exists in your site.

Creating and Editing Product Categories
To create a new category go to [Categories]->[Add Category]. Fill in all the necessary fields and press [Add Category].

Category name - name of the category that will be visible to customers. This category is below - here is the name of a parent category.
Image - picture which you can pick for this category form your local disk in *.jpg, *.gif, *.png format.

After a category is created in the eStore database you will get access to edit category's information and delete category functions. Special indicators show if a category has an Image.

To delete a category press [Delete] in the category string.

Use [Edit] to change the category data. You can change the category information through the edit window. Press [Edit Category] to confirm changes. The modifications are shown in the category list immediately.

Creating and Modifying discounts for products categories NEW!!!

Select [Edit / Delete Category], click on [edit] to start working with discounts for categories.

At the bottom of the page manage category discounts.

To add a new category discount fill in the fields [Input minimum quantity] and [Input discount %], where minimum quantity is the minimum number of purchased products for a discount. After filling in press [Update].

All discounts for a category are displayed at [Category discounts:].

To delete a discount check the box [Delete] and press [Delete] button. To edit a discount click on [edit]. In the field [Input minimum quantity] edit the minimum quantity of the purchased goods from this category for a discount. In the field [Input discount %] edit the discount for selected quantity. To submit changes press [Update].

Creating and Editing Product attributes
To add, edit or delete product attributes go to [Products] -> [Product attributes]. You can add an unlimited number of attribute groups and attributes in it. First of all you must add an attribute group by typing the group name and pressing the [Add] button on the top of the screen. After that you can add and delete the attributes of the created group. For example, if you sell shirts of different sizes you might add a group named 'Size' and attributes named 'S', 'L', 'XL', 'XXL' into it. Now you can change these attributes while adding or editing products. Your customers can select sizes when adding shirts to a shopping cart.

Creating and Editing Product Information
To enter product information into the eStore base you can import data from a local file as described above or use the special form to add products manually.

To add a product manually go to [Products] -> [Add Product]. At the pulldown menu choose a category where you want to add product to. Fill in all the necessary fields, the obligatory fields marked with *. Press [Add Product].

After the products' data have been imported or manually added to the eStore database, you can edit the products information.

Press [View] in a corresponding string to view information about a product.

Press [Edit] in a corresponding string to edit information about a product.

You can do the following in the editing window:

  • change the product name,
  • move the product to another category,
  • change the product description,
  • change the product price,
  • add, change or delete the product image,
  • change the product qty.

Press [Edit Product] to confirm the changes.

Press [Delete] in a corresponding string to delete information about a product.

Additional Functions
Besides the primary functions of managing the eStore data the following functions are also available at Administrative Panel.

Control over Customers
The control over customers function allows you to view the list of customers, detailed information about a customer and his/her orders. To view the list of customers choose the [Service] -> [Customers].

Creating and Modifying discounts for users

This feature enables special pricing policy for every customer you have.

Go to [Service] -> [Customers]. In the field [assigned discount] of the table [customers info] select discount name for a user. [no] is a default option at the dropdown menu, i.e. all users have no discounts by default.

[Add discount:] Create a new discount users category (e.g. Gold user, Silver user etc.), fill in the field [Discount name] and assign discount at the field [Discount %] for this discount category. Press [Add] to submit.

[Manage user discount:] To delete discount categories check the box [delete] and press[Delete] button. To edit discount name and/or discount %, click [edit]. Make necessary modifications by filling in the fields [Discount name] and [Discount %]. Press [Update] to submit changes.

Discount calculation

Once you have set discounts for every user and for each category discounts are calculated as follows: total discount = discount for product category + special discount for user

Example You have assigned Gold user 50% discount to the Customer A and set 10% discount for 5 products of Category 1. If Customer A purchases 5 products from Category 1, he gets 10% discount and then, after entering his details (account name and password), he has to pay 50% of that sum. Suppose normal price of 5 products from Category 1 is $ 500, then total order will be calculated as ($500-$50)*0.5=$225

Order Processing
The Order Processing function allows you to view coming orders, modify an order status (processed or not), enter processing manager's notes. To view the list of orders choose [Service] -> [Orders]. To view detailed order data and modify the order status (processed or nor) click the reference "Order No" against the required order in the list of orders. You will see the order data, payment data and payment status in the window appeared. There you can set the shipment date and/or order status, add comments and/or print a delivery note. To save changes press [Save]. If you wish to save changes and print a delivery note press [Save & Print Delivery Note].

Statistics viewing
To view information about the eStore visits, sales volume and products' popularity rating go to [Service] -> [Statistics].

Sending messages to customers
If you wish to send a message to all customers press [Service] -> [MailList]. Fill in a topic and a message than press [Send Mail]. This letter will be sent to all customers who agreed to receive messages from your server. System Backup
Go to [Options] -> [Back-Up] to backup the eStore database. Press [Backup Now!] so the file with SQL instructions will be created.

Note: Dont forget to back up both database and pictures!

To save the images uploaded into your eStore copy all images from the eStore to your local machine. The products images are stored in the folder [products], the categories images are stored in the folder [categories].

If case of necessity you can restore your data from admin panel. If you dont see the pictures, please, make sure that youve copied the directories with images for products and categories into the eStore webfolder.

Languages editor  NEW!!!

If you want to translate your eStore from English into any other language go to [Options]à[Languages editor].

To create a new language pack fill in the field [Name], select the charset from the dropdown menu. Then press [Add language].

Note: Its preferable to make a copy of the existing language pack so that all notes wont be empty.

All notes are divided into 6 subgroups: admin, admin parseable, admin section, user, user parseable, user section.

The subgroups admin, admin parseable, admin section contain messages for administrative panel. The subgroups user, user parseable, user section contain messages for the storefront.

To start translating select the categories of messages to view (e.g. user) or input the English keywords of the message in the field [Strings whose description or english translation contains following text], then press [Show]. Then click on the message number and youll be redirected to the new window where you can translate or correct the messages.

Here you can make a new subgroup of the messages and move the message to it or to any other subgroups already created.

After applying all necessary changes press [Update]. The window will be closed.

If you want to go back to the languages management click the reference link [Back] at the top of a page.

To enable the created language pack, check the box Enabled and press [Edit].

To delete the language pack press [Delete]. To change the charset select the necessary option at the dropdown menu and press [Edit].

Switching the languages:

You can switch the languages at the top of administrative panel or go to [Options]->[General settings] and select the default language for the eStore (both storefront and admin panel).

Exit from Administrative Panel
To finish working with Administrative Panel correctly press [LogOut].
