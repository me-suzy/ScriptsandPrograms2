Olate Download - Download Management Script
Documentation - Manual

Olate Download has been designed to be easy to use and comes with a secure and powerful administration control panel. This control panel allows you to modify almost every aspect of the script, bar the colours/fonts/etc. This manual will guide you through each section of the administration control panel.

As well as allowing you to submit bug reports or feature requests, development of Olate Download can be tracked at http://www.olate.com/development.

Admin Main

From this screen, you can access every option to control the script. You can also see the username you are logged in as and logout from the admin area via the text near the top right corner of the page. You can return to this screen at any time by clicking on the Administration link near the top left of the page.

Downloads

This section contains three options allowing you to add a new download, edit and existing download and delete an existing download.

Downloads - Add New Download

From this page, you can add a new download to the database. Fill in each field and then click Add to add the file. If you do not have an image or wish to leave out the Custom Fields, leave the Image field blank and no image will be displayed, otherwise, enter the URL to the image location.

Downloads - Edit Existing Download

Select the download you wish to edit and it will be displayed. Click the Update button to make the changes.

Downloads - Delete Existing Download

All available downloads are listed on this screen. Select the download you wish to remove and it will be deleted from the database.

Categories

This section is the same as the Downloads section and allows you to add, edit and delete categories from the database.

Users

To allow multiple people access to the administration control panel, you can add new users in this section. If you select the Master option, it will not be possible to delete this user from the Olate Download admin area. All passwords are MD5 encrypted which means it is not possible to retrieve the password from the database once it has been assigned to the user. MD5 is a one way hashing algorithm which means that once you encode a string with it, it is virtually impossible to decrypt it due to the vast amounts of computing power that would be necessary to do so. Due to this, it is not possible to edit (or retrieve) the password of a user.

Configuration

From here, you can modify a number of general script settings including Number of downloads per page, the script path and whether to enable sorting or not.

You can modify the colours, font sizes and fonts via the style.css file in the /html/css directory.

Languages

This page allows you to view the language currently in use on in the script along with author details and view available languages that you can activate for use in the script.

To install a new language, you simply download the package from the Olate website and then extract and upload the files contained within it, ensuring your maintain the directory structure. If the files have been uploaded correctly, you will see the new download in the drop down menu.

If you are interested in releasing your translation, please see the instructions available on the Olate website.

Other - Updates

This page checks the Olate server for any available updates for Olate Download to ensure that your copy of the script is always up to date. If an update is available, details will be displayed along with a link to allow you to download the update.

Other - Technical Support

This displays details of how to obtain support for the script if you have problems or any questions.
