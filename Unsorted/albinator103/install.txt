Albinator 1.03 - Installation

Installing Albinator is as easy as you could think of. Just unzip 
the downloaded file and put it a directory that is located 
under your webserver. 

You should read the Installation Manual at:
http://www.albinator.com/manual/

You can also download the manual from your Customer's Section.

The directory named install contains a file named install.php please
run that for detailed installation. But before you do that you must
read the online manual.

Your Customer Panel has all the latest utilites and free patches to
download: http://www.albinator.com/user/customer.php

You can email us anytime at: support@mgzhome.com
