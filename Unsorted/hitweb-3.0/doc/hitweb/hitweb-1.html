1. INTRODUCTION

1.1 Description du projet

Ce projet a débuté pour plusieurs raisons :

  • Mon "bookmark" était devenu inexploitable pour rechercher rapidement un site web précis.
  • Je voulais informer les personnes de mon entourage, quand je trouvais un site interessant.
  • Je voulais être informé par d'autre personne, lorsqu'elle trouvais un site cool.
  • Et je voulais apprendre le PHP.

C'est ainsi que je me suis lancé dans la réalisation de ce projet.

Par la suite, j'ai réorienté le projet pour qu'il soit utilisé facilement sur d'autre site web et par d'autre webmaster.

Maintenant j'essaye de l'améliorer, de l'enrichir avec des nouvelles fonctionnalités. (importation des données, templatisation, vérification de lien automatique, etc...)
