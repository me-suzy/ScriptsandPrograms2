4. IMPORTATION

4.1 Importation à partir de fichier CSV

Pour importer des informations dans la base de données, vous devez créer un fichier par table, en respectant correctement l'ordre des champs des tables. Pour avoir plus d'information sur la structure des tables, vous pouvez regarder le fichier hitweb.sql qui se trouve dans le répertoire sql/.

Voici le fichier qui permet de faire l'importation des données.

http://hitweb.free.fr/hitweb/sql/csv2hitweb.php

Il y a deux paramètres à passer le nom du fichier csv, qui doit se trouver dans le respertoire sql/ et le nom de la table correspondant aux informations que vous voulez importer. 

http://hitweb.free.fr/hitweb/sql/csv2hitweb.php?file=categories.csv&table=CATEGORIES -->

Le résultat est immédiat, et vous avez l'information qui est affiché sous cette forme : 

DEBUG : Query = INSERT INTO CATEGORIES VALUES('1','Art','0 ')

DEBUG : Query = INSERT INTO CATEGORIES VALUES('2','Auto/Moto','0 ')

DEBUG : Query = INSERT INTO CATEGORIES VALUES('','Beauté/Forme','0 ')

4.2 Importation à partir du bookmark de Netscape

J'ai travaillé rapidement sur un script qui permet d'importer un fichier bookmark Netscape dans la base de données.
Ce script ce trouve dans le répertoire sql/ de l'application hitweb : bk2hitweb.php.

Il faut avant toute chose mettre le fichier bookmarks.html dans le répertoire sql/ sur le serveur ou tourne l'application hitweb. Ensuite il suffit d'appeler le script bk2hitweb.php. 

http://hitweb.free.fr/hitweb/sql/bk2hitweb.php

ATTENTION !!! ce script n'est pas terminé, donc il ne faut surtout pas recharger la page une fois l'importation terminée, sinon vous relancerez une autre importation du fichier bookmark de Netscape. Je changerai cela prochainement.
