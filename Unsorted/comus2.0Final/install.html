How to install Comus TGP

Before you do anything, check for the latest version of the software. Latest version can be found at http://www.nibbi.net/scripts/comus/.

  1. Unzip archive
    • Admin Folder
      • admin.php
      • blacklist.php
      • categories.php
      • check.list.php
      • ck.gallery.php
      • config.setup.php
      • config.templates.php
      • db_setup.php
      • index.php
      • new.check.php
      • preferred.php
      • view.delete.php
    • Includes Folder
      • Language folder
        • lang-english.php
        • lang-polish.php
        • lang-russian.php
      • archiveinc.php
      • config.fnc.php
      • config.inc.php
      • formcheck.inc.php
      • gnat-tgp-style.css
      • header.php
      • tgpinc.php
    • Templates Folder
      • email_confirm_subject.txt
      • email_confirm.txt
      • email_partner_subject.txt
      • email_partner.txt
      • email_post_accepted_subject.txt
      • email_post_accepted.txt
      • email_post_rejected_subject.txt
      • email_post_rejected.txt
      • html_galtmpl.txt
      • html_recip.txt
      • html_rules.txt
    • Webmaster Folder
      • index.html
      • post.php
    • accept.php
    • install.html (this file)
    • instructions.html
    • License.txt

2. Edit /includes/head.php -

  • Change the base URL to be your domain.
  • Change the Title on the page.. this will be the title for the whole script

3. Upload everything to the root of your website
4. Set permissions (chmod) of the following files to 766 -

  • /includes - config.inc.php
  • /templates - Everything within the folder

5. From a web brower, open up http://yourdomain/admin/ and click on 'Setup' to change settings to fit your site.
6. Open http://www.yourdomain.com/admin/db_setup.php. This will create the MySQL database needed for the script.

Special note to revshare hosted sites....

If your scrpit is having a path error Check for this tag [ $DOCUMENT_ROOT . ] in the script's probelmatic file and replace it with your revshare path like this virtual/revsharehosting.com/var/www/yourdomain.com

That about it for initial setup.

The heart of the site is the "tgpinc.php" and "archiveinc.php" files. These are meant to be includes within your main TGP and Archive pages. To do this, just make your site design and include these files with the code "<?php include ($DOCUMENT_ROOT . "/includes/tgpinc.php") ?>" or "<?php include ($DOCUMENT_ROOT . "/includes/archiveinc.php") ?>". I have included a few template sites for you to work off of.

To view a working example of this script, you can check it out at http://www.tgp-now.com.

A quick word on security

Many people have asked me about securing their admin and includes directories. The way I do it, and I suggest you do to, is use .htaccess and .htpasswd files. For more info on how to do this, just search the web for .htaccess and you will find more than enough information.
