Edit

In this section of the Control Panel, you can easily access previously posted entries to edit or delete. The main table will list your entries. This list can be filtered by using the drop-down lists above the table to filter by weblog, category, status, or date range. The order of display may also be changed.

The main table shows the following columns:

  • Title: The title of the entry. The link will display the entry for you.
  • Edit: The link in this column will allow you to edit the entry. The edit screen is essentially the same as the original Publish screen.
  • Comments: This column lists the number of comments associated with the entry. The "View" link allows you to easily view the comments.
  • Author: The name of the entry author.
  • Date: The date and time assigned to the entry.
  • Weblog: The short name of the weblog to which the entry is assigned.
  • Status: The status to which the entry is assigned.
  • Delete: You may select the comment to be deleted by checking the checkbox in this column.

The Delete button below the table allows you to delete all of the entries that are marked for deletion by having their "Delete" checkbox checked.
