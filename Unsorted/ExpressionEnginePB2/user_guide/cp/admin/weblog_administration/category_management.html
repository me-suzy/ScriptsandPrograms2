Category Management

This section of the Control Panel is for the management of weblog categories. It is where categories and category groups are created, deleted, and preferences are set. Category groups are sets or groups of categories that can be assigned to weblogs. You can create a separate group for each weblog or use the same group on multiple weblogs to ensure that they all have the same available categories.

Main Screen

The main Category Management screen shows a table of all the existing category groups. It lists the category group name and provides links to edit the preferences.

  • Add/edit Categories: Create and edit the categories that belong to the category group.
  • Edit Group: Edit the name of the existing category group.
  • Delete Group: Delete the category group.

The Create a New Category Group link takes you to a screen allowing you to create a new category group.

Create a New Category Group

A link off of the main Category Management page brings up this screen. On this screen you define the new category group.

The only field on this screen is to input the name of the category group. The name must be unique, but spaces are allowed.

Add/edit Categories

This screen allows you to add new categories to the category group or to edit existing categories. The screen lists the existing categories along with their category id number. It also displays the hierarchical relationship between the categories.

Category Group

This box displays the existing categories that belong to this category group. The categories are shown to display the hierarchical parent/child relationship between the categories. The category names are links that will take you to a screen where you can edit or delete the category information.

Category Name

The name of the new category. This is a required field in order to add a new category. The name may include spaces.

Category Image URL

This field is primarily designed to allow for the easy association of an image with a category. You can put in the URL to an image which you can display as you wish in your Templates.

Despite the name of the field it is also possible to store other textual information in this field; a URL is simply the most common choice.

Category Parent

This drop-down list allows you to create a hierarchical relationship between categories. The list dynamically contains all existing categories for this category group. Selecting a parent category means that the new category will be a "child" of the parent in the hierarchy. The "None" option is available and will make the new category a "top level" category with no parent.

In the example screenshot, the "Tech" category will be a child of the "Local" category. It will be at the same level in the hierarchy as the "Entertainment" category.

Edit Category

You may edit an existing category by following the category name link on the Add/edit Categories screen. The edit screen will also allow you to delete the existing category.

Category Name

The name of the new category. The name may include spaces.

Category Image URL

This field is primarily designed to allow for the easy association of an image with a category. You can put in the URL to an image which you can display as you wish in your Templates.

Despite the name of the field it is also possible to store other textual information in this field; a URL is simply the most common choice.

Category Parent

This drop-down list allows you to create a hierarchical relationship between categories. The list dynamically contains all existing categories for this category group. Selecting a parent category means that the new category will be a "child" of the parent in the hierarchy. The "None" option is available and will make the new category a "top level" category with no parent.

Delete Category

This link will delete the selected category. Any child categories of the selected category will automatically be moved up one level in the hierarchy. There is no confirmation on the deletion, so be very careful when deleting categories.

Edit Category Group

This screen allows you to edit the name of an existing category group.

The only field on this screen is to input the edit of the category group. The name must be unique, but spaces are allowed.

Delete Group

The Delete Group link in the main Category Management table allows you to delete the category group. When a category group is deleted, all entries associated with it have the association removed removed. Category group deletion is permanent and irreversible, so be very careful before deleting a category group.

You will be prompted to confirm your decision before you can delete the category group.
