Custom Entry Statuses

This section of the Control Panel is for the management of entry statuses. It is where statuses and status groups are created and deleted. Status groups determine what statuses will be available for each weblog. You can create a separate group for each weblog or use the same group on multiple weblogs to ensure that they all have the same statuses.

Main Screen

The main Custom Entry Statuses screen shows a table of all the existing status groups. It lists the status group name and provides links to edit the preferences.

  • Add/edit Statuses: Create and edit the statuses that belong to the status group.
  • Edit Status Group: Edit the name of the existing status group.
  • Delete Status Group: Delete the status group.

The Create a New Status Group link takes you to a screen allowing you to create a new status group.

Create a New Status Group

A link off of the main Custom Entry Statuses page brings up this screen. On this screen you define the new status group.

The only field on this screen is to input the name of the status group. The name must be unique, but spaces are allowed.

Add/edit Statuses

This screen allows you to add statuses to the status group or to edit existing statuses. The screen lists the existing statuses.

Status Group

This box displays the existing statuses that belong to this status group. Next to the status names are links to edit or delete the status.

Status Name

This is the name for the status. It is what will be displayed when creating a new weblog entry. This is a required field and it may contain spaces or punctuation.

Highlight Color

This optional field allows you to specify a highlight color for this status. This is the color of the status name in the Edit main table. Entries assigned to this status will have their status name in this color. That way you can easily make a specific status stand out in that table.

This field will take standard CSS color values as input. That means you can use color keywords such as red, aqua, and navy; or you could define a specific RGB value in a standard method such as #f00, #ff0000, or even rgb(255,0,0).

Delete Status

This link will delete the status. There is no confirmation on the deletion, so be very careful when deleting statuses.

Edit Status

You may edit an existing status by following the Edit link next to the status name on the Add/edit Statuses screen.

Status Name

This is the name for the status. It is what will be displayed when creating a new weblog entry. This is a required field and it may contain spaces or punctuation.

Highlight Color

This optional field allows you to specify a highlight color for this status. This is the color of the status name in the Edit main table. Entries assigned to this status will have their status name in this color. That way you can easily make a specific status stand out in that table.

This field will take standard CSS color values as input. That means you can use color keywords such as red, aqua, and navy; or you could define a specific RGB value in a standard method such as #f00, #ff0000, or even rgb(255,0,0).

Edit Status Group

The Edit Status Group link in the main Custom Entry Statuses table allows you to edit the name of an existing status group.

The only field on this screen is to input the name of the status group. The name must be unique, but spaces are allowed.

Delete Status Group

The Delete Status Group link in the main Custom Entry Statuses table allows you to delete the status group. Field group deletion is permanent and irreversible, so be very careful before deleting a field group.

You will be prompted to confirm your decision before you can delete the status group.
