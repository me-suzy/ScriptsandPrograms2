Clear Cache Data Files

This section of the Control Panel allows you clear the cache data for the different types of caching that are available.

Main Screen

The main Clear Cached Data Files screen allows you to clear any or all of the different types of data caching.

Choose which type of cache you would like to clear:

  • Page (template) cache files: This lets you clear out page or template caching.
  • Tag cache files: This lets you clear the cached output of individual tags.
  • Database cache files: This lets you clear the database query caches.
  • All cache files: You may also clear all caches at the same time.
