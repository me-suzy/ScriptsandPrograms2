Plugin Manager

This section of the Control Panel allows you to view information about your installed plugins. You can see which plugins are currently installed as well as look at information pertaining to that plugin such as version, author, usage, and more.

Main Screen

The main Plugin Manager screen shows a table of your currently installed plugins. The View Info link leads to a subpage with additional information about the selected plugin.

View Info

This screen will show specific information about the selected plugin.

Name

The name of the selected plugin.

Version

The version number of the selected plugin.

Author

The name of the author of the selected plugin.

Author URL

The URL associated with the plugin's author.

Description

A short description of the plugin purpose.

Usage

This explains how to use the plugin. Usually, it will give a short explanation as well as an example.
