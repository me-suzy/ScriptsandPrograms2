Main Screen

The main PHP Info screen shows you the output of the "phpinfo()" PHP function.

The "phpinfo()" function outputs a great deal of information about your server, server configuration, PHP installation, and more. Many server-side settings, available server modules, PHP configuration settings, and other information are available for viewing.
