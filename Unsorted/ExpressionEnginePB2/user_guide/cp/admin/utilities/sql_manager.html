SQL Manager

This section of the Control Panel allows you to manage your database. You can view information about the database as well as browse the contents, make a database backup, run queries, and more. In many ways this section acts as a stripped down database administration tool such as phpMyAdmin.

Main Screen

The main SQL Manager screen shows a table of your basic database information. In addition, there are a number of links to subpages for various SQL utilities.

SQL Utilities

In addition to displaying basic information about your database, this section of the Control Panel will also allow you to manage your database in many ways.

Manage Database Tables

In this section you can browse the contents of any of your database tables to quickly check data. In addition, you can select one or more tables and perform one of several actions on the table:

  • Optimize selected tables: This will perform an optimization on the table, which can be useful to run occasionally to eliminate database "overhead".
  • Repair selected tables: This performs a standard SQL "repair" on the table.
  • View SQL structure and data: This option will show you the data structure setup used for the selected tables. This shows you the fields, their field type, and other characteristics of the table.
  • Backup selected tables - Text file: This will prompt you to download a text file backup of the selected tables.
  • Backup selected tables - Zip file: This will prompt you to download a Zip file backup of the selected tables. The file will be in the "Zip" archive format.
  • Backup selected tables - Gzip file: This will prompt you to download a Gzip file backup of the selected tables. The file will be in the "Gzip" archive format.

Database Backup

This section will allow you to easily create a backup of your database in one of several formats. You may have the database backup either displayed in the browser or prompt you to download it. You may select to have the backup performed in plain text, the "Zip" archive format, or the "Gzip" archive format.

Database Query Form

This section simply presents a form that you can use to submit any standard database query. Use of this section is only recommended for advanced users since any changes you may make with a query are permanent.

Status Info

Within this area, you can view a table containing various status information about your database. This includes things like the number of bytes sent, the uptime, and more. Typically, this will only be useful for advanced users.

System Variables

Within this area, you can view a table containing various system variables related to your database. These include things like the supported character sets, your table types, and more. Typically, this will only be useful for advanced users.

Process List

This shows a table of your current database processes. Typically, this will only be useful for advanced users.
