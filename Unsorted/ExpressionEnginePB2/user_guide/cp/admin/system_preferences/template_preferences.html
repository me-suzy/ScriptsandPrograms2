Template Preferences

This section of the Control Panel allows you to define general preferences for your Templates.

Main Screen

The main Template Preferences screen shows a table of the preferences. All the template preferences are set via this screen.

Save Template Revisions by Default

If this preference is set to "Yes", then any changes you make to one of your Templates will be saved. This allows you to keep a record of all changes made so that you can easily revert back to an earlier version of a Template if you need to do so.
