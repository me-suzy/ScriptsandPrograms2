Word Censoring

This section of the Control Panel allows you to enable word censoring. It is also where you define the list of censored words.

Main Screen

The main Word Censoring screen shows a short table of the preferences. All the word censoring preferences are set via this screen.

Enable Word Censoring?

You may enable or disable word censoring. If you select "Yes", the system will replace any specified words (see below) in weblog entries or comments with "####". For instance, if you had the word "shucks" in the list, then anytime that word was included in a weblog entry or comment it would be displayed as "####".

Censored Words

Here you define the list of words that you would like to censor. Place each word on a separate line as shown in the screenshot example.
