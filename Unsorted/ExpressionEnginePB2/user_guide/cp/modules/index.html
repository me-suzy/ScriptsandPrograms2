Modules

The idea of using Modules is integral to ExpressionEngine and the Modules section is where you can see your currently installed Modules and change any preferences there may be.

The main page is simply a table listing information about each of the installed Modules:

  • Module Name: This is simply the name of the installed Module. Some Module names will be a link that leads to the User CP for that Module.
  • User CP: Specifies whether or not the Module has a User Control Panel.
  • Description: A short description of the Module purpose.
  • Version: The version of the installed Module.
  • Status: The current status of the Module; whether it is Installed or not.
  • Action: This link will allow you to "uninstall" a Module if the link is Remove. You can install a currently uninstalled Module with Install.

User Control Panels

Some Modules include a small User Control Panel. These "User CP" pages allow you to perform certain actions specific to that Module. For example, in the Referrers Module you may view the referrers or clear the referrers.
