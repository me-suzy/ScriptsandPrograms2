Performance

Although ExpressionEngine renders your web pages incredibly fast, you'll find that there is a direct correlation between page load speed and the amount of dynamic information you build your pages with. The greater the number of tags and variables you use, the more processing cycles the template rendering engine has to perform. Some things like real-time translating of date codes are particularly processor intensive.

For this reason, we highly recommend that you use one or more of the three data caching systems. Doing so will increase the performance of your site and utilize server resources more efficiently.

Please consult the data caching section of the manual.
