Archive Month Links

The weblog "month links" tag allows a list of links like this:

  • October, 2003
  • September, 2003
  • July, 2003
  • June, 2003
  • etc.

Here is the tag syntax:

{exp:weblog:month_links weblog="news" limit="50"}

{month}, {year}<br />

{/exp:weblog:month_links}

Parameters

  • limit="10"
  • weblog="news"

limit=

limit="10"

The number of rows to return.

weblog=

weblog="news"

The name of the weblog to which you want to limit the query.

Variables

  • {month}
  • {month_num}
  • {month_short}
  • {path}
  • {year}
  • {year_short}

month

{month}

Month, long format: September

month_num

{month_num}

Month, numeric: 09

month_short

{month_short}

Month, short format: Sep

path

{path=template_group/template_name}

The URL to the specified template. The month/year part of the link will be added automatically.

For example, this:

<a href="{path=weblog/archives}">{month}, {year}</a>

Will be rendered like this:

<a href="www.yoursite.com/index.php/weblog/archives/2003/12/">December, 2003</a>

Note: The path setting can optionally be enclosed in quotes: {path="template_group/template_name"}

year

{year}

Year, long format: 2003

year_short

{year_short}

Year, short format: 03
