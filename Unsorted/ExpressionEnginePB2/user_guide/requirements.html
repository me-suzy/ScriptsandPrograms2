Requirements

Server Requirements

  • Software
    • PHP Version 4.1 or newer
    • MySQL 3.23.32 or newer
  • Your hosting account must have the following MySQL grant privileges:
    • CREATE
    • ALTER
    • DROP

To take advantage of the image thumbnailing feature your server must support one of these three image manipulation protocols:

  1. GD
  2. ImageMagick
  3. NetPBM

Your server must have XML support in order to send pings and Trackbacks.

Notes

ExpressionEngine does not support PHP versions prior to 4.1.

If you are hosted on an Apache server it needs to be configured with AllowOverride=on in order for the URLs to work with the default system. If that option is not turned on then there are two options:

  1. Manually turn that feature on using an .htaccess file.
  2. In the Control Panel go to Admin > General Configuration and turn on the "Force URL query strings?" option, which will add a standard question mark query strong to the URLs.

If you are hosted on an Apache server and you find that your site links do not work, it means AcceptPathInfo is not enabled. Consult the URL section of the manual for details.
