Integrated Basic Mailing List Tags

ExpressionEngine comes with an integrated Mailing List. To access the Mailing List backed functions, go to the Modules page in your Control Panel. There you will be able to view, search for, and delete email addresses.

To add the mailing list form to any of your pages, use the following tag in any template:

{exp:mailinglist:form}

<p>Join our Mailing List</p>

<p><input type="text" name="email" value="" /></p>

<p><input type="submit" value="submit" /></p>

{/exp:mailinglist:form}

Notes

You do not need to declare the form header or footer. It is done automatically. The only requirement is that you name your input field "email".

The Mailing List is an "opt in" type by default, meaning that when someone adds their email address to the list, they are sent a confirmation email containing a link with an access code. Unless they click the link, their email address will not be added to the list. The opt-in link will expire in 48 hours.

As with all forms, if you have the "secure forms" option enabled in your Security Preferences page it will be much more difficult for automated spam attacks to occur.
