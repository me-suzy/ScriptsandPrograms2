Info
Any of your visitors can see global statistic of your exchange system by going to 'info.php'. The overall statistic on this page is in a real time while the monthly statistic may have a several hours delay.
There is used template 'info.html' for this page. These variables may be used:

#%members%#

Number of members you have

#%i%#

Total number of impressions in you system

#%c%#

Total number of clicks in you system

#%i_1%#
#%i_2%#
#%i_3%#

Total number of impressions in sizes 1, 2 and 3

#%c_1%#
#%c_2%#
#%c_3%#

Total number of clicks in sizes 1, 2 and 3

#%current_month%#

Name of the current month

#%i%#

Total number of impressions in you system in the current month

#%c%#

Total number of clicks in you system in the current month

#%m_i_1%#
#%m_i_2%#
#%m_i_3%#

Total number of impressions in sizes 1, 2 and 3 in the current month

#%m_c_1%#
#%m_c_2%#
#%m_c_3%#

Total number of clicks in sizes 1, 2 and 3 in the current month

#%hourly_graph%#

Complete table with graph displaying today's impressions hour-by-hour

#%month_impressions%#

Complete table with graph displaying impressions in the current month day-by-day

#%month_clicks%#

Complete table with graph displaying clicks in the current month day-by-day

List of all users
Your visitors can see list of all your users in sizes 1, 2 and 3 by going to list.php. If the script is called without variables, it shows banners of size 1, when you want to show banners of size 2 and 3, cal it as list.php?size=2 or list.php?size=3. The list is based on template list.html, these variables are available:

#%size%#

Selected banner size (1, 2 or 3)

#%links%#

Table containing list of users with their banners. Each row of this table is based on template list.txt (see below).

#%prev_next%#

If this list has more than one page, it shows links to previous/next page

To show each row of the table above is used template list.txt. These variables are available:

#%useris%#

Username

#%banner%#

Banner of the user

#%i_m%#

Total number of impressions sent by this account


Toplist

Any of your visitors can see members with the highest number of impressions the current month by going to 'top.php?size=1' or 'top.php?size=2' or 'top.php?size=3'. For this toplist page is used template 'top.html'. In this template you can use these variables:

#%size%#

Exchange size (number 1, 2 or 3)

#%links%#

The top links list - the list is created by template top.txt (see below)

To show each line of the toplist (above) is used template 'top.txt'. There may be used these variables:

#%userid%#

Username of the listed member

#%banner%#

Randomly selected banner of the account

#%url%#

Link URL of the banner

#%i_m%#

Number of impressions displayed by the account in the current month

Referral system Only exchange users can refer new members, not valid for sponsors
If you want to give your members free credits for referring new users, inform them that they need to make a link to your index.php file with their account name. This link will look like: 'index.php?id=accountname' where accountname will be username of the member. The script saves a cookie into the visitor's browser and if the visitor becomes a member, it will automatically add free credits to the referral's account. Also don't forget to set up the number of free credits in your Configuration. In the Configuration you also can set up if these free credits should be added to the referring accounts automatically or manually (= admin can approve or reject it).
The username of a member who sends an impression can be also automatically placed to your logo next to ads and/or to your default banner. Just link these to http://www.yourdomain.com/index.php?id=~USER~. The system replaces the ~USER~ by an username of the user who sends the hit, so if someone clicks on this link and becomes a member, this user gets free credits.

Errors
If any error has occurred (missing field or so), the script shows an error page. This is based on template 'error.html'. These variables may be used:

#%errortext%#

Description of the error

#%adminemail%#

Email of the admin (owner)

Anti-cheat protection  Valid for exchange users only, not for sponsors
In the admin interface you can see accounts which have click ratio lower than a specified number (in %). It usually indicates cheating. Let's say the average click ratio is 3% but some members have this only 0.5%. You can presume they are cheaters.
You also can set maximum number of impressions to count for each member from one IP address daily, so if someone will try to reload the exchange link again and again, the account gets counted only the number of impressions specified by you. However, in the Admin Area you can see members which attempted to cheat by this way. These IP records should be deleted once a day (Daily Job). More info may be found here.

Deferred free impressions  Valid for exchange users only, not for sponsors
These are free impressions which can't be used immediately but in steps. Let's say you give 10,000 free impressions and set that maximum of 1000 of them should be used daily. The script automatically releases 1000 impressions a day so the 10,000 impressions will be used within 10 days.
You can add deferred free impressions to each member individually or add them all users. If you add them all users, the impressions which are just waiting are not displayed on a page of each user but next to the from which is used to add those deferred impressions. Note: If in the days when the deferred impressions are released join some new members, these new members also get the daily portion of the deferred impressions.
