How to create a free (exchange) account

Visitor who wants to become a member should point a browser to 'join.php' and fill in all fields. The script will automatically send an email to both the new member and to admin (optional - it may be enabled/disabled by admin). Then new members should go to the members area to set up ad(s), grab a html code to place to their pages. Once they did it, the script starts to count their impressions.

For the page where a new member can join is used template 'u_join.html'. This template has several variables, these are used only if someone already tried to join (filled in the form and sent it out) but the system rejected it for some reason. The following variables are available:
 

#%info%#

Description of the reason why the application was rejected

#%userid%#

Entered Username

#%userpass%#

Entered Password

#%email%#

Entered Email

#%name%#

Entered Name

#%siteurl%#

Entered URL

Once the new user joined, a page based on template 'u_joined.html' is displayed. In this template may be used these variables: 

#%userid%#

Username of the new user

#%userpass%#

Password of the new user

#%email%#

Email of the new user

#%name%#

Name of the new user

#%siteurl%#

URL of the new user

If the new user has to get an email (it may be enabled/disabled in Configuration, also email is not sent at this time if you accept new users manually), this email will be based on template 'email_u_join.txt' where you can use these variables:

#%userid%#

Username of the new user

#%userpass%#

Password of the new user

#%siteurl%#

URL of site of the new user

#%memberfile%#

URL of your member.php script

If admin has to get an email (it may be enabled/disabled in Configuration), this email will be based on template 'email_admin_u.txt' where you can use these variables:

#%userid%#

Username of the new user

#%userpass%#

Password of the new user

#%siteurl%#

URL of site of the new user

#%memberemail%#

Email of the user

#%adminfile%#

URL of your admin/index.php script
