Special Tools
In folder special_tools may be found two scripts. These are not needed if all goes properly but may be useful if something goes wrong. If you need to use one of them, upload it into your admin directory and open it by your browser.

set_pass.php
Run it if you have forget username and/or password to enter your admin area. It creates a new administrator with all available rights.

fix_tables.php
Run it if some tables which uses Easy Banner Pro have been damaged due a server error. This script tries to fix it.
