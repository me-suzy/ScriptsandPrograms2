Below is a short description of some functions available in the Admin Area.

Configuration
Before you will run your exchange system, you should fill in all necessary fields in Configuration. Be aware while settings of your values. You have a limited space on pages of your members (this space you can set as width and height of the table with exchange links) and if the links of your members will be too long or too big, a piece of the links may be hidden.
If you already have some members and need to change any value marked by * in your Configuration, you should then use function Fix ads. This function corrects all links of your members in compliance with your actual variables. If you will forget to use this function, links of your members stay identical as before.
If you already have members and want to modify categories: If you want to add categories, add it to the end of the listing, never change order of existing categories. The script uses numbers instead of names for categories so if you have (let's say) categories Music,Computers,Kids,TV and then you change it to Music,World,Kids,TV,Computers so all accounts from category Computers get category World. If you really need change order of categories or need remove any category, you then need update category of each account by hand in your admin area.

Default ads
Before you will run your exchange system, you should set up your default links - this function may be found in the Admin Area. These links are displayed if no one account in the necessary category has available credits. You can have different default links for each category. If you will choose to use the same links for all categories, it will be used for all accounts in the relevant exchange size.

Database backup/restore functions
You can use our own function to backup/restore your database or you can use Mysqldump. This is a small program which is installed on most Unix/Linux servers but not at all. It is recommended to use the Mysqldump backup, however if you cannot use this, just uncheck the field and use our own function. Please be patient, it may take up to a few minutes until your database will be saved or restored.
The backup file is saved into your "data" directory, you can download it and use to restore your data if you need it. Please note that the backup file contains only data, not structure of tables. You must have Easy Banner Pro installed to be able restore the database.

Blacklist
You can ban an unlimited number of domains. Members nor sponsors will not be able to use these domains in their URL's.

Tips
If all works fine, you may remove script 'configuration.php' from your server to prevent hackers totally damage your exchange system. You may again upload it if you will need to make any changes.
To optimize your system you should remove inactive users regularly, especially if you have a lot of members.
