Installation

Extract the zip containing Easy Banner Pro. It makes a folder "easybannerpro". You may rename it if you want.

Upload all files onto your server into the correct directories. You have to upload all the files except images in ASCII mode. Don't upload folders "MANUAL" and "special_tools". Don't upload file "rebuild.php' if you don't want to use cron for the Daily Job. Also don't upload file 'update.php' if you do a first time installation. Then create a new empty directory called "userbanners" inside your main directory.

Set permissions for directories and files.
These directories need permission 711:
- directory where your php scripts and other directories will live
- admin directory
These directories need permission 777:
- data directory
- templates directory (it's inside data directory)
- userbanners directory
These files need permission 666:
- All files in the 'templates' directory
- File 'messages.php' in data directory
Setting permissions for other files is not required on most servers.
Note: These permissions are recommended only, if you have a special server configuration, you will need to set up another permissions.

Point your browser to your domain to file 'setup.php'. Now fill in all required fields and hit button 'Install' below. You should see that the script creates some tables and files. Delete setup.php from your server and go to the admin directory to file 'index.php'. Both username and password are 'admin1'. Select item 'Configuration' from the menu on the left and set all required variables and options. You also should change your username and password. Now you can use the script. Create a few test accounts, place html code for the accounts to some webpage to see if the hits are counted and all works properly.

Before you will recruit users, you should set up the regular Daily Job - it contains several actions which must be done every day, otherwise your statistic will be inaccurate.

How to move Easy Banner Pro on a new server/directory
Create a database backup by using out tool (in Admin Area) or by using PHPMyAdmin (data only). Install Easy Banner Pro on the new place and set up all variables, just like when you installed it first time. If your new URL will be another than your old URL, open the database backup in a text editor and replace all occurrences of the old URL by the new one. Save the backup and use it to restore the database by the same way as you created the backup.
