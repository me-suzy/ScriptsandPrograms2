Daily Job

Easy Banner Pro has one job which must to be run on daily basis - Daily Job.
It must be running daily after midnight. It copies statistic of the previous day to the day-by-day and month-by-month statistics, deletes all IP records used in the anti-cheat protection, converts daily portion of the 'Deferred free impressions' to normal free impressions and deletes unpaid sponsor orders older than 7 days. You can run this job also manually in the Admin Area, however it's not recommended.
Please note that it's very important to run the daily job immediately after midnight to have the statistic as accurate as possible.

There are 3 different ways to run this job:
Automatically - just check field 'Run the daily and weekly jobs automatically' in your Configuration. This is a very easy way but it needs additional system resources and it is not recommended. You should never use this way if you have more than 30,000 impressions/day.
By Crontab (Linux) or Task Manager (Windows) - it's the best way but there is needed a little more setting. You can read below how to set a Cron on your server. If you use this way, never check 'Run the daily job automatically' in the Configuration.
Manually - You can run this job manually in the Admin Area.

Using Cron

If you haven't used Crontab before, please go to http://webmasterworkshop.com/guides/crontab_guide.shtml or to http://www.linux-2000.org/tutorial82999.html to read more about this.

You will need to figure out if you will be using lynx or wget to perform the Cron tasks. These are both programs that are commonly available on Linux and Unix servers, but not always. You will need to locate one of these programs on your server. If you do not know where they can be found, contact your server administrator to get that information. You will need the full path to the program (ie /usr/bin/lynx).

Please note that if you set up in your Configuration that time on the server is different from a real time, you must move also the Daily Job. For example if there is +2 hours difference, you have to run the Daily Job after 2 am.

You have to add one of the following lines to your Cron tasks.
It runs this job daily at 0:01 am, it's very important to run them immediately after a midnight.

If you will be using wget, the line is (all between '1' and 'ip'):
1 0 * * * /path/to/wget --spider -q http://www.yourdomain.com/easybannerprodirectory/rebuild.php?word=yoursecretword
If you will be using lynx, the line is (all between '1' and 'null'):
1 0 * * * /path/to/lynx -dump http://www.yourdomain.com/easybannerprodirectory/rebuild.php?word=yoursecretword >/dev/null

You must replace "/path/to/wget" or "/path/to/lynx" with real path to wget or lynx.
You also must replace "http://www.yourdomain.com/easybannerprodirectory/rebuild.php" by a real URL of your rebuild.php file. Also replace "yoursecretword" by your secret word you have entered in the Configuration.

These are only samples. You may want to change the times if you are familiar with Cron.
