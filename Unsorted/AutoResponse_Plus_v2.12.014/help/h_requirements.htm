SYSTEM REQUIREMENTS

The system is Perl based and is designed to run on UNIX or Linux based servers. The following list details the features which your server must have to run Autoresponse Plus successfully.

  • UNIX, Linux or equivalent operating system (not Windows)
  • Your own CGI-BIN directory
  • Perl 5.005 or greater
  • Sendmail
  • Cron jobs
  • Telnet access
  • Ability to forward all e-mails sent to your domain to your main login account - sometimes called "catch all"
  • Ability to set up a .forward or .procmailrc file in your domain to filter incoming e-mails through a Perl program
  • Your web server software (eg Apache) needs to have been set up with DBM file support.

If you are unsure whether your hosting service supports any of these features then contact their support department and ask them. None of these features are unusual and most hosting services should have them.

Note that your browser needs to be Javascript and cookie enabled to use Autoresponse Plus.
