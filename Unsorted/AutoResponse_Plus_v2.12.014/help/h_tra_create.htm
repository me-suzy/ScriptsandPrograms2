CREATING A NEW TRACKING TAG

Overview

  1. Click on Tracking Tags on the menu bar.
  2. Click on Create Tracking Tag on the sub menu bar.
  3. Enter the properties of the new tracking tag. These are explained below.
  4. Click the Create button.

When you click on Create Tracking Tag (Step 2), you will see the Tracking Tags > Create screen where you can enter the properties for the new tracking tag. Here is an explanation of each of the properties.

  • Tag

Enter the new tag here. It can be text or numbers or both up.

  • Description

The description of where the tag will be used (eg the XYZ Weekly E-Zine).

  • Custom subscribe success page URL
  • Custom subscribe failure page URL
  • Custom unsubscribe success page URL
  • Custom unsubscribe failure page URL

These control which of your customized confirmation pages your subscribers will see when they subscribe or unsubscribe to an autoresponder with with this tracking tag.

Check the section Custom Confirmation Pages for more details.

When you have set the properties for the new tracking tag, click the Create button to save the new tag and return to the Tracking Tags screen.
