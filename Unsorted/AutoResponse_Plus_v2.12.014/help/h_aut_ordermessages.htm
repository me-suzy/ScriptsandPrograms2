CHANGING THE ORDER OF FOLLOW-UP MESSAGES

Overview

  1. Click on Autoresponders on the menu bar.
  2. Click on Edit beside the autoresponder to be edited.
  3. Click on the Up and Down links beside the message to be moved.

The order of the follow-up messages can be changed by clicking on the Up and Down links beside the message you want to move.

This is done on the Autoresponders > Edit screen. If you have an immediate message set, you cannot move the second message into first place. Put another way, the immediate message will always be first in the list.

Changing the order of follow-up messages will affect sequences that are currently running. It would be better to set the autoresponder's status to 'Dormant' and wait until all sequences were complete before changing the order of follow-up messages.
