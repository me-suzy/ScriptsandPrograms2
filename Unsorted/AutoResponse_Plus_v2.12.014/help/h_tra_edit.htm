EDITING AN EXISTING TRACKING TAG

  1. Click on the Tracking Tags link on the menu bar.
  2. Click on the Edit link beside the tracking tag to be changed.

When you click on the Edit link, you will see the Tracking Tags > Edit screen where you can change the properties of the tracking tag.

  • Tag
    Enter tag here. It can be text or numbers or both up to a maximum length of 8.
  • Description
    The description of where the tag will be used (eg the XYZ Weekly E-Zine).

When you have entered the tag and it's description, click the Save Changes button to save the changes and return to the Tracking Tags screen.
