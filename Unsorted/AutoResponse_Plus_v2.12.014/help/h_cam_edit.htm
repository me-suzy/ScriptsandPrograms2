EDITING A SUBSCRIBER

Overview

  1. Click on Subscribers on the menu bar.
  2. Find the subscriber you want to edit (perhaps using filtering).
  3. Click on Edit beside the subscriber you want to edit.
  4. Set the properties of the subscriber. These are explained below.
  5. Click the Save Changes button.

When you click on Edit (Step 3), you will see the Subscribers > Edit screen where you can change the properties of the subscriber. Here is an explanation of the fields on this screen:

  • Subscriber's first name

This entry is optional. If you enter a first (given) name for the subscriber, you can use it in follow-up messages with the {FIRSTNAME} tag.

  • Subscriber's last name

This entry is optional. If you enter a last (family) name for the subscriber, you can use it in follow-up messages with the {LASTNAME} tag.

  • Subscriber's full name

This entry is optional. If you enter a full name for the subscriber, you can use it in follow-up messages with the {FULLNAME} tag.

  • Subscriber's e-mail address

You must enter a valid e-mail address here.

  • Tracking tag

If you want a tracking tag to be associated with the subscriber, choose one here.

  • Previous message

This shows the number of the last follow-up message the subscriber was sent. You can set this to any message in the autoresponder's follow-up sequence. Choose 0 to go back to the start - if the first message is immediate then it will be sent immediately when you click the Save Changes button.

  • Status

Choose the status for this subscriber. The choices are

  • Active
  • Suspended
  • User Cancelled
  • Finished
  • Failed.
  • Preferred format

Choose what message format (plain text or HTML) you would like the subscriber to receive.

When you are finished setting the properties, click the Save Changes button. The changes will be saved and you will return to the Subscribers screen.
