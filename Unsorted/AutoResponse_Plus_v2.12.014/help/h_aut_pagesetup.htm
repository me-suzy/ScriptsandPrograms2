CHANGING THE COMMON LAYOUT OF AN AUTORESPONDER'S FOLLOW-UP MESSAGES

Overview

  1. Click on Autoresponders on the menu bar.
  2. Click on Edit beside the autoresponder to be edited.
  3. Click on Page Layout on the sub menu bar.
  4. Change the layout settings as required. These are explained below.
  5. Click the Save Changes button.

When you click on Page Layout (Step 3) on the sub menu bar, you will see the Autoresponders > Edit > Page Layout screen. Here you can work with the common elements of all follow-up messages in this autoresponder.

The body text of each message can be made up of several parts. Here you can choose which of those parts will be used. The order in which these parts will appear in your messages is:

Header
Body text
Footer
Unsubscribe link
Affiliiate link

Here is an explanation of the fields on this screen:

  • Include unsubscribe link

If this box is checked, every follow-up message sent out by this autoresponder will have an "unsubscribe" link at the end. To unsubscribe from your autoresponder, your contact simply clicks on this link. They will then see a confirmation page and will not receive any more messages in the follow-up sequence. The subscriber's status will be set to 'User Cancelled' in the campaign database.

It is recommended that you include unsubscribe links as it would be considered bad netiquette not to offer a way for your subscribers to remove themselves from your follow-up sequence.

  • Include affiliate link

If you are a member of our affiliate program, you can generate revenue by promoting Autoresponse Plus. When this box is checked, every follow-up message sent out by this autoresponder will have a 'Powered by Autoresponse Plus' link at the end. If your contact clicks on the link, they will be taken to our web site. If they purchase from our site within a certain period of time, you will receive commission on all those sales.

You must sign up for affiliate program if you want to take advantage of this feature.

  • Plain message header text

Here you can enter text which will be added at the top of each plain text follow-up message sent out by this autoresponder. Each follow-up message has it's own option for choosing whether this header should be included or not. You can type directly into the box or copy and paste from another application.

  • HTML message header code

HTML code which will be added at the top of each HTML follow-up message sent out by this autoresponder. Each follow-up message has it's own option for choosing whether this header should be included or not. You can type directly into the box or copy and paste from another application.

  • Footer text

Here you can enter text which will be added at the end of each plain text follow-up message sent out by this autoresponder. Each follow-up message has it's own option for choosing whether this footer should be included or not. You can type directly into the box or copy and paste from another application.

  • HTML message footer code

HTML code which will be added at the end of each HTML follow-up message sent out by this autoresponder. Each follow-up message has it's own option for choosing whether this footer should be included or not. You can type directly into the box or copy and paste from another application.

When you are finished making changes, click on the Save Changes button. You will be taken back to the Autoresponders > Edit screen for this autoresponder.
