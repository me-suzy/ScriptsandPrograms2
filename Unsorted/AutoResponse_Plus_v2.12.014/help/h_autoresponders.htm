AUTORESPONDERS OVERVIEW

To work with your autoresponders, click Autoresponders on the menu bar. At any time and from any location, you can click Autoresponders on the menu bar to go to your list of autoresponders.

When you do this, you will see a list of all the autoresponders you currently have set up.

There are four main autoresponder actions:

  • Create a new autoresponder
  • Edit an existing autoresponder
  • Test an autoresponder
  • Delete an autoresponder
  • Create a new autoresponder

To create a new autoresponder on your system, click the Create Autoresponder link on the sub menu bar.

  • Edit an existing autoresponder

To edit an existing autoresponder, click the Edit link beside the autoresponder you would like to edit.

  • Test an autoresponder

Testing allows you to check the format and layout of all messages which make up an autoresponder's follow-up sequence without having to wait for a live sequence to complete.

When you test an autoresponder, all it's follow-up messages are instantly sent to the e-mail address you entered in the Your Profile screen You can enter your e-mail address by clicking Your Profile on the menu bar. To test an autoresponder, click the Test link beside the autoresponder to be tested.

  • Delete an autoresponder

When you delete an autoresponder, all it's messages, header, footer and all other files it uses are permanently deleted and cannot be recovered. To delete an autoresponder, click the Delete link beside it in the list. You will then be asked to confirm that you really want to do this.

Deleting an autoresponder does not delete any subscribers who used it, although current follow-up sequences will not continue.
