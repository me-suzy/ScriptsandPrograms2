EDITING AN AUTORESPONDER

Overview

  1. Click on Autoresponders on the menu bar.
  2. Click on Edit beside the autoresponder you want to edit.

When you click on the Edit link (Step 2), you will see the Autoresponders > Edit screen for the selected autoresponder. It shows a list of the messages which make up this autoresponder with a set of actions for each. These actions are explained below.

The sub menu bar contains some options for working with the autoresponder properies, layout etc. These are also expalined below.

Each follow-up message has four actions:

  • Edit

Click this link to edit this follow-up message and it's properties.

  • Delete

Click this link to delete this message from the follow-up sequence.

  • Up

Clicking this link will move the message up one place in the follow-up sequence. You cannot move the second placed message up if the first message is set as immediate.

  • Down

Clicking this link will move the message down one place in the follow-up sequence. You cannot move the first message down if it is set as immediate.

The following options are available on the sub menu bar of the Autoresponder > Edit screen:

  • Return to Autoresponder List

Clicking this link will return you to your list of autoresponders. This has the same effect as clicking Autoresponders on the menu bar.

  • Add Message

When you click this link, a new message will be added to the end of the follow-up sequence and you will be taken to the Autoresponders > Edit Message screen for that new message. There you can edit the new message and it's properties.

  • Properties

This takes you into the Autoresponders > Edit > Properties screen for the selected autoresponder. The properties page is very similar to the page of settings you entered when you initially created the autoresponder.

  • Page Layout

Clicking this link will take you to the Autoresponders > Edit > Page Layout screen for the selected autoresponder. You can set the general layout of the follow-up messages that make up this autoresponder. You can define the header and footer and choose whether to include an unsubscribe and/or your affiliate link in each message.

  • Generate Code

This generates the subscription code for pasting into your web pages, e-mails, e-zines, e-books, ads etc.
