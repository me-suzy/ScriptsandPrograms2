AUTOMATIC SUBSCRIPTION CODE GENERATOR

Overview

  1. Click on Autoresponders on the menu bar.
  2. Click on Edit beside the autoresponder you want to work with.
  3. Click on Generate Code on the sub menu bar.
  4. Choose the options for the code you need. These are explained below.
  5. Click the Generate Code button.

Contacts can subscribe to your autoresponders in several ways:

  • Completing a form on your web page
  • Clicking a mailto link on your web page
  • Clicking a mailto link elsewhere (eg e-zine ads, e-books etc)
  • By sending a blank e-mail to your autoresponder's listening address
  • ...and others

However, these can be categorized into two methods: web form and e-mail subscriptions.

Autoresponse Plus can generate the code needed for both these methods.

When you click Generate Code (Step 3) on the sub menu bar, the Autoresponders > Edit > Generate Code screen appears where you need to complete some details about the code you want the system to generate for you.

  • Ask for first name

This applies only to subscriptions taken by a form on your web page. If you want to capture your contact's first (given) name, check this box.

  • Ask for last name

This applies only to subscriptions taken by a form on your web page. If you want to capture your contact's last (family) name, check this box.

  • 2nd and 3rd autoresponders

These only applies to web based forms. If you choose autoresponders here, your subscriber will be signed up to these autoresponders as well. This would be used for example, to sign someone up to a course and also add them to your e-zine list at the same time.

  • Tracking tag to include

This applies only to subscriptions taken by a form on your web page. If you want to track subscriptions from several places, you can set up tracking tags. When someone subscribes to your autoresponder, the system will store the tracking tag along with their contact details in the database. This allows you to analyze the source of your subscribers. Before you can choose a tracking tag from this list, you need to have created one.

  • Offer choice of format

This applies only to subscriptions taken by a form on your web page. You can then offer your contact the choice of receiving plain text or HTML messages by clicking a radio button on your signup form.

  • Link text

If you are creating mailto links for web pages, you can type here what the link text should say (eg Click here to subscribe).

When you have set the options, click the Generate Code button. You will then be given three boxes from which you can copy the code, as required.

If you need to change anything, simply alter the settings and click the Generate Code button again to recalculate the code.
