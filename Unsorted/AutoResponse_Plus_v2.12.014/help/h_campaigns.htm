SUBSCRIBERS OVERVIEW

A subscriber is normally created when someone subscribes to one of your autoresponders. The subscriber record contains all the contact data and keeps track of the follow-up sequence. You can also insert subscribers manually.

To work with your subscribers, click Subscribers on the menu bar. At any time and from any location, you can click Subscribers on the menu bar to go to your list of subscribers.

When you do this, you will see a list of all your subscribers.

There are ten main subscriber actions:

  • Filtering the subscriber list
  • Clearing the current filter
  • Manually adding a new subscriber
  • Editing a subscriber
  • Deleting a single subscriber
  • Deleting subscribers by filter
  • Exporting subscribers by filter
  • Changing subscriber status by filter
  • Importing subscribers
  • One-off mailings by filter
  • Filtering the subscriber list

The list of subscribers can grow quite large. You can filter the list to view only those subscribers that match criteria based on

  • autoresponder
  • tracking tag
  • subscriber status
  • subscription date
  • duplicate e-mail address
  • Clearing the filter

This returns to the full, unfiltered subscriber list.

  • Manually adding a new subscriber

A subscriber is usually inserted automatically when someone subscribes to an autoresponder through e-mail or a form on your web site. However, you can insert subscribers manually.

  • Editing a subscriber

Subscribers can be edited to, for example, reset their follow-up sequence or change their name or e-mail address.

  • Deleting a single subscriber

You can permanently delete subscribers individually.

  • Deleting subscribers by filter

Using the subscriber filtering features, you can set the criteria for groups of subscribers which you wish to remove permanently from the system.

  • Exporting subscribers by filter

The system has an exporting feature which allows you to export subscriber contact data in a standard comma separated variable (CSV) format. The exported data can then be imported into contact management systems, databases and other applications. Filters can be set so that you can easily choose the subscribers you want to export.

  • Changing subscriber status by filter

Using the subscriber filtering features, you can set the status (eg Active, Suspended) of groups of subscribers.

  • Importing subscribers

If you have contact data in comma separated variable (CSV) format, this can be imported into the system. Subscriber records will automatically be created for the imported contacts and follow-up sequences will run just as for subscribers inserted in other ways.

  • One-off mailings by filter

Using the subscriber filtering features, you can send one-off e-mail messages to groups of your subscribers. These messages can be in plain text or HTML format and can be personalized. This feature could be used for sending e-zines or one-off sales offers. Sending a one-off e-mail does not affect currently running follow-up sequences.
