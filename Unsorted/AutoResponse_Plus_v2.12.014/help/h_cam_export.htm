EXPORTING SUBSCRIBER DATA

Overview

  1. Click on Subscribers on the menu bar.
  2. Set the filter for the subscribers you want to export (or click Clear Filter on the sub menu bar for all subscribers).
  3. Click on Export on the sub menu bar.
  4. On the export screen, click on the Confirm Export button.

When you click Confirm Export, the data will be exported to the file 'export.csv' in your 'cgi-bin/arplus/data' directory. The exact data directory can be found by clicking on Setup on the menu bar.

The format of the exported data will be:

"first_name","last_name","full_name","email_address"
"first_name","last_name","full_name","email_address
"first_name","last_name","full_name","email_address"

Note that if the file 'export.csv' already exists, it will be overwritten.

When the file has been created, you can copy it across to your local computer using your FTP client in ASCII mode.

EXAMPLE export.csv FILE
"John","Smith","John Smith","js@jsdomain.com"
"","","Fred Bloggs","fb87689@hotmail.com"
"William","","","william9102837@aol.com"

When you click Confirm Export, please wait for confirmation that the process is complete before taking any further action. This may take a few minutes with long lists.
