CREATING TIME-SENSITIVE MESSAGES

Overview

  1. Click on Autoresponders on the menu bar.
  2. Click on Edit beside the autoresponder you want to edit.
  3. Click on Edit beside the follow-up message you want to edit.
  4. In the message body, insert the tags described below.
  5. Click the Save Changes button.

The system has a feature for creating time-sensitive messages of the form:

"If you purchase our widget before midnight on Thursday 4 January 2001, you will receive 5 FREE bonus reports worth $199."

In this example, the date 'Thursday 4 January 2001' was calculated by Autoresponse Plus by adding 5 days onto the current date. The calculation is carried out on the day the message is sent, not the day the message is created.

Time-sensitive messages are created using special tags that you insert in:

  • message subjects
  • plain or HTML message body text
  • headers and footers
  • one-off mailings

The date tags that can be used are:

  • {SHORTDATE offset}
  • {LONGDATE offset}
  • {USDATE offset}
  • {EUROPEANDATE offset}

The date tags are case sensitive and they should always be typed in capital letters.

  • SHORTDATE
    Displays the date in the format 3 Dec 2000
  • LONGDATE
    Displays the date in the format Sunday, 3 December 2000
  • USDATE
    Displays the date in the format mm/dd/yyyy
  • EUROPEANDATE
    Displays the date in the format dd/mm/yyyy

The offset value is optional and will add the specified number of days to the date before it is inserted into the message. The date that will be displayed will be offset from the date the message was sent, not the date the message was created.

Here is an example message:

EXAMPLE MESSAGE BODY TEXT
Dear Friend

Thanks for requesting more information on our widget product. You will find all you need to know in this e-mail.

If you order it before midnight on {LONGDATE 7}, we will include an extra 5 FREE bonus reports worth $199.

WHAT YOUR SUBSCRIBERS SEE
Dear Friend

If you order it before midnight on Tuesday, 16 January 2001, we will include an extra 5 FREE bonus reports worth $199.

When used in subjects or in the header and footer of an autoresponder, simply insert the tags as described above. The same applies when inserting tags into a one-off message.
