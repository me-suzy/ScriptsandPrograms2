FILTERING THE SUBSCRIBER LIST

Overview

  1. Click on Subscribers on the menu bar.
  2. Click on Set Filter on the sub menu bar.
  3. Set the filter criteria on the filter screen. These are explained below.
  4. Click the Show List button.

When you click on Set Filter (Step 2), you will see the Subscribers > Set Filter screen where you can set the criteria for the filter you wish to set. The more options you choose, the more targeted the list will be.

Clicking the Clear Filter link on the sub menu bar of the subscriber list will return to showing all subscribers, with no filtering.

Here is an explanation of the fields on the Subscribers > Set Filter screen:

  • Autoresponder

This list contains all the autoresponders on your system. If you choose an autoresponder here, your subscriber list will be restricted to only those subscribers who subscribed to this autoresponder. Choose 'All' for subscribers of all autoresponders to be shown.

  • Tracking tag

This list contains all the tracking tags currently set up on your system. Choose a tracking tag to restrict the subscriber list to only those subscribers who subscribed with this tracking tag. Choose 'All' for subscribers with all or no assigned tracking tag to be shown. Choose 'None' for subscribers with no associated tracking tag.

  • Current status

This list contains all the possibilities for the subscriber status. Choosing a status here will restrict the subscriber list to only those subscribers who currently have that status. The status choices are:

  • All
    Show subscribers with any status.
  • Active
    Show subscribers whose status is 'Active'. This means subscribers currently in the middle of a follow-up sequences or who are subscribed to an autoresponder whose mode is set to 'Continuous' (ie and is therefore never flagged as 'Finished' by the system).
  • Suspended
    Show subscribers whose status is 'Suspended'.
  • User Cancelled
    Show subscribers whose status is 'User Cancelled'. This means they chose to cancel a follow-up sequence before it was complete.
  • Finished
    Show subscribers whose status is 'Finished'. This means that their follow-up sequence completed successfully.
  • Failed
    Show subscribers whose status is 'Failed'. This means that Autoresponse Plus had tried to send them e-mails but these were returned undelivered. By clicking on System Setup on the menu bar, you can set how many failures will be allowed before a subscriber is flagged as 'Failed'.
  • Date scope

Choose an option here to restrict the list to subscribers who subscribed in the chosen date range. The choices are:

  • All
  • Today
  • This Week
  • This Month
  • This Year
  • Omit duplicate contacts

Check this box to omit duplicate contacts from the subscriber list. A duplicate subscriber is defined as one where the e-mail address is the same as a subscriber higher in the list. The subscriber's name is not taken into account when determining duplicates, only their e-mail address.

When you have chosen your filter criteria, click the Show List button. The list shown will match the criteria you chose.
