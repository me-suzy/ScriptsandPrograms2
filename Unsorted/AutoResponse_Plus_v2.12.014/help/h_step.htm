STEP BY STEP TO YOUR FIRST AUTORESPONDER

Here are the steps for creating your first autoresponder with Autoresponse Plus:

Step

Action

 Description How do I do that?
1 Create tracking tags
(Optional step)		 Create any tracking tags which you will use to track the source of your subscribers Click the Tracking Tags link on the menu bar.
More details
2 Create autoresponder Create a new autoresponder and set it's properties Click the Autoresponders link on the menu bar then click Create Autoresponder on the sub menu bar. Set the properties and click the Create button.
More details
3 Add your messages
(Repeat this step as required)		 Create and edit your immediate and follow-up messages. The autoresponder you just created will be shown in the list. Click the Edit link beside it. To add a message, click the Add Message link on the sub menu bar. This will bring up the page for editing your new message. Enter the details shown and type or paste in your message. Click the Save Changes button.
More details
4 Edit page layout Set the common layout for all your messages (eg headers, footers) Click the Page Layout link on the sub menu bar. Enter the details required and click the Save Changes button.
More details
5 Produce Subscription Code Produce the code for creating web-based subscription forms and links. Click the Generate Code link on the sub menu bar. Choose the options you require and click the Generate Code button. Copy and paste the appropriate code into your HTML editor.
More details

At each of these steps, click the Help link on the menu bar for detailed help on the options available at that point.
