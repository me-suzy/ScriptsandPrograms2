MANUALLY ADDING A NEW SUBSCRIBER

Overview

  1. Click on Subscribers on the menu bar.
  2. Click on Add Subscriber on the sub menu bar.
  3. Set the properties of the new subscriber. These are explained below.
  4. Click the Create button.

When you click on Add Subscriber (Step 2), you will see the Subscribers > Add screen. Here is an explanation of the fields on this screen:

  • Subscriber's first name

This entry is optional. If you enter a first (given) name for the subscriber, you can use it in follow-up messages with the {FIRSTNAME} tag.

  • Subscriber's last name

This entry is optional. If you enter a last (family) name for the subscriber, you can use it in follow-up messages with the {LASTNAME} tag.

  • Subscriber's full name

This entry is optional. If you enter a full name for the subscriber, you can use it in follow-up messages with the {FULLNAME} tag.

  • Subscriber's e-mail address

You must enter a valid e-mail address here.

  • Autoresponder

Choose the autoresponder you wish the new subscriber to be subscribed to.

  • Tracking tag

If you want a tracking tag to be associated with the new subscriber, choose one here.

  • Initial status

Normally you would leave this set to 'Active' which means that the follow-up sequence will be live immediately. You can choose other status values if you wish. These are:

  • Active
  • Suspended
  • Finished
  • Preferred format

Choose what message format (plain text or HTML) you would like the new subscriber to receive.

When you are finished setting the properties, click the Create button. The new subscriber will be created and you will return to the Subscribers screen. If the chosen autoresponder has an immediate message then it will be sent at this time.
