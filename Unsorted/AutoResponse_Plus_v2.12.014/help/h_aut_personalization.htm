PERSONALIZING YOUR MESSAGES

Overview

  1. Click on Autoresponders on the menu bar.
  2. Click on Edit beside the autoresponder to be edited.
  3. Click on Edit beside the message to be edited.
  4. In the message body, insert the tags described below.
  5. Click the Save Changes button.

Personalized messages are created using special tags that you insert in:

  • message subjects
  • plain or HTML message body text
  • headers and footers
  • one-off mailings

The personalization tags that can be used are:

  • {FULLNAME default}
  • {FIRSTNAME default}
  • {LASTNAME default}
  • {EMAIL default}
  • {TRACKINGTAG default}
  • {AUTORESPONDER default}

The personalization tags are case sensitive and should always be typed in upper case.

  • FULLNAME
    The full name of your contact (eg John Smith).
  • FIRSTNAME
    The first name (given) of your contact (eg John).
  • LASTNAME
    The last (family) name of your contact (eg Smith).
  • EMAIL
    Your contact's e-mail address.
  • TRACKINGTAG
    The tracking tag associated with the contact's subscription to this autoresponder.
  • AUTORESPONDER
    The autoresponder's listening address (eg info@yourdomain.com).

The default value is optional. It will be used only when the system does not have the necessary data in it's databases. If your default value needs to contains spaces, use the underscore (_) character instead. For example, Internet_Marketer. The underscores will be replaced with spaces before the message is sent.

If you want to force the system to use the defaults instead of the contact's real data, check the 'Force Default Personalization' box in the System Setup screen. Click Setup on the menu bar to go there.

Here is an example persoanlized message:

EXAMPLE MESSAGE BODY TEXT
Dear {FIRSTNAME Internet_Marketer}

Thanks for requesting more information on our widget product. You will find all you need to know in this e-mail.

WHAT YOUR SUBSCRIBER SEES WHEN THE SYSTEM KNOWS THEIR FIRST NAME
Dear John

Thanks for requesting more information on our widget product. You will find all you need to know in this e-mail.

WHAT YOUR SUBSCRIBER SEES WHEN THE SYSTEM DOES NOT KNOW THEIR FIRST NAME
Dear Internet Marketer

Thanks for requesting more information on our widget product. You will find all you need to know in this e-mail.

When used in subjects or in the header and footer of an autoresponder, simply insert the tags as described above. The same applies when inserting tags into a one-off message.
