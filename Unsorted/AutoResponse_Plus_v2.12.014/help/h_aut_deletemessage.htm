DELETING A MESSAGE FROM THE FOLLOW-UP SEQUENCE

Overview

  1. Click on Autoresponders on the menu bar.
  2. Click on Edit beside the autoresponder you want to edit.
  3. Click on Delete beside the follow-up message you want to delete.
  4. You will be asked to confirm your choice.

Hint
Deleting follow-up messages will affect current follow-up sequences for this autoresponder. Only delete a message when it is no longer required. The best way to do this is to set the 'Status' of the autoresponder to 'Dormant' so that no more subscriptions will be allowed. Then simply wait for all existing campaigns to complete before deleting the message.
