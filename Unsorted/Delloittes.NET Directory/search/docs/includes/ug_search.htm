Search

The deloittes.NET Directory allows users to find relevant results quickly and easily using the powerful feature-rich search engine.

At the core of the search is an intelligent engine that removes duplicate words and filters common keywords from within any given search string. This provides more accurate search results and allows for more generalised free English query type searches.

Boolean Support

The deloittes.NET Directory search engine also supports Boolean searches by selecting either "ALL", "ANY" or "Exact" from the Type Of Search drop down. By default the directory uses the "ANY" option.

For example, performing a search for "Big Apple" using the "ALL" option would only return results containing the full word "Big Apple". Searching for "Big Apple" with "ANY" would return results containing either the word "Big" or the word "Apple" or a combination of both.

If you wish to find exact matches within the directory, use the "Exact" option. This returns only results where the title or description exactly match your search term. Please note: keyword filtering is not performed when you do an exact search.

Narrow your searches

The deloittes.NET Directory offers the ability to narrow your searches to specific categories. You are only provided the opportunity to search specific categories if they have been created as link categories. Please see the "Allow Links" section within the "New Category" administration documentation for more information regarding link categories. Root categories have no resources within them and do not offer the opportunity to search within that category.

Search for URLs

You can search for addresses within the directory by adding "link:" to the start of your search strings. For example to find all resources within the directory from the Microsoft site you could perform the following searches.

link:www.microsoft.com
link:http://www.microsoft.com/
