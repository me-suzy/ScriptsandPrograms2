System Requirements

The deloittes.NET Directory requires the following minimum system features to run:

  • Windows 95/98 or NT/2000/XP Professional
  • Microsoft PWS, Internet Information Services 4 or compatible ASP Server

Recommended Specifications

  • Internet Information Services 5 or above running on Windows 2000 Professional / Server
