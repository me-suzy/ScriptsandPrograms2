Check Errors

Visitors using your directory are provided with the ability to submit error reports on individual resources within your directory to ensure your database is kept tidy and free from bad links or incorrect listings. The check errors administration features provides instant access to your error submissions and offers you the ability to view, modify, move or delete affected resources.

Modify Affected Items within the Directory

Aligned right of each error report is a "Show Resource" link. Clicking this link will take you to the affected resource. This provides you with the opportunity to update the resource, move the resource to another category or simply delete the resource from the directory. Once you have amended the bad resource, please remember to remove the error report.

Removing Error Reports

Error reports are logged to the tblErrors table within the database. To remove individual error reports simply click the "Delete" link aligned right of each report.

NOTE : This page is only available to users with administration rights.
