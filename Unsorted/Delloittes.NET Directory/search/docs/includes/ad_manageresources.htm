Manage Resources

The Manage Resource feature provides advanced search capabilities to enable quick and easy modification of resource information. You can search for resources by any attribute (see opposite) and choose to modify, move or delete your search results.

Managing Resources within the Directory

You can manage resources by using the feature icons aligned right of each resource title. Each feature icon and its functions are detailed below...

  Modify Resource

This icon provides a quick and simple way to modify resource details.

  Move Resource

This option allows you to move individual resources around the directory. All category count information will automatically be updated to reflect any changes.

  Delete Resource

This icon provides a quick and simple way to modify category details. You can modify a category name or specify if resources are allowed within the selected category.

NOTE : This page is only available to users with administration rights.
