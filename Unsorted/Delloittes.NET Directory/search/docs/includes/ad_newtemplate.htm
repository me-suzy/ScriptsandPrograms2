New Newsletter Templates

The Create New Newsletter Templates feature allows you to create and format unlimited newsletter templates for distribution to your mailing list. A newsletter template can include any number of dynamic elements. You can use the following tags to indicate where you would like the dynamic content to appear within the newsletter template.

@DirectoryName@ = Directory Name specified in directory settings page
@DirectoryURL@ = Path2Directory variable specified within configuration_file.asp (URL to directory)
@NewsletterContent@ = Latest submissions to the directory. You can change how many latest submissions appear on the directory settings page.
@DateStamp@ = Today's Date (Long date format - e.g. Monday, March 18, 2002)

These shortcut tags are case sensitive.

Template Name

This is used to identify your newsletter template. You can use any name you wish but a descriptive title would be a good idea.

Newsletter Subject

This is the subject line of the outgoing email. You can use any of the above shortcut tags within this field with the exception of the @NewsletterContent@ tag. The default subject line is @DirectoryName@ Update - @DateStamp@. This would read for example "ProgrammersPortal.com Update - Monday, March 18, 2002" within the subject line when it arrives on a subscriber's email client.

Newsletter Template

This is where you would enter the body of your newsletter. This field is provided as a multi-line text box to provide maximum freedom when creating newsletters. You may wish to add your announcements, latest news or comments to this section. Any of the dynamic tags will work within the area. A default newsletter template is provided which can be customised to suit your needs.

Default template

You can mark any newsletter template as a default template. The default template is indicated wherever you see a list of available templates.

Saving your newsletter

Once you are happy with the newsletter template click the "Save Template" button. You will then be given the opportunity to send a newsletter using your new template. You can also access your saved template at a later date if you wish.
