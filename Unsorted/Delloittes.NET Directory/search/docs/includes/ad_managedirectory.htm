Manage Directory

The Manage Directory pages provide all the features needed to organise, maintain and control your directory. You are presented with a replica view of your directory with administration features to help maintain your directory. You can modify, move or delete any category or resource within the directory enabling complete control and manageability over the data held within the directory.

Managing Categories within the Directory

Categories are managed by using the feature icons aligned right of each category heading. Each feature icon and its functions are detailed below...

  Modify Category Details

This icon provides a quick and simple way to modify category details. You can modify a category name or specify if resources are allowed within the selected category.

  Moving Categories

If you wish to move a specific category around within the directory, use this icon. Once you click the icon you will be presented with a list of categories within your directory and have the option to specify a parent category.

  Delete Category

As the name suggests, this option allows you to delete categories within the directory. You will be provided with a confirmation screen to ensure mistakes are not made. Please remember when you delete a category this will also remove all resources within the root of this category. If the category has sub categories, these will not be deleted. Please remove any sub categories first.

Managing Resources within the Directory

To modify individual resources within the directory you can use the Modify, Move or Delete icons aligned right of each resource within the directory. These options provide complete control over the links within your directory.

  Modify Resource

The Modify Resource icon provides instant access to a specific resource's details. You can modify all aspects of the resource including the resource title, the description, the URL, contact details and whether the resource is a favourite or sponsored listing.

  Move Resource

This feature allows you to easily move resources around within the directory. Once you click this link, you will be presented with a list of all the available link categories with the option to select a new category for the resource.

  Delete Category

This option removes a listing from the directory. A confirmation screen is provided to ensure mistakes are not made.

NOTE : This page is only available to users with administration rights.
