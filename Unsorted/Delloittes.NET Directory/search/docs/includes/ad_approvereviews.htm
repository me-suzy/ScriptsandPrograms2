Pre-screen Resource Reviews

To ensure only relevant screened reviews are submitted to the directory, all review submissions must first be approved by a directory editor. All reviews pending approval are displayed on the approve reviews page. You have a number of options available when approving user reviews.

Edit & Approve

The Edit and Approve option allows you to modify any part of a review before final submission to the directory. This ensures no malicious, vulgar or unsubstantiated reviews will be added to the directory. This is also a great way to ensure content authors do not submit reviews for themselves in order to gain more exposure.

Just Approve

Just Approve allows you to approve reviews instantly if you are happy with the information provided.

Delete Resource

You can remove reviews from the database if they are not suitable. A confirmation screen is provided to ensure reviews are not accidentally deleted.

NOTE : This page is available to all directory editors.
