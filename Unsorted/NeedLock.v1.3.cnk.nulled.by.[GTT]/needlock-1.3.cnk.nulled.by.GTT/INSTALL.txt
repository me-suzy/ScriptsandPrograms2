INSTALLATION NOTES

1. Upload directories "lock" and "protected_area" to your server
   ( directory "protected_area" is for storing all you protected
   contents and can be renamed to what ever you want )
2. Chmod program directory ( "lock" ) to 0777
3. Chmod directory with all protected contents ( "protected_are" ) to 0777
4. Chmod file lock/inf.php to 0666
5. Chmod file lock/denied_logins.dat to 0666
6. Run http://your-site.com/lock/install.php and follow the instructions.
