MyNewsGroups :) is a USENET news client, based totally on a web interface, written in PHP4. It's backended with a MySQL database, which allows us to develop useful tools such as search engines, SPAM filters, subscriptions, stats, and much more.

MyNewsGroups :) is based on the great work of Florian Amrhein, the developer of NewsPortal. With the use of MySQL and the complete library of Floh to query the News Server, MyNewsGroups :) can store all the articles in the news groups querying the news server only ONCE per article.
