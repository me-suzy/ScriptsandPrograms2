1. Anforderungen / Server

Um das FAMS nutzen zu können, muss Ihr Server einige Anforderungen erfüllen. Hier
eine Übersicht der benötigten Services:

- PHP3 oder PHP4 Support
- mySQL Datenbank Anbindung
- ca. 200 KB WebSpace für die PHP Files

Sollte Ihr Server einen dieser Services nicht unterstützen, kann ein korrekter Betrieb
nicht gewährleistet werden und [ BG-Studios ] übernimmt keine Haftung bei etwaigen Schäden!

Sollten alle Services vorhanden sein, so können Sie jetzt zum nächsten Schritt übergehen.

--------------------


2. Schritt für Schritt Installation


2.1
Als erstes laden Sie den Ordner "filearea" mit dem kompletten Inhalt (Programmdateien)
auf Ihren Webserver.
Danach wechseln Sie zuerst in das Verzeichnis "filearea": Dort finden Sie eine Datei mit 
dem Namen "config.php". Diese Datei muss unter chmod (Deteirechte) den Wert '0755' oder 
'777' zugewiesen bekommen.

Jetzt starten Sie Ihren Browser und rufen im Hauptverzeichnis ("filearea") die Datei 
'installer.php' auf. Bitte füllen Sie in diesem Fenster alle Angaben sorgfältig aus:

- mySQL Host: Hier muss entweder der Hostname oder die entsprechende IP stehen.
- Datenbank User und Name
- mySQL Passwort
- WebSite Name
- WebSite Adresse
- Kontakt eMail Adresse
- FAMS Pfad: (Siehe Hiweise direkt bei der Instllation!)
- Files pro Seite (gibt an wie viele Download File pro Seite gelistet werden sollen)


2.2 
Wenn Sie auf "Nächster Schritt" geklickt haben, werden die Daten entsprechend gespeichert
und das Programm erstellt automatisch in Ihrer mySQL Datenbank die 5 benötigten Tables.
Das sind:

- dl_users: Hier werden alle Nutzer des Systems gespeichert.
- dl_categories: Hier werden alle Kategorien abgespeichert, die Sie beliebig einrichten können.
- dl_files: Hier werden alle Daten zu den Downloads abgespeichert.
- dl_wertung: Hier werden die Bewertungen der einzelnen Downloads gespeichert.
- dl_comments: Hier werden die Kommentare zu den Downloads verwaltet.


2.3
Im nächsten Abschnitt legen Sie einen Benutzernamen, die eMail und ein dazugehöriges Kennwort fest.
Diese Daten werden benötigt, um sich für die Administration und Verwaltung der Dateien einzuloggen.
Hier herrscht ein Schutz gegenüber der Öffentlichkeit, um Mißbrauch zu vermeiden. Heben Sie sich diese 
Daten also gut auf.

Seit Version 1.05 können später über die Admin Oberfläche neue Benutzer hinzugefügt werden.
WICHTIG: Zum Einloggen müssen Sie Cookies in Irem Browser aktivieren!


2.4.
Im folgenden Schritt legen Sie die gewünschten Download-Kategorien fest. Diese werden benötigt, 
um vorhandene Files in mehrere Kategorien zu splitten und dem Benutzer so ein schnelles Navigieren
und Finden der Files zu ermöglichen. 
Geben Sie in dem vorhandenen Feld einfach den gewünschten Namen (z.B. Updates, Anleitungen, etc.) ein.
Es können vom System her beliebig viele Kategorien angelegt und verwaltet werden, jedoch muss MINDESTENS
eine Kategorie vorhanden sein, um den korrekten Ablauf zu gewährleisten!
Weitere Kategorien können später über die Admin-Oberfläche eingerichtet, editiert und gelöscht werden.

2.5
Haben Sie die Kategorien eingerichtet, so ist das Setup auch schon vollständig beendet. 
Sie können sich jetzt mit einem Klick auf "Zur Administration..." einloggen.
Geben Sie bei der Passwortabfrage GENAU die Daten ein, die Sie vorher festgelegt haben. 
ACHTUNG: Es wird zwischen Groß- und Kleinschreibung unterschieden!
Sind Sie dann auf der Admin-Ebene eingeloggt, können Sie mit der Dateiverwaltung beginnen!

BEVOR Sie jedoch weitermachen ist es aus Sicherheitsgründen besser, die vor der Installation abgeänderten 
Dateirechte wieder zurückzusetzten.
Das heißt für Sie, dass Sie jetzt der Datei 'config.php' im Hauptverzeichnis  die Dateirechte '0644' bzw. 
'644' zuweisen (rechte Maustaste: chmod - dann den Wert eingeben!).
