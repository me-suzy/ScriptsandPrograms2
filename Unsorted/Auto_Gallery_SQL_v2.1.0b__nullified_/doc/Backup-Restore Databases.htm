Backup/Restore Databases

Backup Instructions

To backup your data follow the steps listed below.
  1. In the administrative interface select the Backup/Restore Databases function, then press the execute function button. You will be taken to a page where you can select the backup file names for your HTML and MySQL data. Enter the filenames you want your data backed up to, or you can just leave them as the default.

  2. Once you have decided on filenames, make sure the Backup Data function is selected in the dropdown menu. Press the execute function button, and you will then be returned to the main administrative page. The backup function will run in the background, so you should allow at least 60 seconds for it to complete before you move on to the next step.

  3. FTP into your server and change into the data directory of your installation. You will find the backup files in this directory and ithey will have the filenames you selected when you did the backup. Download these files to your local computer in ASCII mode for safe keeping.

Restore Instructions

To restore your data, follow the steps listed below.
  1. Locate the last known good backup datafiles. Upload these two files in ASCII mode to the data directory of the AutoGallery SQL installation on your server.

  2. Log in to your administrative interface and select the Backup/Restore Databases function. Once on that page, type in the filenames of the backup files which you just uploaded to the server.

  3. Select Restore Data from the drop down menu and press the execute function button. You will be returned to the main administrative page and the data restoration will run in the background. Allow at least 60 seconds for the data to be imported, and do not continue on to the next step until you have allowed that amount of time to pass.

  4. Run the rebuild all pages function in the administrative interface to make sure all of the good data is being displayed on your TGP page.
