Upgrading AutoGallery SQL v2.1.x

Upgrade Instructions

Use these instructions to upgrade from v2.1.x to any newer version in the v2.1.x series.

  1. Upgrades between these versions are very simple. All you need to do is upload the new files to your server. All of the files that need to be uploaded are listed below. You should not run the autogallerysql.cgi script when upgrading between versions of 2.1.x. Make sure you setup the MySQL username, password, and database name in the ags.pl file before you upload it to your server.

    Main Directory
    ags.pl
    sql.pl
    cgiworks.pm
    http.pl
    init.cgi
    partner.cgi
    post.cgi
    report.cgi

    Admin Directory
    admin.cgi
    extern.cgi
    setup.cgi

    Data Directory
    errors.dat
    lang.dat

    Templates Directory
    All _admin_*.htmlt templates

  2. Once all of the files have been uploaded, your upgrade is complete.
Important Notes

If Perl 5 is not located at /usr/bin/perl be sure to update the first line of all .cgi files with the correct path to Perl on your server.

Make sure you upload all of the files in ASCII mode!

It is not necessary to delete any files from your server before you upload the updated scripts. If you do delete any files, make sure you reset all permissions on the files you upload.
