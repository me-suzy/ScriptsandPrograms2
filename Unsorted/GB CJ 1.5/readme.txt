GB CJ Installation Guide

Requirements:

- PHP4

Installation:

1. Upload the gif files in Binary mode and all other files in ASCII mode to working directory of your CJ 
2. Create subdirectories backupfiles, datafiles, memberfiles and chmod these subdirectories and main directory to 777
3. Chmod these files to 666: best.html, config.php, main1.html, main2.html
4. Run setup.php

Updating of old versions:
- Upload all php files, exclude config.php

Deinstallation:
- Upload and run uninstall.php
- Delete all stayed files
