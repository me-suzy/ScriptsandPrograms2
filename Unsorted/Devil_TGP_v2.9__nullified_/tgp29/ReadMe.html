TGPDevil Thumbnail Preview Add-On

Setup

Setup is pretty simple. First, upload the scripts included in the .zip file replacing the old scripts. Replace your old submit form and partner submit form with the forms included in the .zip. Make sure the scripts permissions are set to 755.

Set the variables in the config file. There are a few new variables in the config file:

$mondir - This is the full system path of the directory you would like the thumbs to be uploaded to.
$monurl - This is the URL of the directory you set with $mondir.
$maxsize - This is the max size of the image (in bytes)that you will allow to be uploaded. The script will count everything submitted to it INCLUDING the text from the fields the submitter filled out so I would recommend adding about 1024 (1K) to whatever value you have chosen.

Now call install.cgi in your browser. You should be provided with a message that the database setup was completed successfully.

Congratulations! The hardest part is over with.

Log in to your admin panel and go to your template management area. You will notice a new template in the list called "Thumbnail Image Preview Template". This is basically the same as the regular line template as it controls the way your image appears on the generated page. Below is an example of a good thumbnail image preview template.

Obviously #URL# will be replaced by the URL to the gallery and #IMGURL# will be replaced by the URL to the thumbnail on your server.

You will need to save a thumbnail image preview template before you can continue!

Last step!

Edit your main gallery page template and place the tag #-THUMB-# wherever you would like a thumbnail to appear. Thumbnails are ordered in the same way as regular text links. By date, rank, and order submitted. Newer/higher ranked thumbs appear higher on the days listings.
