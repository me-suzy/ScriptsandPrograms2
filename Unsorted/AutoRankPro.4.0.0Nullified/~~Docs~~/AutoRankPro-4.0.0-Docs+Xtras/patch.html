Upgrading AutoRank Pro v4.0.x

Upgrade Instructions

Use these instructions to upgrade from v4.0.x to any newer version in the v4.0.x series.

  1. Upgrades between these versions are very simple. All you need to do is upload the new files to your server. All of the files that need to be uploaded are listed below. You should not run the autorankpro.cgi script when upgrading between versions of 4.0.x

    Main Directory
    accounts.cgi
    arp.pl
    quick.cgi
    cgiworks.pm
    imgsize.pm
    http.pl
    out.cgi
    random.cgi
    rankem.cgi
    search.cgi
    sort.pl

    Admin Directory
    admin.cgi
    setup.cgi

    Data Directory
    errors.dat
    lang.dat

    Templates Directory
    All _admin_*.htmlt templates
    _rankem.cgit

  2. Once all of the files have been uploaded, your upgrade is complete. If you have created a customized gateway page, you will need to create a new one at this time and update your rankem.cgi script.
Important Notes

If Perl 5 is not located at /usr/bin/perl be sure to update the first line of all .cgi files with the correct path to Perl on your server.

Unless specially noted in the ChangeLog or recent news, it is not necessary to upgrade any of the add ons when a new version of AutoRank Pro is released.

Make sure you upload all of the files in ASCII mode!

It is not necessary to delete any files from your server before you upload the updated scripts. If you do delete any files, make sure you reset all permissions on the files you upload.
