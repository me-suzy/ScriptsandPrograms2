Demo - URL-only Pre-Submission Form
 
The purpose is to fetch the page, parse its meta html tags and fill that fields in the Full Submission Form whose data is available. Thus we save user's time of typing and thinking about what is appropriate to enter. To use this form in your web-site copy and paste the correspondent form's html code (the code between <FORM...> and </FORM> tags of this page source). By default the first form line is:
<FORM ACTION="/cgi-bin/submit-cgi/fetch.pl" METHOD="get">
Replace /cgi-bin/submit-cgi/fetch.pl with the actual path to your script to make it working properly. Make sure all of the rest html pages and templates containing forms in this software have right script locations specified.
