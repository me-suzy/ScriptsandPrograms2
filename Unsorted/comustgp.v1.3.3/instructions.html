Comus TGP usage instructions.

 

Including tgpinc.php and archiveinc.php files

Both of these files should be used as includes within your site design. As an example, create your TGP main page and save it as "tgp.php" Within this page, decide where you want the TGP list to be displayed and enter the following (without quotes)
"<?php include($DOCUMENT_ROOT . "/includes/tgpinc.php") ?>". Do the same for your archive.php page.

Approving Posts

Within the admin section of the site, you can view, approve or deny posts. To view a post, just click on the link in the left hand column. To approve or deny posts, simply select the yes or no radio buttons and hit "update". You can also select a rating from 1 to 10 for each gallery. Lower numbers will show up higher in the listing than the higher ones. The default setting is 5.

Black list

First decide if you want to use the Blacklist feature. If you do, select "Yes" in the admin configuration page.

To add someone to the blacklist, enter their email address by using the form on the blacklist page. To remove someone from the list, just click on "delete" next to their email address. Before a new post is submitted to the site, the submitters email address is checked against this list. If they are on it, they are informed of being on the blacklist.

Preferred submitter

First decide if you want to use the Preferred Submitter feature. If you do, select "Yes" in the admin configuration page.

To add someone to the preferred submitter list, enter their email address by using the form on the preferred submitter page. To remove them, click on "delete" next to their email address. All posts are checked against the preferred submitter list. If they are on the list, the posts show up on the live site right away and do not need to be approved.

Require Reciprocal Link

First decide if you want to require a reciprocal link. If you do, select "Yes" in the admin configuration page.

If this feature is turned on, the submitter will be required to have a link back to you on their page. Before the post is accepted by the script, it will check their page for your link-back. If its not there, the post is rejected and they are informed that a link-back is required. If the script finds a link-back, the post is accepted.

Checking links

Click on a group to check. A page will load and it will check all links associated with that category. After a while, a table will show up listing the status of each link, if they have your recip link or not, if they have a pop-up window on the page, if they have any banned words on their page. From this resulting page, you can check the links by clicking on them. For all the galleries you would like to delete, simply check on the box next to them and hit submit button.

Bad word check

This feature will check the gallery page and subject line for any words that you choose. If you want to use this feature, just select Yes in the admin setup area and add the words you want to search for in the. Just make sure that you separate the words with a comma. The search will also look for partial words... so if you ban "cat" and they have "catatonic" on their site... it will catch it and stop them from posting.

Use Confirm email

If you have this option set to yes, all non-preferred submissions will be emailed to the webmaster and they will have to reply to that email before you ever see it. This helps weed out junk postings.

Limit Archive Post View

You can limit how many galleries show up in the archive pages. If you start getting thousands of posts, you might want to think of limiting the number that show on the site.. Because if you try to show them all, it will take forever to load.

Tradescript Support

If you are using a traffic trading script, select Yes and add your connection strings below. All code that goes before the gallery URL goes in the top box and all code after the URL goes on the bottom.
