How to install Comus TGP

Before you do anything, check for the latest version of the software. Latest version can be found at http://www.nibbi.net/scripts/comus/.

If at any time you are have problems installing Comus TGP, please feel free to email me at nibbi@nibbi.net and ask questions.

  1. Unzip archive
    • Admin Folder
      • admin.php
      • blacklist.php
      • categories.php
      • check.list.php
      • ck.gallery.php
      • config.setup.php
      • db_setup.php
      • index.html
      • index.php
      • new.check.php
      • preferred.php
      • view.delete.php
    • Includes Folder
      • config.inc.php
      • formcheck.inc.php
      • archiveinc.php
      • tgpinc.php
    • Webmaster Folder
      • index.html
      • post.php
      • pref.php
      • submit.inc.php
      • submit.php (just a sample file)
    • accept.php
    • archive.php (just a sample file)
    • install.html (this file)
    • instructions.html
    • License.txt
    • tgp.php (just a sample file)

     

  2. Upload everything to your web server. The most important thing is to make sure that your "includes" folder and accept.php file is in the root of your web server.
  3. SSH into your webserver and change permissions (chmod) to 777.
  4. From a web browser, connect to your site and open the www.you.com/admin/
  5. Select "Change Settings" and enter all your setting requirements and click "Submit Changes" when done.
  6. Connect to http://www.yoursite.com/admin/db_setup.php (this will set up the databases).
  7. Set up all your category names
  8. You should be all set to go now. Just build your site and include the tgpinc.php and archiveinc.php within them.

That about it for initial setup.

The heart of the site is the "tgpinc.php" and "archiveinc.php" files. These are meant to be includes within your main TGP and Archive pages. To do this, just make your site design and include these files with the code "<?php include ($DOCUMENT_ROOT . "/includes/tgpinc.php") ?>" or "<?php include ($DOCUMENT_ROOT . "/includes/archiveinc.php") ?>". Be sure to save these new pages as "tgp.php" and "archive.php" respectively (I included samples you can build on).

To view a working example of this script, you can check it out at http://www.tgp-now.com.

A quick word on security

Many people have asked me about securing their admin and includes directories. The way I do it, and I suggest you do to, is use .htaccess and .htpasswd files. For more info on how to do this, just search the web for .htaccess and you will find more than enough information.
