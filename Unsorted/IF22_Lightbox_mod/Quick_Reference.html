Quick Reference: Upgrading
IF 2.2 Basic/Pro
to
IF 2.2 Basic/Pro + Lightbox

For full documentation, please visit: http://www.imagefolio.com/users

This document covers the lightbox installation of BizDesign ImageFolio 2.2 "lightbox ready" Edition on Unix and Windows systems. You should have IF 2.2 Basic/Pro working before continuing. Also make sure you have the "lightbox ready" version installed of the IF 2.2 scripts. You can find out if you are using a "lightbox ready" version by opening any IF 2.2 script in a text editor. Somewhere at the top it should say "lightbox ready".

Lightbox Installation

CREATE ON SERVER OUTSIDE CGI DIRECTORY:
imagefolio/lightbox directory (UNIX chmod 777)
imagefolio/lightbox/emailed directory (UNIX chmod 777)
imagefolio/lightbox/zipped directory (UNIX chmod 777)

ADD TO SERVER: (ASCII upload)
imagefolio/lightbox/lb.js
imagefolio/lightbox/bb.js
imagefolio/lightbox/ss.js
imagefolio/lightbox/slideshow.html
imagefolio/lightbox/index.html
imagefolio/lightbox/emailed/index.html
imagefolio/lightbox/zipped/index.html
CREATE ON SERVER INSIDE CGI DIRECTORY:
imagefolio/cgi-bin/if_lightbox directory (UNIX chmod 755)
ADD TO SERVER: (upload in ASCII)
imagefolio/cgi-bin/if_lightbox/lightbox.pl
imagefolio/cgi-bin/if_lightbox/lightbox_build.cgi (UNIX chmod 755, check path to perl)
imagefolio/cgi-bin/if_lightbox/lightbox_config.pl
imagefolio/cgi-bin/if_lightbox/lightbox_send.cgi (UNIX chmod 755, check path to perl)
imagefolio/cgi-bin/if_lightbox/lightbox_zip.cgi (UNIX chmod 755, check path to perl)
EDIT lightbox_config.pl
Open lightbox_config.pl in a text editor and follow directions.
RUN lightbox_build.cgi !!!
Open your Web browser and type in the URL to lightbox_build.cgi. Whenever you make a change to config.pl or lightbox_config.pl you need to run lightbox_build.cgi in your browser. This will create the file config.js in your lightbox directory. This is a JavaScript Source file with all the settings that the lightbox needs in order to function. Just remember: always run lightbox_build.cgi as the last step! ImageFolio 2.2 Pro owners can run lightbox_build.cgi from within the admin area (only after you set "$display_lightbox=1;" in your config.pl)
DONE
Now you only have to set "$display_lightbox=1;" in your config.pl and you are ready. Open your Web browser and type in the URL to imageFolio.cgi. If everything was installed correctly, you will see that your gallery has been updated to include the lightbox. Click on the any category to verify that everything is working properly. If things do not look correct, check your paths and settings in lightbox_config.pl and run lightbox_build.cgi again.
Note for Unix/Linux users...
Most text editors are set up to save files with Windows line endings. If you experience server errors when executing a script, you may need to change/set the preferences in your text editor to save files with UNIX line endings.
