Info Doorway Page Generator "Doorway v1.0"



INSTALLATION:
=============

1.) Bitte entpacken Sie alle Daten in ein Verzeichnis auf Ihren Computer

2.) Öffnen Sie die Datei "doorway.cgi" und ändern Sie die Parameter (Pfad zu Perl, sendmail)

3.) Erstellen Sie auf Ihrem Server ein Ordner "doorway" im cgi-bin Verzeichnis

4.) die HTML Dateien bitte im Binäri Mode, die doorway.cgi im ASCII Modus in diesen Ordner übertragen
    Die Datei "form.htm" muss nicht in das cgi-bin Verzeichnis übertragen werden

5.) Das Script "doorway.cgi" erhält die Rechte 755

6.) Öffnen Sie die Datei "form.htm" in einem Editor (Frontpage, Fusion etc) und ändern Sie die
    URL zum Script innerhalb des <form> Tags

7.) Die Datei "form.htm" auf Ihren Server aufspielen, der Ordner ist esgal

8.) Öffnen Sie zum Test die Datei: "form.htm" und generieren Sie Doorway Pages. Dieses
    sollte ohne Fehlermeldungen geschehen. Die Doorwaypages werden dann an die angegebene Email versendet
