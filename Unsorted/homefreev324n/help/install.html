I N S T A L L A T I O N     I N S T R U C T I O N S:

Installation has been made as easy as possible... all configuration of script variables will be done online via your browser, which means you don't have to worry about editing any of the Home Free 3.24n script's code.

Begin your installation by unzipping the Home Free v3.24 zip file with your favorite unzip program.

Create an images dir on your server and upload all images contained in the images dir from the zip file.

Create a help dir on your server and upload all the files contained in the help dir from the zip file... This will be your support and help files linked from the actual scripts.

Create a data dir on your server that will hold all Home Free's data files. This directory should be below the document root, or somewhere that it can not be viewed from the web. This directory should also be writable by any cgi script. (chmod to 777 for most, 755 for others depending on server setup)

Create a members dir on your server that will hold your members files. This directory should be readable from the web.

Next if your path to perl is different from #!/usr/bin/perl you must change the path to the correct path on your server. This is done by editing the first line of ALL files ending in .cgi (no path to perl is needed for .pl files)

Now upload all .cgi and .pl files in ASCII Mode to your cgi-bin dir, or the dir you want to use for your Home Free v3.24n cgi files.

Once files are uploaded make sure the dir they are loaded in is chmodded to 755, and then chmod all .cgi files to 755. No chmodding is needed for the .pl files, except for the variables.pl file which should be chmodded to 666

Now call up the admin.cgi script in your browser. You should see the main Home Free configuration screen. If you get an error, check the path to perl, and that you uploaded in ASCII and finally that you chmodded the file to 755. If none of this works, try removing and reinstalling the package.

The rest is pretty much self explanatory. The configuration screen has a paragraph explaining the variables you need to set right above the text field. Some may even have the variable configured for you. I have included a script and link to list your server's environment variables if you need it to determine paths or your location for sendmail ;-)!

For help on using and installing Home Free 3.24n, check out your LOCAL HELP MANUAL
