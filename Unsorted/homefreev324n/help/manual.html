H O M E   F R E E     U S E R     M A N U A L:


Using the admin section of Home Free   The most powerful part of Home Free has got to be the administrative section of things.This is where all the configuration of the Home Free variables happens and also all the managing of the users accounts. All configuration of Home Free is browser based, once set up, the cgi scripts never need to be edited.

  General Operations   the General Home Free operations section allows you to configure the main appearance and settings for your Home Free script.These variables should be setup before you start giving out accounts.

  Color customization   Color customization allows the admin to change the the colors of all attributes shown to the end user via new.cgi and manager.cgi

  Required user information   The user information section of Home Free allows you add additional fields to your new.cgi signup form. You can make these fields required, optional, or off completely. These fields can be used to collect valuable demographical information about your users. You can later view the data entered by the user by view the individual account.

  Reserved / Banned usernames, emails and ip address   The reserved banned section is pretty much self-explainitory, or already explained in the admin prorgam itself, but here are a few more tips.

  Categories in Home Free   Another one of HomeFree's amazing features is the ability to add categorizes to your site. These categories allow different sites to join under different areas and can help provide a theme and community for various parts of your site. Using categorizes is a snap if you follow the follow instructions.

  Editing Categories   Editing existing categories.

  Headers and Footers   The Header / Footer option allows an admin to control the html headers and footers for each category, for the new.cgi and members.cgi page, the signup terms, the code to print on new signing up sites, emails and more.

  EZ-Web use and configuration   EZ-Web is a complex html generator that allows those with little or no knowlegde of html to create html pages by filling out different form variables.

  User Guestbook   The guestbook option is another unique and powerful feature that can be offered to all, some or none of your homefree accounts.

  Web Board / WWWBoard   The web board is a unique and powerful feature that allows you to give a simply web board to each Home Free account, selected accounts or no accounts at all. The web board option has many different options and is a great add on to provide for users or even charge them for as an upgrade.

  FTP uploading/importing   This new feature allows members to ftp into the server containing Home Free and upload their files. Please note: this is not true ftp access

  Form Mail   Form Mail allows users to put html forms on their web site, and have the result emailed to them.

  Account Listing/Search Engine   The search.cgi script allows web sites contained in Home Free to be browsed in a directory/search style format.

  Account Browsing   Browsing and searching accounts from admin section of Home Free

  View Account   View and change account details and settings.

  The new accounts "default" web site   As of version 3.2, an entire web site can be create upon a users sign up, for each user, and this site can be customized to what the user entered in their signup form.
