Required user information
Posted on: March 11, 2000, 12:27 a.m.

The user information section of Home Free allows you add additional fields to your new.cgi signup form. You can make these fields required, optional, or off completely. These fields can be used to collect valuable demographical information about your users. You can later view the data entered by the user by view the individual account.

The information asked of the user can be changed at any time by clicking the "User Information" part of the image map in the admin section.

The following are the configurable fields. All can be on,off or required:
  • Name
  • Address
  • Email
  • City
  • State (US States)
  • Zip
  • Country
  • Telephone Number
  • Gender
  • Marital Status
  • ICQ Num
  • Education
  • Income
  • Occupation
  • Birth Date
  • 10 Custom Fields

Custom Fields

The 10 custom fields although you to ask any questions of your new users you like. In any of the custom boxes, select on or required. If the text area, enter the question you want to ask your user. The question then will appear on the sign up form along with a spot for the user to answer.
