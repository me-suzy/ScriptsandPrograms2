Color customization
Posted on: March 11, 2000, 12:19 a.m.

Color customization allows the admin to change the the colors of all attributes shown to the end user via new.cgi and manager.cgi

With color customization, Home Free can take on the look, and match any site. Simply enter the color you wish the attribute to take in the text field and press the submit button. All changes are automatic and instant. Format of colors can either be white or #FFFFFF Do not use quotes of any kind.
  • Background color: The color of the background for all your Home Free signup and manager pages. Default is white, leave blank for default

  • Table Background color: This is the background of all tables in Home Free. The table header contains a short description or header of what is in the table itself. Default is white, leave blank for default.

  • Table Background color #2: Some sections of tables in Home Free use alternating table colors, the first color is set above in "able Background color", this field is the second color Default is #e1e1e1, leave blank for default.

  • Table Header Background color: This is the background of the header in all tables. The table header contains a short description or header of what is in the table itself. Default is body background, leave blank for default.

  • Text Color: Color of text in Home Free, as would appear in a body tag of an html document. Default is black, leave blank for default. (outside of all tables)

  • Link color: Color of all links as would appear in a body tag of a document. Default is blue, leave blank for default.

  • Color of text in table: Enter in this box the color of the text used inside tables. Default is black, leave blank for default.

  • Color of text in header of table: Enter in this box the color of the text for the table header. Default is white, leave blank for default.

  • Color of highlighted text in header of table: In the header of table, sometimes words are needed to standout, this color is used to make those words standout. Default is red, leave blank for default.

  • Font: Which font should be used for all text shown to user. Default is Arial, leave blank for default.

  • Font Size: Size of the text shown to the user. Default is -1, leave blank for default
