Account Listing/Search Engine
Posted on: March 24, 2000, 2:03 a.m.

The search.cgi script allows web sites contained in Home Free to be browsed in a directory/search style format.

The account listing search engine feature bundled with Home Free is run using the search.cgi file. Configuration of this feature uses 5 different variables. They are as follows:

1. Display Status Icons - Checking this check box will then have certain images show up next to an account when it is shown. These 4 images, that come with Home Free are displayed at the following times..

search_never.gif - Account created, yet never logged in
search_new.gif - Account created in the past X days
search_today.gif - Account that has been modified within the last day
search_plain.gif - Account that has been activated, not new and not modified in the last day

2. Number of days for account to be labeled "new" - If icons are to be displayed... this text box should hold the number of days an account will retain the "new" icon.

3. Show only sites with name and description entered - Accounts are listed by what the enter as their site name and description as entered via the file manager by the person owning the account. With this box checked, only accounts that have entered a name and description will be shown.

4 + 5. Header/Footer for search.cgi - Selecting a header and footer here will have the html associated with the header and footer chosen placed above and below the listings in search.cgi

How the listing/search engine works

If the current implementation of Home Free is using categories, then the main page of search.cgi will contain the search text box, and a listing of all the active categories. The category names are clickable, when clicked, a list of all account in that category are shown, 25 per page.

If categories are not used, then the main page will contain each letter of the alphabet below the search text box. Clicking on a letter will bring up all accounts beginning with the letter clicked, 25 per page.

The search engine searches all accounts. It searches the site name and description the user entered, plus the current index.html page in the users account. Clicking on advanced options will bring up different options that can be used in searching the accounts. Accounts are displayed 25 per page unless selected otherwise.
