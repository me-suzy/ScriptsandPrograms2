View Account
Posted on: March 25, 2000, 1:36 a.m.

View and change account details and settings.

The view account option can be found by logging into your admin.cgi file and scrolling down to the View Account section.

Once you have selected an account to view and it's corresponding Category (if you use categories), you can click the view account button. This will take you to a new page that displays all of the account information along with the ability to edit many options.
  • Account name: is not editable but allows you to keep track of just which account you are viewing easily.

  • Categories: is also not editable but allows you to check which category an account is in.

  • Url: allows you to see the url for which an account is posted at. This allows you to easily go to that account's site.

  • Email: allows you see a user's email address for direct user emailing.

  • Password: is the fist editable option. Here you can view or change a user's password which can be very effective for banning a user for a short time if you do not want to simply place them on hold.

  • On Hold: is the next option and can be used to place an account on hold so that the user can not login to edit/delete/upload any new files. You can find out more about the hold function here.

  • Number of files: is not an editable field but does let you see the total number of files a user has in their account.

  • Number of folders is not an editable field but lets you see the total number of folders and sub folders in a user's account.

  • Space Taken: allows you to see the total amount of space taken by the account including how much space they use and how much they are allocated to use.

  • Add more space: allows you to add additional space to a user's account which can be helpful if you have paying accounts, the ability to upgrade an account or a popular site that just needs more space.

  • Display Headers and Footers: allows you to toggle off/on headers and footers for a given account. It is not enabled by default but can be useful once again for paying accounts, account upgrades or an account that just shouldn't have the headers or footers.

  • Allow Web Board: allows a user to have their own web board script if the web board is on and it is not already set to work on all accounts.

  • Allow GuestBook: allows a user to have their own guestbook script if the guestbook is on and it is not already set to work on all accounts.

  • Name and Description: is another editable field and allows you to view a site's entered name and description. These fields will be used in future view account listing/search options.

  • Last logged in: allows you to see when an account was last logged into.
  • Moderator Categories: allows you to make an account a moderator. Moderators are able to login to admin_browse.cgi and view all the accounts in their selected directories. They then have the ability to put any account on hold if they feel it should be.



All the following fields have to do with entered account information from new.cgi. These fields are all editable and contain the user's demographic information. This data can be used as you see fit.
  • Address: This field can be used to collect the user's physical address for mailing them information.

  • City: This field can be used to collect information about a user's city for demographics or mailing address.

  • State: This field can be used to collect information on a user's state code for demographics or a mailing address.

  • Zip: This field can be used to collect information on a user's zip code for demographics or a mailing address.

  • Country: This field can be used to collect information on a user's country for demographics or a mailing address.

  • Telephone Number: This field can be used to collect information for contacting a user via phone.

  • Gender: This field can be used for collecting additional demographic information about your users.

  • Age: This field can be used to collect information on your user's age demographics.

  • ICQ Number: This field can be used to find out more about your users and for easier contacting.

  • Education: This field can be used for more demographics on your users.

  • Income: This field can be used to find out even more about your users and their income.

  • Occupation: This field can be useful for judging what types of people use your site for advertising use.

  • Birth Date: This field can be useful for demographics and to make sure users are over a certain age if required.

After all the user information options, you will see a submit button to save the data. Click this if you have made changes or simply want to return to the main admin screen. If you wish to actually login to the user's account, click the "Login to account manager for "account"" and you will be taken to that account's manager.cgi page and be able to edit/delete/rename/upload additional fields to their account.

There is also a directory listing after the login to account option. This directory listing lets you easily delete the entire account or just a few selected files. It also provides for simple account file viewing.

All of the viewing options allow you to work with your user's account, see their files, edit their options and view their demographic information. Once an account has be edited, you can not return the information to the old setting without updating it yourself.
