Using the admin section of Home Free
Posted on: February 25, 2000, 2:35 p.m.

The most powerful part of Home Free has got to be the administrative section of things.This is where all the configuration of the Home Free variables happens and also all the managing of the users accounts. All configuration of Home Free is browser based, once set up, the cgi scripts never need to be edited.

Main Menu & Stats - The main admin screen that displays the number of users and current stats about your Home Free

Paths & Directories - The main configuration variables entered during initial setup of Home Free.

General Operations - Customizing the many options and settings of Home Free [more...]

Color Customize - Customize the colors and look used throughout the html displayed to users.

Logging and Reports - Set and view security logs. Also set "bad words" that will trigger a log entry.

User Information - Configure Home Free to allowing/require user to fill out extra data fields during signup. [more...]

Reserved / Banned - Setup reserved account names that no one can register, and ban users fromjoining by email or IP address

File Types - Add file types that users are allowed to upload to their accounts.

Categories - Ability to set different communities and categories for users. Configure, add, edit and delete categories here. [more...]

Headers / Footers - Manage files that allow you to customize the look and feel of your Home Free, and add html to your users html pages. [more...]

EZ Web - EZ Web is a html builder for users. It has a few settings and configurationoptions. [more...]

Guest Book - Settings for the user guest book feature. [more...]

WWWBoard - Settings for the user wwwboard [more...]
