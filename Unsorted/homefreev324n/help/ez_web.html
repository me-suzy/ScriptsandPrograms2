EZ-Web use and configuration
Posted on: February 28, 2000, 1:29 a.m.

EZ-Web is a complex html generator that allows those with little or no knowlegde of html to create html pages by filling out different form variables.

For members to use EZ-Web, they should press the "Create" button from the main screen on their file manager.They will then have a selection to use the EZ-Web system or plain html.

Configuration options for EZ-Web can be accessed from the drop down box in the main admin page of admin.cgi.Once that screen has been accessed, you will see 4 text boxes. Labeled as Background Images, General Images,Horizontal Rule, Email Images. Image urls entered in these text boxes will appear in the EZ-Web system in drop down boxes, so the users can select them. Urls entered into the text boxes must be in the format of:

image url,image name,image width,image hieght

The image name should be a short 2 or three word description of the image that will be used.As is in something like Small Smiley. Width and Height attributes are optional and are not needed at all for background images.Inlimited number or images are available, but each one must be on an new line. All attributes must be seperated by a single comma.

An example of what one might put in the text box:

http://localhost/images/welcome.gif,Flashing Welcome,200,75
http://localhost/images/under_c.gif,Under Construction,100,100
http://localhost/images/button.gif,Small Button,88,31
