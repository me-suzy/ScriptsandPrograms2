User Guestbook
Posted on: February 28, 2000, 1:36 a.m.

The guestbook option is another unique and powerful feature that can be offered to all, some or none of your homefree accounts.

It is simple to use, simple to setup and provides a great option for your users.

To use the guestbook you must first login to your amdin.cgi script and then select the Guest Book option from the drop down list.Once you have selected this option, you will be taken to a page with several options. If you have not enabled the web board then you must follow the following instructions to do so:
  • To enable a GuestBook at all, you must check the box next to the "Guestbook as a feature: " option.
Once the guestbook is enabled, you can edit or enable any of the following options:
  • The option "Run Guestbook as an admin option per account" disables the use of a guestbook by any account unless the webmaster decides to enable it for that account. To enable a guestbook for any account, simply go to the view account screen and check the "enable guestbook" option for that account.
  • By enabling the "Should Guestbook files count towards account size" option, every message posted on an account's guestbook will count towards their total space used. This option can be helpful in limiting the amount of messages on the severand keeping disk usage down.
  • If you select the "Use different headers and footers for Guestbook file" option, you will need to fill outthe two following text areas. These boxes allow you to paste separate html code to be sued on the header/footer of the guestbook messages and script. This option is not required and should not be checked if you want to use the same headers and footersas in the rest of your homefree script.

These options can be used to effectively help you run a powerful free home page site and allow an extreme amount of flexibility in your account. All additional guestbook configuration options for colors, text and names can be found in the user's account under the configure guestbook option.Additional information on using those options can be found on the Account Manager tutorial.
