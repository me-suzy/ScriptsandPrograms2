Web Board / WWWBoard
Posted on: March 24, 2000, 2:03 a.m.

The web board is a unique and powerful feature that allows you to give a simply web board to each Home Free account, selected accounts or no accounts at all. The web board option has many different options and is a great add on to provide for users or even charge them for as an upgrade.

To use the web board you must first login to your amdin.cgi script and then select the Web Board option from the drop down list. Once you have selected this option, you will be taken to a page with several options. If you have not enabled the web board then you must follow the following instructions to do so:
  • To enable a web board at all, you must check the box next to the "Web Board as a feature:" option.
Once the web board is enabled, you can edit or enable any of the following options:
  • The option "Run Web Board as an admin option per account" disables the use of a web board from any account unless the webmaster decides to enable it for that account. To enable a web board for any account, simply go to the view account screen and check the "enable web board" option for that account.
  • The "Limit number of possible messages per account" option allows you to limit how many messages any one account can have on their web board. This is very useful if you feel that an account is overusing your transfer with the web board or if you simply want to keep cpu usage down.
  • By enabling the "Should Web Board files count towards account size:" option, every file posted on an account's web board will count towards their total space used. This option can be helpful in limiting the amount of messages on the sever and keeping disk usage down.
  • If you select the "Use different headers and footers for web board files:" option, you will need to fill out the two following text areas. These boxes allow you to paste separate html code to be used on the header/footer of the web board messages and script. This option is not required and should not be checked if you want to use the same headers and footers as in the rest of your home free script.

These options can be used to effectively help you run a powerful free home page site and allow an extreme amount of flexibility in your account. All additional web board configuration options for colors, text and names can be found in the user's account under the configure web board option. Additional information on using those options can be found on the Account Manager tutorial.
