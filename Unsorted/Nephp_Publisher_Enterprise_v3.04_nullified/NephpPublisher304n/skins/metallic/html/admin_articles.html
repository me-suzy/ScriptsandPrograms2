Section Information:

1. New Article
This section allow you to add new article entries. In this section, there are two options. One is dynamic editor and the other is normal editor. Dynamic editor requires correct browsers in order to work. If your browser is not listed in the requirements, please use the normal editor. Normal editor normally takes HTML source code only. Otherwise, the output will look horrible.

2. Edit/Remove Articles:  this section allows you to modify the existing article entry and as well as removing it.

3. Approve Articles: If approval is required in the setting, then all the new articles that submit or modify by members will need your approval before they can be listed.
