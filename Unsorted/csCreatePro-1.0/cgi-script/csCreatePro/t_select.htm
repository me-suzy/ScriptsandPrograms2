Browse the server to get to the directory where the file or directory exists or will exist. Then choose one of the 3 actions you can perform:
  • To add a file to the system, select the file you wish to add to the system then choose "Import a Page" from the action dropdown.
  • To create a new file, select "Create a Page" from the action dropdown.
  • To create a directory, choose "Create a Directory" from the action dropdown list.
