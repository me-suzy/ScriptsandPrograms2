Olate Download - Download Management Script
Documentation - Readme

Welcome to the readme.htm file for the Olate Download script provided free of charge from Olate! This is the index to the other documentation pages you will find in the /docs directory of the download.

Olate Download allows you to add a downloads area on your own website which is easily customisable (via a central CSS file) and easy to administer (via an extensive administration area).
