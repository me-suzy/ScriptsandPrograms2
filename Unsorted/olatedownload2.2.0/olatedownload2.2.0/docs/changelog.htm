Olate Download - Download Management Script
Documentation - Change Log

This is a complete list of changes made to Olate Download since version 2.0.0. As well as allowing you to submit bug reports or feature requests, development of Olate Download can be tracked at http://www.olate.com/development.

Changes In v2.2.0

  • Renamed LATEDownload > Olate Download
  • New Feature: (ID: 4 & 5) Installation & Upgrade system
  • New Feature: SQL injection vulnerability sites removed/secured
  • New Feature: Performance tweaks
  • New Feature: Modified admin login/logout/authentication functions to make up to date and compatible with Windows
  • New Feature: (ID: 20) Search facility with ability to disable via admin area
  • New Feature: Improved method of obtaining script path without need for user input
  • New Feature: Ability to mark users as 'Master' preventing them from being deleted via the admin area
  • New Feature: (ID: 8) Ability to define 2 custom fields for each download that are displayed on the details.php page if present
  • New Feature: (ID: 9) 'All Downloads' link displays all downloads on page with the ability to sort and disable this feature from the admin area
  • New Feature: (ID: 36) Ability to enable and display ratings
  • New Feature: (ID: 19) Downloads can be sorted by name, rating and downloads from the main index with ability to disable via admin area
  • New Feature: Added entry of essential data checks - non entry of essential data in admin area will result in abortion of the insertion of the data into the database and will display a friendly error message
  • New Feature: Removed all global variables where possible
  • New Feature: (ID: 32) Moved constants into array and renamed file to config.php
  • New Feature: Script version displayed in General Config
  • New Feature: (ID: 23) Script path required for language features - editable in admin area
  • New Feature: (ID: 1) Full language support in user area
  • New Feature: (ID: 2 & 21) Full language support in admin area: separate page showing current language, available languages plus language file information including author name and URL
  • New Feature: Tidied up layout code and made user area XHTML Transitional 1.0 compliant
  • New Feature: Added a dot · between each additional data field in index.php and details.php to allow easier reading
  • New Feature: Modified and improved admin login/error screens
  • New Feature: Error handling made to look nicer
  • New Feature: Code tidied and reduced
  • New Feature: Made fully compatible with all future versions of PHP by changing $HTTP_*_VARS[''] to $_*['']
  • Bug Fix: Only display content boxes if download exists.
  • Bug Fix: (ID: 31) Friendly error now displayed if no downloads/categories are in the database and edit/delete existing is selected
  • Bug Fix: Removed redundant config options from database and main functions
  • Bug Fix: Wrong filename in functions_admin.php in error handling
  • Bug Fix: Fixed incorrect error.php redirect If user accessed an admin page whilst not logged in
  • Bug Fix: Description text too far to the right when no image present problem fixed
  • Bug Fix: "Not rated" displayed on details.php now rather than 0 when no votes have been cast for file
  • Bug Fix: "Notice: Undefined index: category" error in index.php on Windows/IIS servers fixed

  • New Feature: Display name of category in /admin/downloads/edit_view.php rather than number
  • New Feature: Additional information added to /admin/other/update.php
  • New Feature: Description added to /admin/other/license.php
  • New Feature: Change log added to documentation and /docs/css/style.css updated
  • Bug Fix: "Undefined variable: admin_username" error in Admin Area on Windows/IIS servers fixed
  • Bug Fix: session_start() error in Admin Area on Windows/IIS fixed
  • Bug Fix: Fixed issue where if current category is kept selected in /admin/downloads/edit_view.php, the category is set to 0
  • Bug Fix: Prevented 'Default' category being displayed on /admin/categories/edit.php
  • Bug Fix: Name field enlarged to 35 characters in /admin/downloads/add.php and /admin/downloads/edit_view.php
  • Bug Fix: Added separator between current category name and list of categories in /admin/downloads/edit_view.php
  • Bug Fix: Removed useless table column in documentation pages
