The Feedback Forum allows you to leave comments about your experiences with your eBay trading partners. Your comments are extremely valuable to the eBay community, and you will find that your own Feedback Profile is an asset at eBay, helping you earn the trust and respect of others in the eBay community.

What is a Feedback Rating?
A Feedback Rating is a measure of an eBay user's reputation in the community. Every registered user at eBay has a Feedback Rating that is calculated by

  • Adding 1 point (+1) for each positive comment received.
  • Subtracting 1 point (-1) for each negative comment received.
  • Awarding 0 points (0) for each neutral comment received.

Even though a single person can give Feedback on the same user multiple times, he/she will be able to affect that user's Feedback Rating by only one point. For example, if he/she leaves three negative comments, the Feedback Rating will be decreased by only one. If a user leaves the same number of positive and negative comments, his/her contribution to the rating will be zero. In other words, you can "vote" as many times as you want, but in the end, you are only saying, "yes" (+1), "no" (-1), or "neutral" (0).

The number in parenthesis (##) beside each person's User ID or e-mail address represents his/her total feedback rating. A star may appear next to the feedback rating.
