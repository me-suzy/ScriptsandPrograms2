Installationsanleitung für
das Besuchertauschscript 2.5

Schritt 1

Installieren Sie bitte zuerst die Datenbank aus dem Textfile
in dem Unterverzeichnis "Datenbank".
Muß komplett hochgeladen werden


Schritt 2

Das muß geändert werden:

öffnen Sie die Datei  prepend.inc.php und header.inc.php im Hauptverzeichnis mit einem beliebigen Texteditor
geben Sie oben die Verbindungsdaten zu Ihrer Datenbank ein.
Die Datei admin/header.inc.php und member/header.inc.php auch.

Neu: in der Datei var.inc.php Datenbank Daten eingeben
     


Nun kannst Du alles auf deinen Webspace hochladen.
Jetzt kannst Du die Restlichen Daten uber den Admin Bereich ändern.
gehe dazu auf www.DeinSeitennamen.de/admin
( dieses Verzeichnis unbedingt durch ein Passwort schützen. Solltest Du nicht wissen wie das geht Frage bitte deinen Provider. )

So das wars schon jetzt sollte alles Funktunieren :-)
