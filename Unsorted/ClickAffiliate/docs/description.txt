A package of perl scripts to help you create a link affiliate program for your site. Very
 easy to use. It allows affiliates to link to you and keep track of clicks. It has 2
 protection levels to count only unique visitors (by domain check + cookies). It helps
 with payment process by printing labels for envelopes to send payments. It allows
also to send a message to all affiliates. Can work with any currency and click rate you
 define.
