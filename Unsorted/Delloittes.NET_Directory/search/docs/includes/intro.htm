Introduction

The deloittes.NET Directory brings together your most important electronic resources within your company and provides employees, partners or customers centralized access to company information, documentation and links. Using the directory you can index and manage links, articles, images, company documentation in fact any kind of electronic content and allow your users to access this information quickly and easily in one central location. The intelligent search-engine provides fast relevant paged results providing instant access to the collective intelligence of your organization.

Complete Control

You can easily manage every aspect of your information and directory structure using the secure directory administration pages. The comprehensive suite of administration tools provides complete role-based control of your categories, resources, newsletters, mailing lists and directory behaviours.

Seamless Integration

Template driven the deloittes.NET Directory can integrate into your existing corporate or personal website, company intranet or extranet and with just a few lines of code the whole integration process can take less than 1 hour. The use of Classic ASP includes and encapsulation allows you to easily wrap your custom designs around the directory functions with very little effort.

Flexible Design

The deloittes.NET Directory is written completely in Classic ASP 3.0 and requires no additional ActiveX DLL components to operate. All business and presentation code is provided and can be modified to suite your specific requirements. This allows you to develop additional custom features should you require.

Scalable Database

The directory is provided with support for both Microsoft Access and Microsoft SQL Server databases. This allows the database to scale as your data requirements demand. You can easily switch databases at any time by modifying the configuration_file.asp. Support for several popular email components is also provided ensuring minimal changes to your current hosting environment.


This documentation assumes you are familar with web technologies and servers.
