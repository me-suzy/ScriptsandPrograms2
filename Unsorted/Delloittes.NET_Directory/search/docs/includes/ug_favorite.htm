Your Favourites

The "Your Favourites" feature offers visitors the ability to bookmark individual items within your directory. This allows visitors to easily access or monitor resources of interest within the directory at a later date from the "Your Favourites" Page.

Please remember the "Your Favourites" features requires a cookie-enabled browser to work. The cookie holds a comma delimited list of resource IDs which is used to extract the bookmarked items from the tblSites table within the database.
