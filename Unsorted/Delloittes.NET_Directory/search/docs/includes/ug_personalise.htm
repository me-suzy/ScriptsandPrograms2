Personalise

The personalisation options available with the deloittes.NET Directory allow your visitors to personalise various features within the directory.

Visitors can personalise the following...

  • The number of sub categories to show below main categories,
  • The character used to separate the navigation links,
  • Directory results per page,
  • Search results per page,
  • Number of "Latest Additions" to show on homepage,
  • Number of "Popular Resources" to show on homepage,
  • Number of "Favourites" to show on homepage.

All personalisation settings are saved within the tblConfiguration table within the database. The cookie is used to save the user's personalisation ID which is then read from the database upon each visit within the configuration_file.asp

Users must have cookies enabled for personalisation to work.
