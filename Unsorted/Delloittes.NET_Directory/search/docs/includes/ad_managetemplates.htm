Manage Newsletter Templates

You can modify or remove existing newsletter templates using the Manage Templates features. You'll find all your newsletter templates presented on this page.

Modify Template

Modify Template allows you to modify the template name, subject line and body content of any given newsletter. This is very much like creating a new template except the form will be pre-populated with your information.

Preview & Send

The Preview & Send link next to each template allows you to view your newsletter with the dynamic tags replaced with the actual content. You can then choose to send a newsletter using this template to your mailing list by clicking the "Send this Newsletter..."

Delete

This allows you to delete newsletter templates. A confirmation screen is provided to ensure no templates are accidentally removed.
