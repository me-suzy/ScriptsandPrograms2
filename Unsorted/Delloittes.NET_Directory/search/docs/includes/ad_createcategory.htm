Creating a New Category within the Directory

The "New Category" option provides a quick and simple way to create a new subject category within the directory. The deloittes.NET Directory supports unlimited hierarchical categories. The various options presented when creating a new category are listed below.

Category Title

This is the title of the category as it would appear within the directory.

Allow Links

If you allow links within a category this becomes a link category. Link submissions are allowed within link categories. A link category can contain both resource submissions and sub categories.

Categories that do not allow links are known as root categories. Root categories can only hold sub categories, they cannot contain link submissions.

Parent Category

The parent category contains a complete list of categories already within your directory. From this list you can select a parent category which your new category will live under.
