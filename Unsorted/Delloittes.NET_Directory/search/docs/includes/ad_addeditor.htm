Add Editors

The Add Editors page allows you to create profiles for additional directory editors. The deloittes.NET Directory administration pages support unlimited directory editors. When creating a new user you will be asked to provide the user's Full Name, Username and Password. You can also specify the type of access to the directory administration pages by checking the admin checkbox.

Users with administration privileges have unlimited access to all the features within the administration pages. A user with no administration privileges accessing the directory admin pages, will only be provided access to the approved resources and approved reviews pages.

NOTE : This page is only available to users with administration rights.
