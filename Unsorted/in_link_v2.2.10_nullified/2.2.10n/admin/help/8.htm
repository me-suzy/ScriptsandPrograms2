8       Upgrading and Support

Upgrading from previous installation is almost the same as a doing a new installation.  We recommend that you backup all of the old files in a separate folder or by downloading the files on your local computer.  Pay attention to all the files that you added or modified within the In-link system after the initial install (such as your customized templates, modules or language files). Also, we recommend backing up the database and also moving it into a different location.

 

After your completed the backup process, upload the new system into the same or a different folder and run admin/install.php

 

If youre upgrading from an earlier version, that may require a change in the database structure that the installation program will perform.
