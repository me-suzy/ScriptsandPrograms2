ImageFolio Commerce offers all the same features of the Professional Edition plus:
Configuration

  • Support for different currencies
     
  • Sell digital downloads, service products, tangible products or a mix of all
     
  • Global store pricing or based on category possible
     
  • Choose between radio fields or pull-down menu site-wide
     
  • Thumbnail border options: none, slide like, keyline.
     
  • Settings for your own color scheme
     
  • Wishlist function
     
  • Searchwords can be viewed and linked to new searches
     
  • Option of "Show Similar" link on product page which returns similar products
     
  • Hiding of Categories/Subcategories possible through the admin area for development or secure access
     
  • Uploading of Category image trough the admin area
     
  • Specify maximum search result possible
     
  • Full IPTC support:
    • editing through admin area
       
    • create your own fields
       
    • searching of data
       
    • option to show underneath thumbnails
       
  • Anti image Leech making it impossible to steal your image links (will return access denied image if accessed from other domain
     
  • If ImageMagick is installed then the option of watermarking is possible (with option to cache watermarked image for quick access)
     

Taxation Options

  • No tax
     
  • Tax by state/province/country
     
  • Flat tax
     

Shipping Options

  • No shipping
     
  • Based on number of items
     
  • Shipping by weight
     
  • Shipping by price
     
  • Max package weight setting (divide in multiple packages if you go over)
     
  • Handling charge (will always be added to every package)
     
  • Setup your own list of couriers the customer can choose from or:
    • Use UPS system to calculate shipping cost based on weight, zipcode. Customer can choose from the options you select like overnight, ground, etc. (USA only)
       
    • Choose countries that your customers can order from
       

Transaction Support

  • Gateway options (more then one can be used at the same time to give customer the option!):
    • ImageFolio Commerce's own gateway (if you can process the CC yourself)
       
    • PayPal
       
    • AuthorizeNet
       
    • fax/phone/check
       
  • After PayPal and AuthorizeNet verify the payment they will write back to the ImageFolio Commerce database (Unique!!)
     

Customers

  • Automated customer account creation process
     
  • Optional automated log-in for customers via cookies
     
  • Checkout automatically fills in customer's contact and shipping information
     
  • Customers can create their own product wishlists
     
  • Lost password finder feature
     
  • Ability to view order status and review past orders
     

Orders

  • Customers can view order status and review past orders
     
  • Customers and site owners both receive order confirmations/invoices by email
     
  • Credit card numbers are optionally stored encrypted on your server using a proprietary encryption system
     
  • Credit card numbers are optionally NOT sent via email
     
  • Admin ability to:
    • view, modify, and delete orders
       
    • print orders
       
    • print invoices
       
    • view decrypted credit card numbers
       
    • perform fraud checks - address mapping, phone number verification, address verification, area code check, zip code check
