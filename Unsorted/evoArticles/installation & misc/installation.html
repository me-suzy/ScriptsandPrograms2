Requiments

  • PHP (atleast 4.10) & mySQL (3.22 or better)
  • Safe Mode is disabled
  • one mySQL database with ALTER, CREATE and DROP permission
  • atleast 5mb of web space
  • *optional* mod_rewrite enabled for Apache servers. This is required for the SES(search engine safe) url rewriting feature.


    Installation

    1. Open config.php file in /admin/ folder
    2. You will see this :
      $db['user'] = "root"; // mysql username
      $db['pass'] = ""; //mysql password
      $db['host'] = "localhost"; //mysql host/server
      $db['name'] = "uart"; //mysql database name


      Change the $db['user'] to match your mySQL username
      Change the $db['pass'] to match your mySQL password
      Change the $db['host'] to match your mySQL host (leave 'localhost' if you don't know what youre doing)
      Change the $db['name'] to match your mySQL database name
    3. Upload all files
    4. Run http://yourdomainname.com/articles/setup.php
