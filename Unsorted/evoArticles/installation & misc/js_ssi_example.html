use the following code

<script language="Javascript" src="ssi.php?get=latest&js=1"></script>

you can change the "latest" to 3 types of feeding.
latest - will display latest articles
toprated - will display articles with the highest readers rating
popular - will display articles with the most views

you can also use the following variables to customize your feed
cat=x - will only display articles from a certain category. Replace x with a category id.
new=1 - if new is set to 0, all links will open in a new window (target="_blank"), else; it will open in the same window
total=x - will change the limit of displayed headlines. change x to an integer.


examples:
  • Displaying 50 most Popular Articles. (with links that open in new windows)
    <script language="Javascript" src="ssi.php?get=popular&js=1&limit=50&new=0"></script>
