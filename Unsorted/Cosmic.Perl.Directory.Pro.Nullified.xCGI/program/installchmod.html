Install problem, chmod not permitted.

The install can still be completed. But your server is not permiting this installation program to chmod files. For the installation to be completed you must manually set the file permissions to the appropriate chmod settings.

IMPORTANT NOTE:- You must set the CHMOD settings before AND after the install. Settings before and after are different.

Pre install CHMOD settings

adminpass.txt 777
manager.cgi 777
search.cgi 777
submit.cgi 777
dirpro.lib 755
dirprovar.lib 777

Post install CHMOD settings

adminpass.txt 777
manager.cgi 755
search.cgi 755
submit.cgi 755
dirpro.lib 755
dirprovar.lib 777

Once you have set the pre-install chmod settings click below to start the install. To view this page again open "installchmod.html" from your tempory program folder. E.G. "C:\temp\dp\program\installchmod.html".
