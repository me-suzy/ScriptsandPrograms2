Usage Guide

This is the usage guide for CosmicPerl.com's Directory Pro.

This software will allow you to create your own directory similar to that of Yahoo or the DMOZ. This package contains all you need to run and maintain your directory, with advanced features such as keyword advertising and extra user fields.

First you'll want to configure the look and feel of your directory. Selecting how categories and search results are displayed. With Directory Pro you are not confined to a set list of styles, our template system allows you to fully configure the appearance of your directory.

For a guide on editing all of the template files please click here.

Once you have changed and uploaded the template files you'll need to add a search box to your site. A HTML page containing an example search box is "search.html". You search your directory by using search.cgi. The search page example contains a form like:-

<form action="http://www.yourserver.com/cgi-bin/dp/search.cgi" method="GET">
<input type="text" size="34" name="terms">
<input type="hidden" name="want" value="search">
<input type="hidden" name="field" value="normal">
<input type="submit" value="Search!" name="submit">
</form>

The hidden form variable "field" dictates which site fields are searched. The various options are below:-
"normal" (or nothing) - Searches site name and description.
"u1" to "u10" - Searches the specified extra user field.
"all" - Searches all the user fields, the site name and description.
By having a selection box you can allow your visitors to select which fields to search.

You may also want to link to your major categories. As category pages are generated as HTML files (for maximum speed and lowest CPU resources), you link to their folder within your search data folder.
E.G.
"http://www.yourserver.com/searchdata/CATEGORYNAME".
So for the category "Files" you would have:-
"http://www.yourserver.com/searchdata/Files".

Now point your browser to:-
"http://www.yourserver.com/cgi-bin/dp/manager.cgi?want=admin".

The default Username is "admin", the default Password is "adminpass". Input this into the form and enter the management area.

First you should setup your own username and password. There is a link in the manager page for doing this.

The options on the manager page are:-

View Submissions
View Advertising
Edit Categories
View Modifications
Generate Search Data

Change Username/Password
Configure Variables

Click on the links above to find out about each feature of this program.

The Faster Search option

With this package there is a refined search.cgi file named "search2.cgi". This file is more difficult to setup, which is it is not the default. If you are new to setting up cgi scripts then we suggest you stay with the default search.cgi. If your directory has grown large, or you just want that extra bit of speed then click here for the setup guide.

Allowing visitors to navigate your site with ease helps increase return visitors. A popular, well done search engine will attract visitors to your site and help promote visitors to bookmark, or add your site to their favorites.
