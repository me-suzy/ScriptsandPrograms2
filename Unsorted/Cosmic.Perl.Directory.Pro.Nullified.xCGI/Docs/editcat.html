Usage Guide

Editing the Categories

Once you have entered the manager area select "Edit Categories". All of your main categories will be displayed (none if it's your first time). At the bottom is a form to create new main categories.

Each category displayed will have 6 options:-

Modify: If you change the category description this button will update the changes.
Remove: Selecting this will remove the category if it has no sub-categories.
View sub: This will display any sub-categories. Each sub-category will have these 6 options.
View sites: This will display all standard sites within the category, where you can edit and delete them.
View topsites: This will display all topsites within the category, where you can add, edit and delete them.
View advertising: This will display the category specific banner, where you can edit or delete it.

Setting up sub-categories
If you select "view sub-categories" from any category you will be able to add new sub-categories at the bottom of the page.

Setting up topsites
If you select "view topsites" from the edit category page, you will be able to modify/remove current topsite listings. You can add new topsite listings using the form at the bottom of the page.

Managing sites
If you select "view sites" from the edit category page then you will be able to modify/remove current standard listings.

Setting up category advertising.
If you select "view advertising" from the edit category page then you will be able to modify/remove/add the current categories advertising.
