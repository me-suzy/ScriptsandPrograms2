Usage Guide

Faster Searching

The search2.cgi file contains a refined version of search.cgi. This file has variables and the templates built in so that time is not wasted opening and closing files. Edit this file using a text editor such as notepad, or wordpad (always save as text-only in wordpad).

At the top of the file is the path to perl, make sure this matches perl on your server. Under the copyright is a list of variables, each of these should contain the HTML from the appropriate template file. IMPORTANT NOTE:- The HTML should be placed between the ' and the ';. If your HTML contains a ' then problems will arise as it will mistake this for the end ';. You can either remove the ' from your HTML or add a \ in front of it, such as \'. The \ tells perl to ignore the '.

Below the template variables are some of the configuration variables. You must set these to match your setup.

Once you have setup the search2.cgi file, rename it to search.cgi and upload it to your server replacing the old search.cgi file. Make sure you upload in ASCII.
