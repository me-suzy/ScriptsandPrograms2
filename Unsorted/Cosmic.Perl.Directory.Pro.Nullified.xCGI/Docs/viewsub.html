Usage Guide

Viewing Submissions & Modifications

By selecting "view submissions" on the main management page, all submissions will be displayed. You will have the options to add or remove (reject) the submission, you can also modify the fields. You must be sure to refresh the submissions page after adding or removing each submission.

By selecting "view modifications" on the main management page, all site modification submissions will be displayed. You will have the options to add or remove (reject) the modification, you can also modify the fields. You must be sure to refresh the modifications page after adding or removing each submitted modification. If you are not allowing users to modify their listings, or you have set auto modification on, then you will not need to access this part of management.
