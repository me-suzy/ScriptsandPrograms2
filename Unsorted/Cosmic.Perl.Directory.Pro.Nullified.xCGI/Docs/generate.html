Generating Search Data

To get the maximum performance, all the site and category data are modified and refined for search efficiency. When you make any changes to your directory, such as adding new sites, new categories, removing sites, etc, you should select this option. If changes are made and this isn't selected then those changes wont appear in search results. So if you add a new site, it will be displayed in it's category, but not in any search's, until you generate the search data.
