'Easy Install' Guide

Please follow the steps below to install this software package.

Step 1.
Using your preferred Text editor open the unzipped file install.cgi. E.G. On a windows system open 'notepad' then open the file "C:\temp\dp\program\install.cgi".

Step 2.
The top line of the install.cgi file contains the line:-
#!/usr/local/bin/perl
You need to change this line to the path to perl on your server. This is usually:-
#!/usr/bin/perl
Or
#!/usr/local/bin/perl
Some Servers have different paths to perl your host should provide this information on their technical cgi documentation.

Once you have changed this line save the file and exit the text editor.

Step 3.
Using your preferred FTP program, upload the contents of the temporary directorypro folder "program" (E.G. C:\temp\dp\program) to a folder on your server (usually "cgi-bin/dp/"). You will need to create this folder on your server. You MUST be sure to upload all ".cgi" and ".lib" files in ASCII mode. For an FTP guide visit our site http;//www.cosmicperl.com.

Step 4.
Using your preferred FTP program select the file 'install.cgi' and chmod this file 755. For a guide to using Chmod visit our site http://www.cosmicperl.com/support.shtml or view your FTP program documentation.

Step 5.
From now on its 'Easy install', no more chmod or FTP. Point your browser (E.G. "Internet explorer™" or "Netscape™") and browse to the file install.cgi which you uploaded to your server. E.G. Address:- "http://www.yourserver.com/cgi-bin/dp/install.cgi". The new 'Easy install ©' system will guide you through the rest of the install process.
