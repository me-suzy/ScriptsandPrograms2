Readme for CosmicPerl.com's Directory Pro V2.1


Step 1.
Unzip the contents of directorypro.zip to a temporary folder on your system. 
 E.G. "C:\temp\".

Step 2.
Using your prefered browser (E.G. "Internet Explorer" or "Netscape") open the 
file index.html from within the temporary folder. E.G. "C:\temp\dp\index.html".
This will guide you through the install process.
