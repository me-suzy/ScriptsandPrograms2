  1. mysql

Edit mysql.inc file with your MySQL login information.
Create database using createdb.php script. To do this just type URL of this
script in your browser.
After that use index.html to upload and store Excel files in MySQL database.

  2. html

Use index.html to upload and view Excel file data.

  3. demo

This directory contains demo of ExcelExplorer that you can see on-line at
http://yourdomain/excelexplorer/demo/
To run this demo please assure that the "sessdata" and "uploads" directories
are writable.
