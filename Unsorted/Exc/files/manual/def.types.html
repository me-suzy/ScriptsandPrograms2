Appendix A. Changes history
Types


GetCellType method returns the cell type, GetCellData method returns the data according to the cell's type. Use the following table in order to work with the returned data correctly:

Type ID Excel Type Returned Data Type Description
0 empty null Empty cell. This cell type indicate that this cell is not defined.
1 number integer or float Integer or floating point number. If it >=0 it can be used to convert into the date type format.
2 percentage float Percentage. Floating point number must be multiplied by 100 to take persentage.
3 text string Strings can be stored in packed unicode (8-bits per character) or unpacked unicode (16-bits per character) formats. To find out in which format this string is stored use IsUnicode method. To correctly print string in HTML format use AsHTML method. You can also use AsPlain method to force converting 16-bits to 8-bits format. To get data unmodified use AsIs method. Also, if string represents correct numeric value you can use AsDate method convert it into the date format.
4 logical boolean Boolean value. It can be either TRUE or FALSE.
5 error code integer Error code:
 
Error code Error value Description
0x00 #NULL! Intersection of two cell ranges is empty
0x07 #DIV/0! Division by zero
0x0F #VALUE! Wrong type of operand
0x17 #REF! Illegal or deleted cell reference
0x1D #NAME? Wrong function or range name
0x24 #NUM! Value range overflow
0x2A #N/A! Argument or function not available
6 date array Date - an associative array containing the date information, as the following array elements:
"string" - string representation of the date, using Excel cell format
"year" - integer, year
"month" - integer, month (1-12)
"day" - integer, day of the month (1-31)
"hour" - integer, hour
"min" - integer, minutes
"sec" - integer, seconds
"usec" - integer, microseconds

Currently supported most of all build-in and some custom Excel date formats. But if cell formatted in Excel using unknown custom date format, it will be returned as number. In this case you can use AsDate method to obtain date structure from the cell.
No matter what date system was used to create an Excel file - 1900 (Windows) or 1904 (Macintosh) - class automatically detect date system and return properly converted date.
7 blank null Blank cell. This cell type indicate that this cell is defined (in difference to empty cell), but no data stored in it.
8 merged null This cell type indicate that this cell is within merged cells area. See the merged cells explanation for more information.
