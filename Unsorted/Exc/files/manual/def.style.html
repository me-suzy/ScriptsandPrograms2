Appendix A. Changes history
Style


Information about cell style (font, format, background color, alignment, borders) returned as array with the following structure:

Array Index (key) Returned Data Type Description
bgcolor array Background color. If undefined then no background color assigned to the cell (see notes).
font array Font used to display contents of the cell. Always defined (see notes). However if this index is undefined there are no font record associated with the cell (can occur only in corrupted or erroneously generated Excel files).
font_index integer Index of the font record in array returned by GetFontsList method. Always defined (see notes). However if this index is undefined there are no font record associated with the cell (can occur only in corrupted or erroneously generated Excel files).
format string Format string (text type). This value is undefined if build-in format used.
format_index integer Index of the Excel cell format string in array returned by GetFormatsList method or build-in format ID (see below). Always defined (see notes). However if this index is undefined there are no format string associated with the cell (can occur only in corrupted or erroneously generated Excel files).
align integer See alignment info array structure. Always defined (see notes).
valign integer See alignment info array structure. Always defined (see notes).
word_wrap integer See alignment info array structure. Always defined (see notes).
border array Information about borders. Always defined (see notes).

Notes: if some read options specified for exploring Excel file then some values can be undefined. For instance if 'read_font' set to FALSE then 'font' and 'font_index' will be undefined as no font information will be readed from an Excel file.
