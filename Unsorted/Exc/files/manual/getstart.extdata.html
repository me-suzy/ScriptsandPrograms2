Extracting Data


After object creation you can start to extract data from an Excel file. There are different methods available to do this (see note below):

integer Explore ( string $excel_file_contents [, array $options] )
Parameters
$excel_file_contents
String that holds binary Excel file contents.
$options
Exploring options (see below).
Return Value

Returns 0 if file explored successfully or error code otherwise.

Remarks

This method extract data from an Excel file and store it in internal structures. To obtain extracted data you can use methods provided by this class.

integer Explore_file ( string $excel_file_name [, array $options] )
Parameters
$excel_file_name
Name or full path to the Excel file need to be explored.
$options
Exploring options (see below).
Return Value

Returns 0 if file explored successfully or error code otherwise.

Remarks

This method works exactly as Explore, but works with file rather than with a string. Useful also when you plan to work with ExcelExplorer in memory-safe mode.

Important!

If you use Explore_file in memory-safe mode then you must call Close method before terminating the script.

boolean Close ( )
Parameters
none
Return Value

Always returns TRUE.

Remarks

This method must be called before terminating the script if Explore_file in memory-safe mode was used to read data. This will close the file that was opened by ExcelExplorer.


Error codes

 
Error code Description
0 Exploring done without error.
1 File corrupted or not in Excel 5.0 and above format.
2 Unknown Excel file version.

Exploring options

You can specify some options to maximize efficiency or Excel file processing. Options passed to Explore or Explore_file methods in array containing keys as option name, and a value as option value. If some option was not specified default value will be used.

Option Type Description
read_only_sheets_info boolean If TRUE then only worksheets info (title, type, visibility, etc) will be readed from an Excel file. Useful when you need only to extract worksheets names for example.
Default value: FALSE
Note: if this option is set to TRUE all other options has no effect.
explore_sheet int If specified, then data from the specified worksheet only will be readed. Otherwise data from all worksheets will be readed.
read_font boolean Set this to FALSE if you don't need to read info about cell fonts.
Default value: TRUE
read_format boolean Set this to FALSE if you don't need to read info about cell formats.
Default value: TRUE
read_align boolean Set this to FALSE if you don't need to read info about cell alignment.
Default value: TRUE
read_bgcolor boolean Set this to FALSE if you don't need to read info about cell background color.
Default value: TRUE
read_border boolean Set this to FALSE if you don't need to read info about cell borders.
Default value: TRUE
read_formula boolean Set this to FALSE if you don't need to read data from cells that contains the formula.
Default value: TRUE
read_link boolean Set this to FALSE if you don't need to read info about hyperlinks.
Default value: TRUE

Note: if you want to read data from several Excel files you can use Explore or Explore_file method more than once. But if you want to read several sheets from one Excel file you can use ExploreWorksheet method (which will not explore the whole file again - therefore it will works much faster) with different parameters:

<?php
  // ... initialization

  // read data from the first sheet
  $ee->Explore_file($excel_file_name,array('explore_sheet' => 0));

  // ... working with readed data

  $ee->ExploreWorksheet(1); // read data from the second sheet

  // ... working with readed data

  $ee->ExploreWorksheet(2); // read data from the third sheet

  // ... working with readed data

?>

But take in mind, that you can use ExploreWorksheet only after call to Explore or Explore_file.

Another opportunity ExcelExplorer provide is that you can serialize readed data (store) somewhere in order to unserialize (read) it back and works with readed data without calling Explore or Explore_file again. This feature cannot be used in memory-safe mode. See SerializeData and UnserializeData functions for detailed description.
