News Display Options
This section specifies what happens when a user clicks on one of the individual news item links.
Options:
1) You can have the 'Body of the News item' displayed in a popup window or the same window/frame where the link resides.
2) You can specify ' NO Link' if you feel the 'Title', 'Date', 'Brief Description' and 'Location' is all that is required.
3) You can point the link to an existing web page, a link on your server or anywhere on the Internet.
4) You can also upload a file and have it automatically linked (pdf, doc, xls, etc.
