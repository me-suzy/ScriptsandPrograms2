Installation and using "Ad Master 1.0"

Possibilities:

By means of the "Ad Master 1.0" you can create and control Ad management site.
The main possibilities include:
  • User can be registered, add their sites, choose banner category, banner itself, type of banner code (for example, for new window banner or the same window banners);
  • User can add (change) the params of their bank account and personal info and see their status of payment (for banner clicks). Admin can add banner categories, banners themselfs, code types;
  • Admin can search for any user, delete him (them) from DB or change his (them) information;
  • Admin can control money transfer for every user;
  • There are three example categories of baner in "Ad Master 1.0", there are two example programs for Computers & Internet category and there are two kind of example banner links in Business to Business@ program. See and try.


Requirements: PHP4, MySQL, JS1.2 compartable browser.



Installation

  1. Unzip archive and copy "lib" directory content to where you are going to store all php files. Add that directory to "include_path" in php.ini;
  2. Change params of DB in Conf.inc (lib/sys/Conf.inc), change $USER_PAGE_PATH to where you are going the store the main page of the system, change $UPLOAD_PATH to where you are going to upload images;
  3. Inserrt data.sql (lib/sql/data.sql) in your db (you can PHPMyAdmin from http://www.phpwizard.net);
  4. Copy directory admin and files: index.php, insert_user.php into your WWW directory;
  5. Run index.php from that directory. If you will not see the main page, you had a problem in previous installation steps;
  6. If you can see the main page you can try to register in system;
  7. Run insert_user.php if you want to add random test users in your db;
  8. Protect admin directory by means of HTTP server (for example htaccess in Apache);
  9. Go to admin/ and change the templates to what you want;
