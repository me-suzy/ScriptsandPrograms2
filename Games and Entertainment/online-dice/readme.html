Online Game Dice v. 1.05 - Readme

Adding dice to your online game is easy. As long as you have a web site with PHP enabled, you just have to add this script and you're good to go. Just follow these simple instructions:

  1. Download and unzip the Online Game Dice folder
  2. Copy the folder to your web site directory using your preferred method (FTP, for example).
  3. Edit the three variables in the config.php files with your favorite editor (Notepad, for instance).
  4. Include the file dice-include.php wherever you want to show the dice. The code is simple, just add <?php include('online-dice/dice-include.php'); ?>, if your web page is in the root directory.
  5. Make sure to save the file with a .php extension so that the include works properly.

Feel free to take a look at the example file, example.php, for a demonstration of how to use the include.
