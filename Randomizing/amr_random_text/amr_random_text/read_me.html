5 easy steps to installing and setting up your own AMR Randomized Text.

1.: Decompress the amr_zip file you just downloaded.
2.: Open index.php with a text editor such as TextEdit [mac] and edit the variables.

Ex.: $title = "Testimonials"; [Change Testimonials to whatever you want.]

3.: Okay your almost done. Now use one of the codes below to call the script.

Use this code to call the file to you php script. <?php include("index.php"); ?>

Or use the following code to call it to a HTML page.

<iframe frameborder=0 src="http://www.ursite.com/amr_random_text/index.php" width="430" height="300"></iframe>

4.: Now add your text files to the root folder that contains the index.php file.
Make sure you name them as follows.

Ex.: 0.txt, 1.txt, 2.txt, 3.txt, etc... Do this to as many times as you like.

5.: Open your FTP client software and navigate to your domain. Upload the decompressed amr_text_randomizer folder. And your done.
