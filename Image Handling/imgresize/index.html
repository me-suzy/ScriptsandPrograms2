This script is meant to help you easily resize pictures. Main features:

  • Create image from different files like JPEG, GIF, WBMP, PNG
  • Several resize modes like
    • "fit to given rectangle"
    • "cover to given rectangle"
    • "resize to fit exactly to given dimensions" (geometrical distortions may occur)
    • "resize to given width, no matter what resulting height might be"
    • "resize to given height, no matter what resulting width might be"
  • Output modes
    • Direct output to the browser in JPEG or PNG format
    • Output to file in JPEG or PNG formats
  • GD2 support to work with true color images which greatly improves image quality.
