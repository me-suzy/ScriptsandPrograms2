Software Requirements
This script has only been tested in a LAMP (php4) environment via web server. It currently only supports PNG file exporting.

Required:
- PHP4
- GDlib
- True Type Font Support

Installation
Upload the 3 included files to any directory.

Once uploaded open rss_button.php.

Configure your font path on line 20.
Example:
$FONT_PATH='/home/myusername/public_html/fonts/'; //(case sensitive)

Configure your font file on line 23.
Example:
$FONT_FILE='arial.ttf'; //(case sensitive)

Once your settings are in place save the file and/or re-upload it.

Configuration
After you have setup your font settings, you can add the PHP and form code in graphic_config.php to any page. When the form is submitted it will generate a link that is echoed to the browser upon success.

Optionally you can call rss_button.php directly to auto-create buttons on your web site:
Example:
Use this inside a <img src=""> tag - it will return a png image.
rss_button.php?MAKE_GRAPHIC=true&LS_FONT_TEXT=RSS&LS_FONTCOLOR=FFFFFF&LS_BACKGROUND=352021&RS_FONT_TEXT=FEED&RS_FONTCOLOR=352021&RS_BACKGROUND=A2AEB0&VBAR=30
